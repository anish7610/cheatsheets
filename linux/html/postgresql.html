PostgreSQL Linux Command

Description

The postgresql command is used to start the PostgreSQL database server. It initializes the database system, listens for client connections, and performs database operations.

Common Options

Option Description
-D datadir Specify the data directory.
-p port Set the port number to listen on.
-k directory Set the directory for Unix domain socket connections.
-c name=value Set a configuration parameter.
-h hostname Listen for connections on the specified host.

Usage Examples 


# Start PostgreSQL with a specific data directory
postgresql -D /usr/local/pgsql/data

# Start PostgreSQL on a custom port
postgresql -D /usr/local/pgsql/data -p 5433

# Start PostgreSQL and listen on a specific network interface
postgresql -D /usr/local/pgsql/data -h 192.168.1.20
