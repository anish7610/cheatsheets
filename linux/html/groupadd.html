Linux `groupadd` Command

The groupadd command is used to create a new group on a Linux system.

Common Options

Option Description
-g, --gid GID Specify the group ID for the new group.
-f, --force Force removal of existing group with the same name.
-K, --key KEY=VALUE Override /etc/login.defs defaults.
-r, --system Create a system group.

Usage Examples 

# Create a new group called 'developers'
sudo groupadd developers

# Create a new group with a specific GID
sudo groupadd -g 2001 staff

# Create a system group
sudo groupadd -r sysadmins

Cheat Sheet 

# Basic command
groupadd 

# With GID
groupadd -g  

# Force with existing name
groupadd -f 

# Create system group
groupadd -r
