lspci Command Guide

Description

The lspci command lists all PCI devices on a Linux system, detailing information about hardware components like graphics cards, network devices, and more.

Common Options

Option Description
-v Display verbose information about devices.
-vv Show very verbose output with detailed information.
-k Show kernel drivers handling each device.
-s <bus>:<slot>.<func> Only show devices in the specified domain.
-d <vendor>:<device> Display devices with specific vendor and device ID.

Usage Examples 

# List all PCI devices
lspci

# Display verbose information
lspci -v

# Show kernel drivers handling each device
lspci -k

# Display devices in a specific domain
lspci -s 00:02.0

# List devices with a specific vendor and device ID
lspci -d 8086:1237

Cheat Sheet

lspci
