dpkg-reconfigure Command Guide

Description

The dpkg-reconfigure command is used to reconfigure an already installed package on Debian-based systems.

Common Options

Option Description
-f Specify the frontend to use (e.g., dialog , readline ).
-p Set the priority of the questions (e.g., low , medium , high , critical ).
--default-priority Use the default priority set in the system.

Examples 

# Reconfigure a specific package interactively
sudo dpkg-reconfigure 

# Use a specific frontend, like readline
sudo dpkg-reconfigure -f readline 

# Reconfigure with low priority questions
sudo dpkg-reconfigure -p low
