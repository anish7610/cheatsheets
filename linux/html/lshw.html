Linux Command: lshw

The lshw command is used to list detailed information about the hardware configuration of a Linux system.

Common Options

Option Description
-short Displays output in a short format.
-html Outputs information in HTML format.
-xml Generates output in XML format.
-json Produces output in JSON format.
-C [class] Shows specific hardware class (e.g., CPU, network).
-sanitize Removes sensitive information like serial numbers.

Usage Examples 

# List complete hardware information in short format
sudo lshw -short

# Get hardware details in HTML format
sudo lshw -html > hardware.html

# View CPU information only
sudo lshw -C CPU

# Show output without sensitive data
sudo lshw -sanitize

# Output the hardware configuration in JSON
sudo lshw -json > hardware.json
