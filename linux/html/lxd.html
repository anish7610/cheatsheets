LXD Command Guide

LXD is a next-generation system container manager.

Common Options

Option Description
init Initialize LXD with default settings or a configuration file.
launch Launch a new container from an image.
start Start an existing container.
stop Stop a running container.
list List all containers and their status.
delete Delete an existing container.
exec Execute a command inside a running container.

Usage Examples 


# Initialize LXD
lxd init

# Launch a new Ubuntu container
lxd launch ubuntu:20.04 my-container

# List all containers
lxd list

# Start a container
lxd start my-container

# Execute a command inside a container
lxd exec my-container -- /bin/bash

# Stop a container
lxd stop my-container

# Delete a container
lxd delete my-container

Cheat Sheet
