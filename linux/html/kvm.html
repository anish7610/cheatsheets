KVM Command Overview

Description

The kvm command is a part of QEMU and is used to launch virtual machines using Kernel-based Virtual Machine (KVM). It allows for efficient virtualization by leveraging hardware extensions for virtualization.

Common Options

Option Description
-m Allocate a specific amount of memory to the VM (e.g., -m 512M for 512 MB).
-smp Set the number of CPUs for the virtual machine.
-hda Specify the hard disk file to use as the primary drive.
-cdrom Use an image file as the CD-ROM.
-boot Specify boot order (e.g., -boot d for booting from CD-ROM).
-net Configure network settings for the VM.
-display Set the display option; often -display none for headless operation.

Usage Examples 

# Start a VM with 1 GB RAM, 2 CPUs and a specific disk image
kvm -m 1024 -smp 2 -hda /path/to/disk.img

# Boot a VM from a CD-ROM image in a non-graphical mode
kvm -cdrom /path/to/cdrom.iso -boot d -display none

# Start a VM with a network configuration
kvm -m 512 -hda /path/to/disk.img -net nic -net user

Cheat Sheet 


# Allocate memory: -m SIZE
# Set CPUs: -smp NUM
# Primary disk: -hda FILE
# Boot from CD-ROM: -cdrom FILE -boot d
# Network config: -net nic -net user
# Headless mode: -display none
