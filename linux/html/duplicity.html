Duplicity Command Overview

Duplicity is a Linux command-line utility for creating encrypted, bandwidth-efficient, incremental backups locally or remotely.

Common Options

Option Description
full Create a full backup.
incremental Create an incremental backup since the last backup.
restore Restore files from a backup.
--encrypted Encrypt the backup with a GPG key.
--sign-key Sign the backup with a GPG key.
--exclude Exclude specific files or directories from the backup.
--include Include only specified files or directories in the backup.

Usage Examples 



Full Backup

duplicity full /home/user file://var/backup


Incremental Backup

duplicity incremental /home/user file://var/backup


Restore Backup

duplicity restore file://var/backup /home/user


Exclude Files

duplicity --exclude /home/user/temp /home/user file://var/backup


Encrypted Backup

duplicity --encrypted --sign-key ABC123 /home/user remote://user@server/backup

Cheat Sheet 


duplicate full /source /dest
duplicate incremental /source /dest
duplicate restore /dest /source
duplicate --exclude /path/to/exclude /source /dest
duplicate --encrypted --sign-key KEYID /source /dest
