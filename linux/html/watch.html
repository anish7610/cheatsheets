Linux Command: watch

Description

The watch command in Linux is used to execute a program periodically, showing output in a fullscreen window.

Common Options

Option Description
-n seconds Specifies the interval in seconds between executions.
-d Highlights the differences between updates.
-t Turns off the header information at the top.
-p Precise mode, tries to run commands at exact intervals.
-g Stops the watch when output changes.

Usage Examples 

# Monitor disk usage every 2 seconds
watch -n 2 df -h

# Display and highlight changes in network connections every 5 seconds
watch -d -n 5 netstat -tuln

# Check the number of users logged in, without header
watch -t who

# Monitor the contents of a directory and leave upon change
watch -g ls -l

Cheat Sheet 

# Basic usage
watch command

# Set interval
watch -n seconds command

# Highlight differences
watch -d command
