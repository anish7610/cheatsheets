QEMU Command Overview

qemu is a generic and open-source machine emulator and virtualizer, allowing you to run programs for one type of machine on a different one.

Common Options

Option Description
-hda <file> Use the specified file as the first hard drive image.
-m <size> Set the guest RAM size, e.g., -m 512 for 512 MB.
-cdrom <file> Use the specified file as a virtual CD-ROM.
-boot order=<order> Set boot order, e.g., -boot order=dc for disk then CD-ROM.
-cpu <model> Select CPU model and features.
-enable-kvm Enable KVM full virtualization support.
-smp <cpus> Set the number of CPUs for the guest system.

Usage Examples 

# Start a VM with 1 GB RAM and a specified disk image
qemu-system-x86_64 -hda /path/to/disk.img -m 1024

# Boot from an ISO image with KVM support
qemu-system-x86_64 -cdrom /path/to/os.iso -enable-kvm

# Specify CPU model and multi-core support 
qemu-system-x86_64 -cpu host -smp 4 -hda /path/to/disk.img

# Boot with a specific order, disk then CD-ROM
qemu-system-x86_64 -boot order=dc -hda /path/to/disk.img -cdrom /path/to/os.iso

Cheat Sheet
