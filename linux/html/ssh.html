SSH Command Overview

Description

SSH (Secure Shell) is a command-line tool used for securely connecting to remote systems over a network. It provides encrypted communications for security.

Common Options

Option Description
-p Specify the port to connect to on the remote host.
-i Specify the identity file (private key) for public key authentication.
-l Specify the user to log in as on the remote system.
-N Create a connection without executing any command.
-T Disable pseudo-tty allocation.

Usage Examples 

# Connect to a remote server
ssh user@hostname

# Connect using a specific port
ssh -p 2222 user@hostname

# Use a specific private key
ssh -i /path/to/private_key user@hostname

# Run a command on the remote server
ssh user@hostname 'ls -l /var/www'

# Establish a tunnel
ssh -L 8080:localhost:80 user@hostname

Cheat Sheet 

# Basic connection
ssh user@host

# Specify port
ssh -p port user@host

# Identity file
ssh -i keyfile user@host

# Run remote command
ssh user@host 'command'
