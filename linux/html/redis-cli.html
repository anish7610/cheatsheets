redis-cli Command Overview

The redis-cli command-line interface is used to interact with Redis, a key-value store database. It allows users to execute commands, manage data, and retrieve information from a Redis server.

Common Options

Option Description
-h Specify the Redis server hostname (default: 127.0.0.1).
-p Specify the Redis server port (default: 6379).
-a Authenticate with the specified password.
-n Select the database number (default: 0).
-c Enable command line history and completion features.

Usage Examples 

# Connect to local Redis server
redis-cli

# Connect to Redis server with specific host and port
redis-cli -h myredisserver.com -p 6380

# Connect with authentication
redis-cli -a mypassword

# Select a different database
redis-cli -n 2

# Set and get a key-value pair
redis-cli set mykey "Hello"
redis-cli get mykey

Quick Reference
