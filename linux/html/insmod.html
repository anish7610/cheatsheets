Linux `insmod` Command

The insmod command is used to insert a module into the Linux kernel. It's typically employed to add functionality to the kernel, such as drivers or other extensions.

Common Options

Option Description
[MODULE] The module file to insert into the kernel (required).
[module_options] Optional parameters to pass to the module on insertion.

Usage Examples 

sudo insmod /path/to/module.ko

Insert a module named module.ko into the kernel. 

sudo insmod /path/to/module.ko option1=value1 option2=value2

Insert a module with parameters option1 and option2 set.

Cheat Sheet 


insmod [MODULE] [module_options]
