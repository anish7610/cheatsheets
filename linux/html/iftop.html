Linux Command: iftop

Description

iftop is a command-line tool that displays bandwidth usage on an interface by host. It provides a real-time overview of network traffic.

Common Options

Option Description
-i <interface> Select the network interface to monitor.
-n Display numerical addresses instead of resolving hostnames.
-b Display bandwidth in bytes instead of bits.
-f <filter code> Use a filter to select specific traffic to monitor (e.g., ‘host 192.168.1.1’).
-N Suppress DNS lookups for hostnames.

Usage Examples 

# Monitor traffic on the default interface
iftop

# Monitor traffic on eth0 in bytes
iftop -i eth0 -b

# Monitor traffic on eth0 without DNS lookups
iftop -i eth0 -n

# Use a filter to monitor traffic from a specific host
iftop -f "host 192.168.1.1"

Cheat Sheet 

iftop -i     # Select interface
iftop -n                # Disable hostname resolution
iftop -b                # Display bytes
iftop -f ""     # Apply a filter
