Linux Command: depmod

Description

The depmod command is used to generate modules' dependency files for the Linux kernel. It analyzes the kernel modules in the specified directory and outputs a dependency file.

Common Options

Option Description
-a , --all Consider all modules in the directory (default).
-n , --dry-run Perform a dry run without making any changes.
-v , --verbose Provide verbose output.
-F System.map Use the specified System.map file.
-b , --basedir DIR Use a directory as the base for module paths.

Usage Examples 

# Generate dependencies for all modules
depmod

# Verbose output for debugging purposes
depmod -v

# Perform a dry run to see changes without applying them
depmod -n

# Specify a different base directory for module paths
depmod -b /alt/modules

Cheat Sheet
