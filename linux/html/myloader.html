myloader Command in Linux

The myloader command is a multi-threaded tool used for restoring MySQL database dumps created by mydumper .

Common Options

Option Description
-d , --directory Specify the directory of the dump files created by mydumper.
-h , --host MySQL database host to connect to.
-u , --user Username for connecting to the database.
-p , --password Password for the database user.
-P , --port Port number for connecting to the database.
-t , --threads Number of threads to use for restoring the dump.
-B , --database Restore only the specified database.
--overwrite-tables Overwrite existing tables during restore.

Usage Examples 

# Restore a dump using 4 threads
myloader -d /path/to/dump -h localhost -u user -p password -t 4

# Restore a specific database
myloader -d /path/to/dump -h localhost -u user -p password -B database_name

# Restore and overwrite existing tables
myloader -d /path/to/dump -h localhost -u user -p password --overwrite-tables

Cheat Sheet 

myloader -d [dir] -h [host] -u [user] -p [password] -t [threads]
myloader -B [db]
myloader --overwrite-tables
