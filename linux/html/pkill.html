Linux `pkill` Command

The pkill command is used to terminate processes based on name, user, group, or other attributes. It's an advanced tool for process management.

Common Options

Option Description
-u Match processes by the effective user ID.
-g Match processes by process group ID.
-x Match the exact process name.
-c Display the number of processes matched.
-signal Send a specific signal to the processes (e.g., -9 for SIGKILL).

Usage Examples

Terminate All Processes Named "firefox" 

pkill firefox

Terminate All Processes for a Specific User "john" 

pkill -u john

Send SIGKILL to Processes Named "gedit" 

pkill -9 gedit

Restrict Termination to Exact Match for "bash" 

pkill -x bash
