Linux Command: aws

Description

The aws command-line interface (CLI) is a unified tool to manage your AWS services. With just one tool, you can control multiple AWS services from the command line and automate them through scripts.

Common Options

Option Description
--region Specify the AWS region to interact with.
--output Set the format for command output (e.g., json, text, table).
--profile Select a specific profile to use from the AWS credentials file.
--help Show help for any command or subcommand.

Usage Examples 

# List all S3 buckets
aws s3 ls

# Start an EC2 instance
aws ec2 start-instances --instance-ids i-0abcd1234efgh5678

# Describe RDS instances with JSON output
aws rds describe-db-instances --output json

# Configure AWS CLI with a wizard
aws configure

# Use a specific profile to list S3 buckets in a region
aws s3 ls --region us-west-2 --profile myprofile
