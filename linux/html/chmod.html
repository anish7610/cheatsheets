Linux `chmod` Command

The `chmod` command changes the file mode (permissions) of files and directories.

Common Options

Option Description
+r Add read permission
+w Add write permission
+x Add execute permission
-r Remove read permission
-w Remove write permission
-x Remove execute permission
u User (owner) permissions
g Group permissions
o Other users' permissions
a All users' permissions

Usage Examples 

# Give the owner executable permission on a script
chmod u+x script.sh

# Remove write permission from others on a file
chmod o-w file.txt

# Allow everyone to read a document
chmod a+r document.pdf

# Change permissions for a directory recursively
chmod -R 755 /path/to/directory

Cheat Sheet 

chmod ### file.txt      # Set numeric permissions
chmod u+x file.txt     # Add execute for owner
chmod g-w file.txt     # Remove write for group
chmod o-r file.txt     # Remove read for others
chmod a+w file.txt     # Add write for all
