Linux uname Command

Description

The uname command prints system information about the current machine and operating system.

Common Options

Option Description
-a Displays all system information.
-s Prints the kernel name.
-n Displays the network node (hostname).
-r Shows the kernel release.
-v Displays the kernel version.
-m Prints the machine hardware name.
-p Displays the processor type (if available).
-i Shows the hardware platform (if available).
-o Prints the operating system.

Usage Examples 

# Display all system information
uname -a

# Display the system's kernel name
uname -s

# Print the hostname
uname -n

# Show the kernel release
uname -r

Cheat Sheet 


uname -a   # All information
uname -s   # Kernel name
uname -n   # Hostname
uname -r   # Kernel release
uname -v   # Kernel version
