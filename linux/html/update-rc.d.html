Understanding `update-rc.d`

`update-rc.d` is a command used in Debian-based Linux systems to manage the start and stop behavior of services at boot and shutdown.

Common Options

Option Description
defaults Automatically set the default start/stop links for a service.
remove Remove all start/stop links for a service.
start RUNLEVEL Set up start links for specific runlevels.
stop RUNLEVEL Set up stop links for specific runlevels.

Usage Examples 

sudo update-rc.d apache2 defaults

Enable Apache2 service to start on boot with default runlevels. 

sudo update-rc.d apache2 remove

Remove Apache2 service from all runlevels. 

sudo update-rc.d apache2 start 2 3 4 5 . stop 0 1 6 .

Configure Apache2 to start in runlevels 2, 3, 4, and 5, and stop in 0, 1, and 6.

Quick Reference Cheat Sheet 

sudo update-rc.d <service> defaults
sudo update-rc.d <service> remove
sudo update-rc.d <service> start <runlevels> . stop <runlevels> .
