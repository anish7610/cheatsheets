Linux `search` Command

Description

The search command is not a standard command in Linux. You might mean grep , which is used to search text using patterns within files. It outputs lines that match a specified pattern.

Common Options

Option Description
-i Ignore case distinctions in patterns and data.
-r Recursively search directories.
-n Display line numbers with output lines.
-l Show filenames only, not the matching text.
-v Invert the match, displaying non-matching lines.
-c Count the number of matches.

Usage Examples 

# Search for the word 'example' in a file
grep 'example' filename.txt

# Case-insensitive search for 'hello' in all .txt files
grep -i 'hello' *.txt

# Recursively search for 'error' in all files in a directory
grep -r 'error' /path/to/directory

# Display line numbers for matches in a file
grep -n 'todo' filename.txt

# Count occurrences of 'success' in a log file
grep -c 'success' logfile.log

Cheat Sheet 


grep 'pattern' file        # Basic usage
grep -i 'pattern' file     # Ignore case
grep -r 'pattern' dir      # Recursive search
grep -n 'pattern' file     # Show line numbers
grep -l 'pattern' *        # List files with matches
