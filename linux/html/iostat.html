iostat Command Overview

Description

The iostat command reports Central Processing Unit (CPU) and input/output statistics for devices and partitions, helping to monitor system performance and identify bottlenecks.

Common Options

Option Description
-c Display only the CPU usage report.
-d Display only device utilization report.
-x Display extended statistics.
-m Display statistics in megabytes per second.
-k Display statistics in kilobytes per second.
-t Add a timestamp to each report line.

Usage Examples 

# Display CPU and device statistics every 2 seconds
iostat 2

# Display extended I/O statistics for devices
iostat -x

# Display I/O statistics in megabytes with a timestamp
iostat -m -t

# Display CPU statistics only
iostat -c

# Display device statistics in kilobytes every 1 second
iostat -dk 1
