Linux `w` Command Guide

Description

The w command displays information about users currently logged into the system and their activities.

Common Options

Option Description
-h Omit the header in the output.
-s Short format, skipping the login name and from fields.
-f Toggle printing of the from (remote hostname) field.
-i Display IP addresses instead of hostnames.
-u Ignore the idle time.

Usage Examples 

# Display all user activities
w

# Display user activities without header
w -h

# Display in shorter format
w -s

# Display with IP addresses instead of hostnames
w -i

Cheat Sheet 

w       # Show all user activities
w -h    # No header
w -s    # Short format
w -i    # Show IP addresses
