Linux Command: i18n

Description

The i18n command in Linux is associated with internationalization, managing locale settings for software applications to support multiple languages and regions more efficiently.

Common Options

Option Description
locale Display information about the current locale settings.
export LANG= locale Set the default locale for applications.
localectl Query or change system locale and keyboard layout settings.
locale-gen Generate locales based on system configuration.

Usage Examples 

# Display currently set locale
locale

# Set the default system language to US English
export LANG=en_US.UTF-8

# Generate missing locale data
sudo locale-gen

# Check current system locale setting
localectl status

Cheat Sheet
