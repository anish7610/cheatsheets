Firewalld Command Overview

Firewalld is a firewall management tool for Linux operating systems that provides a dynamically managed firewall with support for network zones.

Common Options

Option Description
--state Check whether the firewalld is running.
--reload Reload firewalld rules without losing current connections.
--add-service=SERVICE Add a service to the default zone.
--remove-service=SERVICE Remove a service from the default zone.
--list-all List all settings for the default zone.

Usage Examples 

# Check if firewalld is running
firewalld --state

# Reload firewalld rules
firewalld --reload

# Add the http service to the default zone
firewalld --add-service=http

# Remove the ssh service from the default zone
firewalld --remove-service=ssh

# List all settings for the default zone
firewalld --list-all

Cheat Sheet 

Check state: firewalld --state
Reload rules: firewalld --reload
Add service: firewalld --add-service=SERVICE
Remove service: firewalld --remove-service=SERVICE
List settings: firewalld --list-all
