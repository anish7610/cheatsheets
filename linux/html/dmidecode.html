Linux Command: dmidecode

The dmidecode command retrieves detailed information about a computer's hardware, as described in the system's DMI (Desktop Management Interface) table.

Common Options

Option Description
-t, --type Filter output by hardware type, e.g., 'memory', 'processor'.
-s, --string Show specific information, e.g., 'bios-vendor'.
-q, --quiet Less verbose output.
-u, --dump Dump the DMI data in a binary format.
-h, --help Display help information.

Usage Examples 

# Get BIOS information
dmidecode -t bios

# Get processor details
dmidecode -t processor

# Retrieve memory device details
dmidecode -t memory

# Show system manufacturer
dmidecode -s system-manufacturer
