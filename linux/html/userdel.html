Linux Command: userdel

The userdel command is used to delete a user account and related files from a Linux system.

Common Options

Option Description
-r Remove home directory and mail spool.
-f Force removal of user account, even if currently logged in.

Usage Examples 

# Delete a user without removing home directory
sudo userdel username

# Delete a user and their home directory
sudo userdel -r username

# Force delete a user who is currently logged in
sudo userdel -f username

Cheat Sheet 

# userdel username        (delete user)
# userdel -r username     (delete user + home dir)
# userdel -f username     (force delete)
