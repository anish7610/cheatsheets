Restic Command Guide

Short Description

Restic is a fast, secure, and efficient backup tool that ensures your data is reliably saved and restored.

Common Options

Option Description
init Initialize a new repository.
backup [directory] Create a backup of the specified directory.
restore [snapshotID] Restore the specified snapshot.
snapshots List all snapshots in the repository.
forget Remove snapshots based on a policy.
prune Remove unneeded data from the repository.
check Verify the integrity of the repository.

Usage Examples 

restic init --repo /path/to/repo
restic backup /home/user/documents
restic restore 12345678 --target /restored/files
restic snapshots --repo /path/to/repo
restic forget --keep-last 5 --prune
restic check --repo /path/to/repo
