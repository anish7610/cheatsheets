Linux Command: kmod

The kmod command is used to manage Linux kernel modules. It can insert, remove, list, and manage dependencies of modules in the kernel.

Common Options

Option Description
--help Display help information and exit.
--version Show version information and exit.
list List all currently loaded modules.
load module_name Load a specified module into the kernel.
remove module_name Remove a specified module from the kernel.

Usage Examples 

# List all loaded modules
kmod list

# Load a module named 'dummy'
kmod load dummy

# Remove a module named 'dummy'
kmod remove dummy

Cheat Sheet 

# List modules
kmod list

# Load module
kmod load [module_name]

# Remove module
kmod remove [module_name]
