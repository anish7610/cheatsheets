Linux `locales` Command

Description

The locales command displays information about the current locale settings in Linux, which define region-specific formats for dates, times, numbers, and more.

Common Options

Option Description
-a List all available locales.
-m List available charmaps.

Usage Examples 

# List all available locales
locales -a

# Display available charmaps
locales -m

# Check current locale settings
locale
