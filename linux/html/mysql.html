MySQL Command Overview

The mysql command-line tool allows users to execute MySQL queries, manage databases, and perform administrative tasks.

Common Options

Option Description
-u , --user Specify the MySQL username
-p , --password Prompt for the database password
-h , --host Connect to a specific host
-P , --port Specify the port number
-e , --execute Execute a SQL statement directly
--database , -D Select the database to use

Usage Examples 

# Connect to a MySQL server
mysql -u username -p

# Execute a SQL command
mysql -u username -p -e "SHOW DATABASES;"

# Connect to a specific database on a remote server
mysql -u username -p -h hostname -D database_name

# Import a SQL file into a database
mysql -u username -p -D database_name < file.sql

Cheat Sheet 

mysql -u user -p            # Connect and prompt for password
mysql -h host -P port        # Specify host and port
mysql -e "SQL"               # Execute SQL statement
