Linux Command: netstat

The netstat command is used to display network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.

Common Options

Option Description
-a Show all sockets (listening and non-listening).
-t Display TCP connections.
-u Display UDP connections.
-l Show only listening sockets.
-n Show numerical addresses instead of resolving hosts.
-p Show the PID and name of the program to which each socket belongs.
-r Display the routing table.
-s Display network statistics (like recv/send counts).

Usage Examples 

# Display all connections and listening ports
netstat -a

# Show TCP connections only
netstat -t

# Show listening TCP sockets with numerical addresses
netstat -ltun

# Display UDP connections with PID and program name
netstat -up

# Show routing table
netstat -r

# Display network statistics
netstat -s
