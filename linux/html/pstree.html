Linux `pstree` Command Guide

Description

The pstree command displays running processes as a tree, showing the parent-child relationships between them.

Common Options

Option Description
-p Show PIDs of the processes.
-n Sort processes by PID.
-a Show command line arguments.
-u Show user IDs.
-h Highlight the process tree of the current user.

Usage Examples 

# Display the process tree with PIDs
pstree -p

# Display the process tree sorted by PID
pstree -pn

# Show command line arguments with process tree
pstree -a

# Highlight the current user's process tree
pstree -h

# Show user IDs in the process tree
pstree -u

Cheat Sheet 

pstree -p  # Show PIDs
pstree -n  # Sort by PID
pstree -a  # Show arguments
pstree -u  # Show user IDs
pstree -h  # Highlight user tree
