passwd Command Guide

Description

The passwd command is used to change user passwords in Linux. It can also update password attributes and lock or unlock user accounts.

Common Options

Option Description
-l Lock the specified user's account.
-u Unlock the specified user's account.
-d Delete the user's password (disable account access).
-S Display account status information.
--stdin Read password from standard input (useful for scripting).

Usage Examples 


# Change the current user's password
passwd

# Lock a user's account
passwd -l username

# Unlock a user's account
passwd -u username

# Delete a user's password
passwd -d username

# Display user account status
passwd -S username

# Change password using a script
echo "newpassword" | passwd --stdin username

Cheat Sheet 


passwd             # Change password
passwd -l user     # Lock account
passwd -u user     # Unlock account
passwd -d user     # Delete password
passwd -S user     # Show account status
