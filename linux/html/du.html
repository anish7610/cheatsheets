Linux `du` Command

The du command estimates and displays the disk usage of files and directories.

Common Options

Option Description
-h Show sizes in human-readable format (e.g., KB, MB).
-s Display only the total size for each argument.
-a Include individual files in the output.
-c Display a grand total at the end.
--max-depth=N Limit the output to directories up to depth N.

Usage Examples 

# Display disk usage of all files and directories in a human-readable format
du -h

# Display total size of a specific directory
du -sh /path/to/directory

# Show disk usage including individual files
du -ah

# Display disk usage with a grand total
du -ch

# Limit the output to top-level directories
du -h --max-depth=1

Cheat Sheet
