OpenStack Command

The openstack command is a command-line client for interacting with OpenStack services. It allows users to manage resources like instances, networks, volumes, and more in an OpenStack cloud environment.

Common Options

Option Description
--os-cloud <cloud> Specifies the cloud configuration to use.
--os-project-name <project> Sets the project name for the command.
--os-username <username> Specifies the username for authentication.
--os-password <password> Sets the password for authentication.
--os-auth-url <auth url> Provides the authentication URL.
--debug Enables debug output.

Usage Examples 

# List all instances
openstack server list

# Create a new instance
openstack server create --flavor m1.small --image cirros --network private my-instance

# Delete an instance
openstack server delete my-instance

# List available flavors
openstack flavor list

# List all networks
openstack network list
