Linux `semanage` Command

Description

The semanage command is used to configure SELinux (Security-Enhanced Linux) policy components, including contexts, ports, and file labeling.

Common Options

Option Description
-a , --add Add a new object
-d , --delete Delete an object
-m , --modify Modify an object
-l , --list List all objects
-n , --noheading Omit headings in output
-C , --locallist Display local customizations

Examples 

# Add a new port for http service
semanage port -a -t http_port_t -p tcp 8080

# Delete a custom SELinux context for a file
semanage fcontext -d '/custom/path(/.*)?'

# List all defined SELinux users
semanage user -l

# Modify a port definition for ssh
semanage port -m -t ssh_port_t -p tcp 2022

Cheat Sheet 

Add port: semanage port -a -t TYPE -p PROTOCOL PORT
Delete fcontext: semanage fcontext -d 'PATH'
List users: semanage user -l
Modify port: semanage port -m -t TYPE -p PROTOCOL PORT
