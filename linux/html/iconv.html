Iconv Command Guide

Description

The iconv command is used to convert text from one character encoding to another.

Common Options

Option Description
-f , --from-code Specifies the source encoding.
-t , --to-code Specifies the target encoding.
-o , --output Sends output to a file instead of standard output.

Usage Examples

Convert a File from ISO-8859-1 to UTF-8 

iconv -f ISO-8859-1 -t UTF-8 input.txt -o output.txt

Convert and Display Directly to Terminal 

iconv -f UTF-16 -t UTF-8 input.txt

Convert a File with Output to Another File 

iconv --from-code=ISO-8859-1 --to-code=UTF-8 < input.txt > output.txt

Cheat Sheet 


# Convert file encoding
iconv -f FROM_ENCODING -t TO_ENCODING input.txt -o output.txt

# Display converted content directly
iconv -f FROM_ENCODING -t TO_ENCODING input.txt

# List available encodings
iconv -l
