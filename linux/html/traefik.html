Traefik Command Overview

traefik is an open-source edge router that makes deploying microservices easy. It is designed to automatically manage the routing of requests and integrates seamlessly with various service discovery backends.

Common Options

Option Description
-c, --configFile Specifies the path to the Traefik configuration file.
--log.level Sets the log level (e.g., "DEBUG", "INFO").
--api Enables the Traefik API and monitoring dashboard.
--providers.docker Enables Docker as a provider for service discovery.
--entrypoints.web.address Defines the address to listen on for the web entry point.

Usage Examples 

# Start Traefik with a specific configuration file
traefik -c /path/to/traefik.yml

# Run Traefik with Docker provider and web entrypoint
traefik --providers.docker --entrypoints.web.address=:80

# Enable API and set log level to DEBUG
traefik --api --log.level=DEBUG

Cheat Sheet 

traefik -c <configFile>         # Set config file
traefik --log.level=DEBUG          # Set log level
traefik --api                      # Enable API/dashboard
traefik --providers.docker         # Use Docker for discovery
traefik --entrypoints.web.address  # Listen on web port
