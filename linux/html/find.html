Understanding the Linux find Command

Description

The find command is used in Linux to search for files and directories within a directory hierarchy. It offers multiple options to refine and filter search results based on specific criteria.

Common Options

Option Description
-name Searches for files by name.
-type Filters search by file type (e.g., f for files, d for directories).
-size Finds files by size (e.g., +1M for larger than 1 MB).
-mtime Searches for files modified within a certain number of days.
-exec Executes a command on each found file.

Usage Examples 

# Find all .txt files in the current directory and subdirectories
find . -name "*.txt"

# Find all directories named 'backup'
find / -type d -name "backup"

# Find all files larger than 100MB
find / -size +100M

# Find files modified in the last 7 days
find . -mtime -7

# Find and remove all files named 'tmp.log'
find . -name "tmp.log" -exec rm {} \;

Cheat Sheet 

find [path] [expression]
-name "pattern"      # Search by name
-type [f|d]         # Search by type (file or dir)
-size [+|-]N[c|k|M] # Search by size
-mtime [+|-]N       # Search by modification time
-exec command {} \; # Execute command
