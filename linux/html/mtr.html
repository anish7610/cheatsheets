Linux Command: mtr

Description

mtr (My Traceroute) is a network diagnostic tool that combines the functionality of ping and traceroute . It provides continuous, real-time analysis of the route packets take to a remote host.

Common Options

Option Description
-r Generate a report without interactive output.
-c <count> Set the number of cycles mtr will run.
-s <packetsize> Specify the packet size to use.
--report-wide Generate a wide-format report.
--xml Output results in XML format.

Usage Examples 

# Interactive traceroute to a host
mtr example.com

# Generate a report for 10 cycles
mtr -r -c 10 example.com

# Set packet size to 128 bytes and run 5 cycles
mtr -c 5 -s 128 example.com

# Generate a wide-format report
mtr --report-wide example.com

# Output in XML format for 8 cycles
mtr -r -c 8 --xml example.com
