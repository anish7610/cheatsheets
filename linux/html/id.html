Linux `id` Command

Description

The id command is used to display the user and group information for the specified user or the current user if no username is provided.

Common Options

Option Description
-u Display only the user ID.
-g Display only the group ID.
-G Display all group IDs.
-n Display the name instead of numeric IDs (used with -u , -g , or -G ).
--help Display help message and exit.

Usage Examples

Display Current User Information 

id

Display Only User ID for Current User 

id -u

Display Only Group ID for Current User 

id -g

Display All Group IDs for Current User 

id -G

Display User and Group Names for a Specific User 

id -nu

Cheat Sheet 


id          # Show user and group info
id -u       # Show only user ID
id -g       # Show only group ID
id -G       # Show all group IDs
id -nu      # Show user name
id -ng      # Show group name
