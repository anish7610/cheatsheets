Linux `chown` Command

The chown command changes the owner and group of files and directories.

Common Options

Option Description
-R Recursively change ownership of directories and their contents.
--reference=file Use the owner and group of the specified file.
-v Output a diagnostic for every file processed.
-c Like verbose but reports only when a change is made.

Usage Examples 

      
# Change owner to 'user' and group to 'group' for a file
chown user:group filename

# Recursively change owner and group for a directory
chown -R user:group directory/

# Change owner to match that of another file
chown --reference=referencefile targetfile

# Verbosely change owner to 'user'
chown -v user filename

Cheat Sheet 

      
chown user:group file          # Change owner and group
chown -R user:group dir/       # Recursive change
chown --reference=file target  # Use reference
chown -v user file             # Verbose output
