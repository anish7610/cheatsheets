Linux `dump` Command

Description

The `dump` command is used for backing up file systems on Unix/Linux systems. It creates a backup of an entire filesystem on a storage device or file.

Common Options

Option Description
-0 Perform a full backup.
-1, -2, ..., -9 Perform an incremental backup of the specified level.
-f [file] Write the backup to the specified file.
-u Update the dump record file after completing the backup.
-v Enable verbose mode, displaying detailed progress.

Usage Examples 

# Full backup of /dev/sda1 to a file
dump -0 -f /backup/full_backup.dump /dev/sda1

# Incremental backup (level 1) of /dev/sda1
dump -1 -f /backup/incremental_backup.dump /dev/sda1

# Full backup with verbose output and update record
dump -0 -u -v -f /backup/full_backup_with_record.dump /dev/sda1

Cheat Sheet 

Full backup:      dump -0 -f [file] [filesystem]
Incremental:     dump -[1-9] -f [file] [filesystem]
Update record:   dump -0 -u -f [file] [filesystem]
Verbose:         dump -0 -v -f [file] [filesystem]
