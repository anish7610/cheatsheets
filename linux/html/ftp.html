FTP Command in Linux

The ftp command is used to transfer files to and from a remote network server using the File Transfer Protocol.

Common Options

Option Description
-v Enables verbose mode, showing all responses from the FTP server.
-n Suppresses automatic login upon initial connection.
-i Disables interactive prompting during multiple file transfers.
-g Disables filename globbing.
-d Enables debugging mode.

Usage Examples 

# Connect to an FTP server
ftp ftp.example.com

# Upload a file to the FTP server
ftp> put localfile.txt remotedir/remotefile.txt

# Download a file from the FTP server
ftp> get remotedir/remotefile.txt localfile.txt

# Enable passive mode and download a file
ftp -p ftp.example.com
ftp> get remotefile.txt

# Login without automatic login prompt
ftp -n ftp.example.com
ftp> user username
ftp> password

Cheat Sheet 

# Connect and upload
ftp [-v] [-n] [-i] [-g] server
