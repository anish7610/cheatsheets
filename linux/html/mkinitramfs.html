mkinitramfs Command

The mkinitramfs command is used to create an initial RAM filesystem (initramfs) in Linux. This is essential for booting the operating system, especially during system initialization.

Common Options

Option Description
-o <file> Specifies the output file for the initramfs image.
-k Keeps the temporary files used for creating the image.
-v Enables verbose mode, providing detailed output.
-d Specifies a different directory for temporary files.
-r Include resume hooks for suspend-to-disk functionality.

Usage Examples 

# Create a basic initramfs image
sudo mkinitramfs -o /boot/initramfs.img

# Create an initramfs image with verbose output
sudo mkinitramfs -o /boot/initramfs.img -v

# Create an initramfs image and keep temporary files
sudo mkinitramfs -o /boot/initramfs.img -k

Cheat Sheet 

# Basic usage
sudo mkinitramfs -o <output-image>

# With verbose output
sudo mkinitramfs -v -o <output-image>

# Keep temporary files
sudo mkinitramfs -k -o <output-image>
