Linux `upgrade` Command

Description

The `upgrade` command is often used with package managers like apt to update installed packages to their latest versions.

Common Options

Option Description
-y Automatically answer 'yes' to prompts
--dry-run Show what would be upgraded, but do not actually upgrade
--with-new-pkgs Install new packages in addition to upgrading existing ones
--only-upgrade Do not install new packages, only upgrade existing ones

Usage Examples 

# Upgrade all installed packages
sudo apt upgrade

# Upgrade all packages and assume "yes" to prompts
sudo apt upgrade -y

# Simulate an upgrade to see what will change
sudo apt upgrade --dry-run

# Upgrade all packages and include installing new packages
sudo apt upgrade --with-new-pkgs

# Only upgrade existing packages, do not install new ones
sudo apt upgrade --only-upgrade

Cheat Sheet

sudo apt upgrade
