smartctl Command Guide

Description

smartctl is a command-line utility to control and monitor storage systems using the SMART (Self-Monitoring, Analysis and Reporting Technology) system built into most modern ATA/SATA/SCSI/NVMe drives.

Common Options

Option Description
-i Show device information.
-a Display all SMART information for the device.
-H Show SMART health status.
-t short Initiate a short self-test.
-t long Initiate a long self-test.
-l error Display the device error log.
-x Show extended SMART information.

Usage Examples

Show Device Information 

smartctl -i /dev/sdX

Check Overall Health Status 

smartctl -H /dev/sdX

Display All SMART Information 

smartctl -a /dev/sdX

Run a Short Self-Test 

smartctl -t short /dev/sdX

View Error Log 

smartctl -l error /dev/sdX

Cheat Sheet
