Linux `lsof` Command

Description

The lsof command lists open files and the processes that opened them. It's useful for identifying files that are open and their associated processes.

Common Options

Option Description
-i List files opened by network connections.
-u <username> Show open files by a specific user.
-p <pid> Display files opened by a specific process ID.
+D <directory> List files under the specified directory.
-t Only display process IDs.

Usage Examples 

# List all network connections
lsof -i

# Show files opened by user 'john'
lsof -u john

# Display files opened by process with PID 1234
lsof -p 1234

# List all files under /var/log
lsof +D /var/log

# Show only process IDs of open files
lsof -t

Cheat Sheet 

lsof -i                # Network connections
lsof -u      # Files by user
lsof -p           # Files by PID
lsof +D     # Files in directory
lsof -t                # Only PIDs
