Linux Command: who

The who command in Linux displays information about users who are currently logged into the system.

Common Options

Option Description
-a Display all information, including empty entries.
-b Show the last system boot time.
-m Show information about the current terminal session.
-q Quickly display only the username and the number of users logged in.
-u Display idle time for each user.

Usage Examples 

# Display all logged-in users
who

# Show all information about logged-in users
who -a

# Display when the system was last booted
who -b

# Show current terminal session information
who -m

# Quickly show usernames and total count
who -q

# Include idle time for each user
who -u

Cheat Sheet 

who          # Show logged-in users
who -a       # All details
