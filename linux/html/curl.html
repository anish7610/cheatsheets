Linux `curl` Command

curl is a command-line tool used to transfer data to or from a server, using various protocols such as HTTP, HTTPS, FTP, and more.

Common Options

Option Description
-o <file> Write output to <file> instead of stdout
-O Save the file using remote file name
-L Follow redirects
-I Show HTTP headers only
-u <user:password> Server user and password authentication
-d <data> Send specified data in a POST request
-H <header> Pass custom header(s) to the server
-X <command> Specify the request command to use, like GET, POST

Usage Examples 

# Download a file
curl -O https://example.com/file.txt

# Save output to a file
curl -o output.txt https://example.com

# Follow redirects
curl -L https://example.com

# Send a POST request with data
curl -d "name=John&age=30" https://example.com/form

# Access a URL with user authentication
curl -u username:password https://example.com

# Custom header with GET
curl -H "Authorization: Bearer token" https://api.example.com/resource

# Display only headers of a URL
curl -I https://example.com

Cheat Sheet 

curl -o <file> -O -L -I -u <user:pass> -d <data> \
  -H <header> -X <command> https://example.com
