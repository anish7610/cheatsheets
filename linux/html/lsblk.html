Linux Command: lsblk

Description

The lsblk command lists block devices, showing detailed information about them. It provides a tree-like structure of all available or specified block devices.

Common Options

Option Description
-a Include empty devices.
-f Display file system information.
-l Use a list format instead of the default tree view.
-m Show owner, group, and mode in the listing.
-n Do not print the header line.
-o Specify output columns to display.
-r Print in raw format.
-t Print topology information.

Usage Examples

Basic Usage 

lsblk

Show File System Information 

lsblk -f

List Devices in a Flat Format 

lsblk -l

Include Empty Devices 

lsblk -a

Cheat Sheet 


lsblk         # List all block devices in tree format
lsblk -f      # Include file system information
lsblk -l      # Use list format
lsblk -a      # Include empty devices
