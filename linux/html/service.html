Linux Service Command

The service command is used to manage and control services on a Linux system. It allows starting, stopping, and restarting services, among other functions.

Common Options

Option Description
start Starts the specified service.
stop Stops the specified service.
restart Restarts the specified service.
status Shows the current status of the service.
reload Reloads the service configuration without stopping it.

Usage Examples 

# Start a service
service apache2 start

# Stop a service
service apache2 stop

# Restart a service
service nginx restart

# Check the status of a service
service ssh status

# Reload the configuration of a service
service apache2 reload

Cheat Sheet 

# Use "service [service_name] [option]" format

service [name] start    # Start service
service [name] stop     # Stop service
service [name] restart  # Restart service
service [name] status   # Status of service
service [name] reload   # Reload service
