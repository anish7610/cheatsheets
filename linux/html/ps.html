Linux `ps` Command

The ps command is used to display information about active processes on a system.

Common Options

Option Description
-e Show all processes.
-f Show full listing.
-u Show processes for a specific user.
-a Show processes for all users.
-x Include processes without a controlling terminal.
-l Display a long format.

Usage Examples 

# List all processes
ps -e

# List all processes with detailed information
ps -ef

# Show processes for a specific user (e.g., user 'john')
ps -u john

# Show all processes including those without a terminal
ps -ax

# Display processes with long format
ps -l

Cheat Sheet 

ps -e      # All processes
ps -ef     # Full format
ps -u user # User's processes
ps -ax     # All, no terminal
ps -l      # Long format
