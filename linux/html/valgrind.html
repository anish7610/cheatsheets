Valgrind

Valgrind is a programming tool for memory debugging, memory leak detection, and profiling.

Common Options

Option Description
--tool=memcheck Default tool for detecting memory errors.
--leak-check=yes Enable detailed memory leak detection.
--track-origins=yes Track the origins of uninitialized memory.
--log-file= file Write the log output to a specified file.
--show-reachable=yes Show reachable memory in the leak report.

Usage Examples 

# Basic memory check
valgrind ./myprogram

# Check for memory leaks with detailed info
valgrind --leak-check=yes ./myprogram

# Track origins of uninitialized memory
valgrind --track-origins=yes ./myprogram

# Write output to a log file
valgrind --log-file=valgrind-out.txt ./myprogram

Cheat Sheet 

valgrind --tool=memcheck [options] ./program
--leak-check=yes : Detailed leak info
--track-origins=yes : Track uninit memory
--log-file=file : Log output to file
