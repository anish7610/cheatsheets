Logger Command Overview

The logger command provides a shell interface to the system log service, allowing you to log messages from the command line or scripts.

Common Options

Option Description
-p Specify the priority of the message (e.g., user.info ).
-t Tag the message with a specified identifier.
-f Log the contents of a file.
--id Log the process ID with the message.

Usage Examples 

# Log a simple message
logger "System maintenance scheduled at midnight."

# Log a message with a specific priority
logger -p local0.notice "A custom notice from local program."

# Log a message with a tag
logger -t myscript "Script executed successfully."

# Log the contents of a file
logger -f /path/to/logfile

# Log a message with the process ID
logger --id "Process finished successfully."

Quick Cheat Sheet 

logger "message"                 # Log a simple message
logger -p user.info "message"    # Log with priority
logger -t tag "message"          # Log with tag
logger -f file                   # Log contents of a file
logger --id "message"            # Log with process ID
