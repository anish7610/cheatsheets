Linux Command: lsmod

Description

The lsmod command lists the currently loaded kernel modules in Linux.

Common Options

Option Description
--help Display help information.
--version Displays version information.

Usage Examples 

# List all loaded kernel modules
lsmod

# Display version information
lsmod --version

Cheat Sheet 

# List modules
lsmod

# Help
lsmod --help

# Version
lsmod --version
