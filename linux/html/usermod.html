Linux `usermod` Command

The usermod command is used to modify user account properties in Linux systems.

Common Options

Option Description
-l new_login Change the username of the account.
-d home_dir Change the user's home directory.
-m Move the contents of the home directory to a new location.
-G group1,group2,... Set additional groups for the user.
-a Append the user to the supplementary groups.
-s shell Change the user's login shell.
-L Lock the user account.
-U Unlock the user account.

Usage Examples 

# Change the username from 'oldname' to 'newname'
usermod -l newname oldname

# Change the home directory to '/home/newdir' and move contents
usermod -d /home/newdir -m username

# Add the user to 'sudo' group
usermod -a -G sudo username

# Change the user's login shell to '/bin/zsh'
usermod -s /bin/zsh username

# Lock the user account
usermod -L username

# Unlock the user account
usermod -U username

Cheat Sheet 

-l  Change username
-d  Change home directory
-m  Move home directory contents
-G  Set additional groups
-a  Append to groups
-s  Change login shell
-L  Lock account
-U  Unlock account
