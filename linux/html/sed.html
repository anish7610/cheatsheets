Understanding the `sed` Command

sed is a stream editor used to perform basic text transformations on an input stream (a file or input from a pipeline).

Common Options

Option Description
-e Add the script to the commands to be executed.
-i Edit files in place (overwrite).
-n Suppress automatic printing of pattern space.
-f Read commands from a file.
-r Use extended regular expressions.

Usage Examples 

# Replace 'old' with 'new' in a file and print to stdout
sed 's/old/new/' file.txt

# Replace 'old' with 'new' in a file in-place
sed -i 's/old/new/' file.txt

# Print lines 1 to 10 from a file
sed -n '1,10p' file.txt

# Delete lines containing a specific word
sed '/word/d' file.txt

# Read commands from a script file
sed -f script.sed file.txt

Cheat Sheet 

# Basic substitution
sed 's/find/replace/'

# In-place editing
sed -i 's/find/replace/'

# Print specific lines
sed -n 'Np'

# Delete lines
sed '/pattern/d'
