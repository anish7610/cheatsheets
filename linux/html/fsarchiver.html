fsarchiver

Description

fsarchiver is a tool used to create backups of file systems as archives. It supports a variety of file systems and can perform compression and encryption.

Common Options

Option Description
savefs Creates an archive of a file system.
restfs Restores a file system from an archive.
-z, --compress Sets compression level (1-9).
-e, --encrypt Encrypts the archive with a password.
-j, --jobs Specifies number of threads for compression.
-v, --verbose Displays detailed operation messages.

Examples 

# Backup a file system to an archive
fsarchiver savefs /backup/system.fsa /dev/sda1

# Restore a file system from an archive
fsarchiver restfs /backup/system.fsa id=0,dest=/dev/sda1

# Backup with compression and verbosity
fsarchiver savefs -z 3 -v /backup/compressed.fsa /dev/sda2

# Encrypt and backup a file system
fsarchiver savefs -e password123 /backup/encrypted.fsa /dev/sda3

Cheat Sheet 

fsarchiver savefs [options] archive_path device
fsarchiver restfs archive_path [options] destination
