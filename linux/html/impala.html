Impala Command Overview

Description

The impala command is used to interact with Cloudera Impala, a high-performance SQL engine for Hadoop, allowing users to execute SQL queries on data stored in HDFS or HBase.

Common Options

Option Description
-i , --impalad=hostname:port Specify the Impala daemon to connect to.
-f , --query_file=file Execute SQL queries from a specified file.
-q , --query=query Execute a specified SQL query directly.
-B , --quiet Suppress welcome messages and query output headers.
-o , --output_file=file Redirect the output to a specified file.
--print_header Include column headers in the output.

Examples 

# Connect to an Impala daemon and run a query
impala-shell -i impala-server:21000 -q "SELECT * FROM my_table LIMIT 10;"

# Execute queries from a file and save output to another file
impala-shell -i impala-server:21000 -f queries.sql -o output.txt

# Run a query quietly without headers and redirect output
impala-shell -i impala-server:21000 -q "SELECT id, name FROM employees;" -B -o employees.txt

Cheat Sheet 

impala-shell -i host:port -q "SQL_QUERY"
impala-shell -i host:port -f queries.sql -o output.txt
impala-shell -i host:port -q "SQL_QUERY" -B -o result.txt
