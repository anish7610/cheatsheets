Linux `uptime` Command

Description

The uptime command displays how long the system has been running, the number of users currently logged in, and the system load averages for the past 1, 5, and 15 minutes.

Common Options

Option Description
-p Displays the uptime in a pretty format.
-s Shows the system boot time.

Usage Examples

Basic Example 

uptime

Displays the current time, uptime duration, number of users, and load averages.

Pretty Format 

uptime -p

Outputs the uptime in a human-readable format, like "up 5 hours, 10 minutes".

System Boot Time 

uptime -s

Shows the exact date and time when the system was last booted.

Cheat Sheet 

        uptime      # Show system uptime and load
        uptime -p   # Pretty format
        uptime -s   # System boot time
