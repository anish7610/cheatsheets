grub2 Command Overview

The grub2 command is used to manage the GNU GRand Unified Bootloader (GRUB) version 2, which is responsible for loading the operating system on many Linux systems. It provides a wide range of options to configure and update the boot process.

Common Options

Option Description
grub2-install Install GRUB to a device (e.g., hard drive).
grub2-mkconfig Generate a new grub.cfg configuration file.
grub2-set-default Set the default boot entry in grub.cfg .
grub2-reboot Set the next boot to a specific entry.
grub2-editenv Modify the GRUB environment block.

Usage Examples 

# Install GRUB to /dev/sda
sudo grub2-install /dev/sda

# Generate a new grub.cfg configuration file
sudo grub2-mkconfig -o /boot/grub2/grub.cfg

# Set the default boot entry to 2 (third entry)
sudo grub2-set-default 2

# Set the next boot to boot entry 1 (second entry)
sudo grub2-reboot 1

# List current GRUB environment variables
sudo grub2-editenv list

Cheat Sheet 


# Install GRUB
sudo grub2-install /dev/sdX

# Generate grub.cfg
sudo grub2-mkconfig -o /boot/grub2/grub.cfg

# Set default boot entry
sudo grub2-set-default N

# Reboot to specific entry
sudo grub2-reboot N
