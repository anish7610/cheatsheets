Linux dpkg Command

Description

The dpkg command is a low-level package manager for Debian-based systems, used to install, remove, and manage `.deb` packages.

Common Options

Option Description
-i, --install Install a package.
-r, --remove Remove a package.
-P, --purge Remove a package and its configuration files.
-l, --list List all installed packages.
-L, --listfiles List files installed by a package.
-s, --status Show the status of a package.
-S, --search Search for files that belong to a package.

Usage Examples 

# Install a package
sudo dpkg -i package.deb

# Remove a package
sudo dpkg -r package_name

# Purge a package
sudo dpkg -P package_name

# List all installed packages
dpkg -l

# List files installed by a package
dpkg -L package_name

# Show package status
dpkg -s package_name

# Search for a file in installed packages
dpkg -S /path/to/file
