Linux Command: head

Description

The head command displays the first few lines of a file or standard input.

Common Options

Option Description
-n NUM Show the first NUM lines of the file.
-c NUM Show the first NUM bytes of the file.
-q Suppress the header displaying filenames.
-v Always show the header with filenames.

Usage Examples 

# Display the first 10 lines of a file (default)
head example.txt

# Display the first 5 lines of a file
head -n 5 example.txt

# Display the first 20 bytes of a file
head -c 20 example.txt

# Display the first 10 lines of multiple files without headers
head -q file1.txt file2.txt

# Display the first 10 lines of multiple files with headers
head -v file1.txt file2.txt

Cheat Sheet 

head [OPTION]... [FILE]...
-n NUM  # Number of lines
-c NUM  # Number of bytes
-q      # Suppress headers
-v      # Show headers
