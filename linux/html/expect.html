Linux `expect` Command

Description

The expect command automates interactions with programs that require user input. It scripts dialogue with other interactive applications like telnet, ftp, etc.

Common Options

Option Description
-c <command> Execute a command in expect shell.
-d Enable debug mode to display what expect is doing.
-f <file> Run a script from a specified file.
-h Display help information.
-v Display version information.

Usage Examples 


#!/usr/bin/expect
spawn ssh user@hostname
expect "password:"
send "your_password\r"
interact
    

#!/usr/bin/expect -f
spawn ftp ftp.example.com
expect "Name*:"
send "username\r"
expect "Password:"
send "password\r"
interact

Cheat Sheet 


# Autologin via SSH
expect -c "
   spawn ssh user@host
   expect \"password:\"
   send \"pass\r\"
   interact
"

# Run Expect script from file
expect -f script_file.exp
