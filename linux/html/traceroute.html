Traceroute Command

Description

The traceroute command is used to display the route and measure the transit delays of packets across a network.

Common Options

Option Description
-n Display addresses numerically, without resolving hostnames.
-m max_hops Set the maximum number of hops (TTL) to the given max_hops .
-p port Use the given port for the first test instead of the default port (33434).
-q nqueries Set the number of probe packets sent per hop.
-w timeout Specify the time (in seconds) to wait for a response to a probe.

Usage Examples

Basic Traceroute 

traceroute example.com

Traceroute Without DNS Resolution 

traceroute -n example.com

Set Maximum Hops 

traceroute -m 15 example.com

Specify Number of Queries per Hop 

traceroute -q 1 example.com

Cheat Sheet 


traceroute [options] [destination]
-n          # No DNS resolution
-m    # Max hops
-q       # Number of queries
-p    # Initial destination port
-w    # Timeout per probe
