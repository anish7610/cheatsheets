Linux `fg` Command

Description

The fg command is used to bring a background job to the foreground in a shell environment.

Common Options

Option Description
%job-id Specifies the job ID to bring to the foreground.

Examples 

# Start a job in the background
sleep 100 &

# View background jobs
jobs

# Bring the first job to the foreground
fg %1

# Bring job with job ID 2 to the foreground
fg %2

Cheat Sheet 


# Bring the most recent background job to foreground
fg

# Bring a specific job to foreground using job ID
fg %job-id
