Linux Command: hive

The hive command is used to interact with Apache Hive, a data warehouse software that facilitates reading, writing, and managing large datasets stored in distributed storage using SQL.

Common Options

Option Description
-e <query> Execute a Hive query directly from the command line.
-f <file> Execute Hive commands from a file.
--hiveconf <key=value> Set Hive configuration variables.
--database <database> Specify the database to use.
--silent Suppress non-error messages in the output.

Usage Examples 

# Run a simple query
hive -e "SELECT * FROM my_table LIMIT 10;"

# Execute commands from a file named script.hql
hive -f script.hql

# Set a Hive configuration variable and run a query
hive --hiveconf mapreduce.job.queuename=myqueue -e "SHOW TABLES;"

# Specify a database and execute a query within it
hive --database my_database -e "SELECT COUNT(*) FROM another_table;"

Cheat Sheet 

# Run a Hive query
hive -e "QUERY"

# Execute commands from a file
hive -f FILE

# Set Hive configuration
hive --hiveconf KEY=VALUE

# Use a specific database
hive --database DB_NAME

# Suppress messages
hive --silent
