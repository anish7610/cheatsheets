Linux `restore` Command

Description

The restore command is used to restore files from a backup created using the dump command. It is primarily utilized for recovering data from backup tapes or files.

Common Options

Option Description
-i Interactive mode: allows selecting specific files to restore.
-r Restores the entire backup.
-t Lists the files contained in the backup.
-x Extracts specific files from the backup.
-f Specifies the backup file or tape to restore from.

Usage Examples

Restore Entire Backup 

restore -rf /dev/st0

List Files in Backup 

restore -tf /backup/dumpfile

Interactive Restore of Specific Files 

restore -if /backup/dumpfile

Extract Specific File 

restore -xf /backup/dumpfile /path/to/file

Cheat Sheet 


  # Restore entire backup
  restore -rf [device/file]

  # List contents of backup
  restore -tf [backup]

  # Interactive restore
  restore -if [backup]

  # Extract specific file
  restore -xf [backup] [file]
