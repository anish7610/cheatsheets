Linux blkid Command

Description

The blkid command is used to locate and print block device attributes. It provides information such as filesystem type and universally unique identifier (UUID).

Common Options

Option Description
-o Specify output format (e.g., udev , full ).
-p Proceed even if the device is not initialized.
-s Show only specified tags (e.g., TYPE, UUID).
-t Search by specified tag (e.g., type, uuid).
-L Follow symbolic links.
-V Verbose output.

Usage Examples

Print all available block devices' information 

blkid

Print information for a specific block device 

blkid /dev/sda1

Display only UUID of a block device 

blkid -s UUID -o value /dev/sda1

Search for devices by filesystem type (e.g., ext4) 

blkid -t TYPE=ext4
