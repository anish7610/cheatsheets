Linux Command: last

Description

The last command displays a list of the most recent user logins on a Linux system.

Common Options

Option Description
-n number Show the specified number of lines.
-R Omit the hostname column.
-F Display full date and time.
-x Show system shutdown entries and run level changes.

Usage Examples 

# Show the last 5 logins
last -n 5

# Show full date and time for each login
last -F

# Display logins, omitting the hostname
last -R

# Include shutdown and run level changes
last -x

Cheat Sheet
