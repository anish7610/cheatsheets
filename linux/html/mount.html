Linux Mount Command

Description

The mount command is used to attach filesystems and partitions to directories in the Linux file system hierarchy.

Common Options

Option Description
-t Specifies the filesystem type (e.g., ext4, ntfs).
-o Specifies options, like rw (read-write), ro (read-only).
-a Mounts all filesystems listed in /etc/fstab .
-v Enables verbose mode, providing detailed output.
--bind Mounts a subtree somewhere else.

Usage Examples 

# Mount a USB drive with ext4 filesystem
mount -t ext4 /dev/sdb1 /mnt/usb

# Mount an ISO file
mount -o loop /path/to/file.iso /mnt/iso

# Mount all filesystems in /etc/fstab
mount -a

# Bind mount a directory to a different location
mount --bind /home/user/data /mnt/data

Cheat Sheet 

mount -t [type] [device] [directory]
mount -o [options] [device] [directory]
mount -a
mount --bind [source] [target]
