Linux Command: lsusb

Description

The lsusb command lists all USB devices connected to a Linux system. It provides detailed information about each device, including vendor and product IDs.

Common Options

Option Description
-v Verbose output with detailed information about each device.
-s [bus]:[device] Display only devices with specified bus and device numbers.
-d [vendor]:[product] Show only devices with the specified vendor and product ID.
-t Dump the physical USB hierarchy as a tree.
-D [device] Retrieve detailed information from a specified device file.

Usage Examples 

# List all connected USB devices
lsusb

# Get detailed information about all USB devices
lsusb -v

# Show only the device with vendor ID 1234 and product ID 5678
lsusb -d 1234:5678

# Display USB devices as a hierarchy tree
lsusb -t
