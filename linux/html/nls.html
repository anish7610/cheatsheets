Linux Command: nls

Description

The nls command lists specific information or translations related to native language support settings on a Linux system.

Common Options

Option Description
-l, --list List available character sets or translations.
-v, --verbose Display detailed information about the selected character set.
-h, --help Show help message and exit.
-V, --version Output version information and exit.

Usage Examples 

# List available character sets
nls -l

# Show detailed information about a specific character set
nls -v utf8

# Display help information
nls -h

Cheat Sheet 

nls -l         # List character sets
nls -v [set]    # Verbose info for [set]
nls -h          # Display help
nls -V          # Show version
