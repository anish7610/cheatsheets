Linux Command: chgrp

Description

The chgrp command is used to change the group ownership of files and directories in Linux.

Common Options

Option Description
-c Report changes that are made.
-f Suppress most error messages.
-R Recursively change group of directories and their contents.
--preserve-root Do not operate recursively on the root directory.
--reference=FILE Use group of the specified reference file.

Usage Examples 

# Change group of a single file
chgrp developers myfile.txt

# Change group of a directory and its contents recursively
chgrp -R admins /var/www

# Change group and report changes
chgrp -c marketing report.docx

# Change group using a reference file
chgrp --reference=example.txt newfile.txt

Cheat Sheet 

chgrp [options] GROUP FILE...
chgrp -R GROUP DIRECTORY
