dbeaver Command

DBeaver is a free, multi-platform database tool that supports various database management systems. It provides an SQL editor, data viewer, and management features for developers and database administrators.

Common Options

Option Description
-con Connects to a specified database.
-data Specifies the workspace location.
-nosplash Starts without the splash screen.
-exit Exits after executing commands in the command line.
-nl Launches with a specified locale.

Usage Examples 

# Start DBeaver with a specified workspace
dbeaver -data /path/to/workspace

# Connect to a database and exit after execution
dbeaver -con "mydatabase" -exit

# Launch DBeaver without splash screen
dbeaver -nosplash

# Set locale to French
dbeaver -nl fr_FR

Cheat Sheet 

-con "db"    # Connect to 'db'
-data "/path"  # Set workspace
-nosplash      # No splash screen
-nl "locale"   # Set locale
-exit          # Exit after running
