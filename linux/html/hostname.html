Linux Command: hostname

Description

The hostname command is used to display or set the system's host name.

Common Options

Option Description
-s Display the short hostname (up to the first dot).
-f Display the full hostname, including the domain.
-i Display the network address(es) of the host.
-I Display all network addresses of the host.
--help Display help information.

Usage Examples 

# Display the system's current hostname
hostname

# Set the hostname to 'myserver'
sudo hostname myserver

# Display the short hostname
hostname -s

# Display the full hostname
hostname -f

# Display the IP address of the current host
hostname -i

Cheat Sheet 

# Display current hostname
hostname

# Set a new hostname (requires root)
sudo hostname [newname]

# Display short / full hostname
hostname -s / hostname -f

# Display IP address(es)
hostname -i / hostname -I
