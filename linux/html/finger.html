Linux Command: finger

The finger command displays information about system users, including their login name, real name, terminal name, idle time, login time, office location, and office phone number.

Common Options

Option Description
-l Provides a long format listing including detailed information.
-s Displays information in a short format (default).
-p Suppresses the display of the user's .plan file.

Usage Examples 

# Display information about all logged-in users
finger

# Display detailed information about a specific user
finger -l username

# List information without showing the user's .plan file
finger -p username

Cheat Sheet 

finger           # List all current users
finger username  # Info about a specific user
finger -l        # Detailed info about all users
finger -p user   # User info, no .plan file
