rmmod Command in Linux

The rmmod command is used to remove a module from the Linux kernel. It helps manage kernel modules by unloading them when they are no longer needed.

Common Options

Option Description
-f , --force Forcefully remove a module, even if it is in use or does not have a rmmod function (not recommended).
-s , --syslog Send messages to the syslog instead of standard output.
-v , --verbose Provide detailed information during execution.
--wait Wait until the module is no longer busy before removing it.

Usage Examples 

# Remove a module named 'module_name'
rmmod module_name

# Forcefully remove an in-use module
rmmod -f module_name

# Verbosely remove a module and log to syslog
rmmod -v -s module_name

# Remove a module and wait if it is currently busy
rmmod --wait module_name

Cheat Sheet 

rmmod [options] module_name
-f: Force removal
-s: Use syslog
-v: Verbose output
--wait: Wait if busy
