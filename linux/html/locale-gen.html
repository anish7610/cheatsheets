Linux Command: locale-gen

Description

The locale-gen command is used to compile and generate locale data files from the settings in /etc/locale.gen . It enables the system to support multiple languages and regional settings.

Common Options

Option Description
--purge Remove any locale data files not specified in /etc/locale.gen .
-v , --verbose Provide detailed output during generation.

Usage Examples

Generate All Locales Specified in /etc/locale.gen 

sudo locale-gen

Generate Specific Locale

Edit /etc/locale.gen and uncomment the desired locale, then run: 

sudo locale-gen

Purge Unspecified Locales 

sudo locale-gen --purge

Verbose Output 

sudo locale-gen --verbose

Cheat Sheet 


sudo locale-gen              // Generate locales from /etc/locale.gen
sudo locale-gen --purge      // Remove unlisted locales
sudo locale-gen --verbose    // Verbose output
