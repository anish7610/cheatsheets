Linux Command: yum

The yum command is a package manager used in RPM-based Linux distributions like CentOS, Fedora, and Red Hat Enterprise Linux. It helps to install, update, and remove packages, and manage dependencies.

Common Options

Option Description
install Installs one or more packages.
remove Removes one or more packages.
update Updates one or more packages.
list Lists available, installed, or updates to packages.
search Searches for a package by name or description.
info Displays detailed information about a package.
clean Cleans up temporary files.

Usage Examples 

# Install a package
yum install package-name

# Remove a package
yum remove package-name

# Update all packages
yum update

# List all installed packages
yum list installed

# Search for a package
yum search search-term

# Get information about a package
yum info package-name

# Clean cached files
yum clean all

Cheat Sheet 

yum install pkg - Install package
yum remove pkg  - Remove package
yum update        - Update all packages
yum list          - List packages
yum search term - Search for package
yum info pkg    - Package info
yum clean all     - Clean cache
