dstat Command

The dstat command is a versatile and flexible tool for monitoring system performance in real-time, providing statistics on CPU, memory, I/O, network, and more.

Common Options

Option Description
-c Show CPU usage
-d Show disk usage
-n Show network usage
-m Show memory usage
--top-cpu Display top processes by CPU usage
--top-mem Display top processes by memory usage
-a Show all statistics except -f and process stats
--ipc Show IPC statistics

Usage Examples 

# Monitor CPU and disk usage combined
dstat -cd

# Show all CPU, disk, and network statistics
dstat -cdn

# Monitor system including top memory-using processes
dstat -m --top-mem

# Monitor entire system excluding filesystem stats
dstat -a

Cheat Sheet 

dstat -cd       # CPU and disk
dstat -cdn      # CPU, disk, network
dstat -m --top-mem # Memory and top memory processes
dstat -a        # All except filesystem
