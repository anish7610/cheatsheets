Nmap Command Overview

Nmap (Network Mapper) is a powerful open-source tool used for network discovery and security auditing. It's commonly utilized to discover hosts and services on a computer network, thus creating a "map" of the network.

Common Options

Option Description
-sS Perform a stealth SYN scan.
-sT Perform a TCP connect scan.
-sU Perform a UDP scan.
-p Specify ports to scan.
-A Enable OS detection, version detection, script scanning, and traceroute.
-O Enable OS detection.
-Pn Treat all hosts as online; skip host discovery.
-v Increase verbosity level.
-oN Output scan results in a normal human-readable format to a file.

Usage Examples 

# Scan a single host using default options
nmap 192.168.1.1

# Perform a stealth SYN scan on hosts in a subnet
nmap -sS 192.168.1.0/24

# Scan specific ports on a host
nmap -p 22,80,443 192.168.1.1

# Enable OS and service detection
nmap -A 192.168.1.1

# Treat all hosts in the subnet as online
nmap -Pn 192.168.1.0/24

# Output results to a file
nmap -oN scan_results.txt 192.168.1.1

Cheat Sheet 


# Quick scan host
nmap [host]

# Stealth scan subnet
nmap -sS [subnet]

# Scan specific ports
nmap -p [ports] [host]

# Full scan with OS detection
nmap -A [host]

# Skip host discovery
nmap -Pn [subnet]

# Save results
nmap -oN [file] [host]
