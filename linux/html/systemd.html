systemd Command Overview

Description

Systemd is a system and service manager for Linux operating systems, providing a way to manage system processes after booting.

Common Options

Option Description
start Start a service
stop Stop a service
restart Restart a service
enable Enable a service to start on boot
disable Disable a service from starting on boot
status Show the status of a service

Usage Examples 

systemctl start nginx.service
systemctl stop apache2.service
systemctl restart sshd.service
systemctl enable docker.service
systemctl disable mysql.service
systemctl status cron.service
