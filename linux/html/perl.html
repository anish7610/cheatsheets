Perl Command

The perl command runs scripts and performs various text processing tasks using the Perl programming language.

Common Options

Option Description
-e Execute the provided Perl code.
-n Iterate over lines of input, executing code for each line.
-p Similar to -n , but also prints each line after processing.
-i Edit files in-place (modify them directly).
-l Handle line-endings correctly, automating chomp and print .
-w Enable warnings.

Usage Examples 

perl -e 'print "Hello, World!\n"'

Print "Hello, World!" 

perl -ne 'print if /pattern/' file.txt

Print lines matching "pattern" in file.txt . 

perl -pi -e 's/old/new/g' file.txt

Replace "old" with "new" globally in file.txt in-place. 

perl -ple '$_ = reverse' file.txt

Reverse each line in file.txt .

Cheat Sheet 


        # Print a message
        perl -e 'print "Message\n"'

        # Search for a pattern
        perl -ne 'print if /pattern/' file.txt

        # In-place substitution
        perl -pi -e 's/old/new/g' file.txt

        # Reverse lines
        perl -ple '$_ = reverse' file.txt
