Linux Command: ln

The ln command is used to create links between files. It supports both hard links and symbolic (soft) links.

Common Options

Option Description
-s Create a symbolic link instead of a hard link.
-f Force the creation of a link by removing any existing destination files.
-v Enable verbose mode, showing files as they are processed.
-n Do not overwrite an existing destination file.

Usage Examples 

# Create a hard link
ln source.txt link.txt

# Create a symbolic link
ln -s /path/to/source.txt /path/to/link.txt

# Forcefully create a symbolic link, removing existing link
ln -sf /new/source.txt /existing/link.txt

# Verbose symbolic link creation
ln -sv /path/to/source.txt /path/to/link.txt

Cheat Sheet 

ln source link             # Create hard link
ln -s source link          # Create symbolic link
ln -sf source link         # Force create symbolic link
ln -v source link          # Verbose output
