Linux `dd` Command Guide

Description

The `dd` command is a versatile tool used for low-level copy and conversion of raw data in Linux.

Common Options

Option Description
if= Input file (source)
of= Output file (destination)
bs= Block size (defines read/write chunk size)
count= Number of blocks to copy
status= Progress status reporting

Usage Examples 

# Create a bootable USB drive
dd if=/path/to/iso of=/dev/sdX bs=4M status=progress

# Backup an entire disk to a file
dd if=/dev/sdX of=/path/to/backup.img bs=1M status=progress

# Restore a disk from a backup file
dd if=/path/to/backup.img of=/dev/sdX bs=1M status=progress

# Wipe a disk with zeros
dd if=/dev/zero of=/dev/sdX bs=1M status=progress

# Clone a partition
dd if=/dev/sdX1 of=/dev/sdY1 bs=1M status=progress

Cheat Sheet 

dd if= of= bs= count= status=progress
