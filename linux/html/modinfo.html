modinfo Command

Description

The modinfo command is used to display information about a Linux kernel module. It provides details such as the module description, author, license, and parameters.

Common Options

Option Description
-n Show the path of the module file.
-a Display the author of the module.
-d Show the description of the module.
-p List the parameters that can be passed to the module.

Usage Examples 

# Display detailed information about the 'usbcore' module
modinfo usbcore

# Show the filename of the 'e1000' module
modinfo -n e1000

# List parameters for the 'i915' module
modinfo -p i915

Cheat Sheet 

modinfo [options] [module_name]

-n  : Show module file path
-a  : Display author
-d  : Show description
-p  : List parameters
