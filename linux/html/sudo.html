Description of the sudo Command

The sudo command allows a permitted user to execute a command as the superuser or another user, as specified by the security policy.

Common Options

Option Description
-u Run command as a specified user.
-b Run the command in the background.
-i Run the shell as the login shell for the user.
-l or --list List the allowed (and forbidden) commands.
-k or --reset-timestamp Invalidate the user's timestamp.

Usage Examples 

# Update package lists as superuser
sudo apt update

# Run a command as another user
sudo -u username command

# Open an interactive root shell
sudo -i

# Background execution of a command
sudo -b command

Cheat Sheet 


sudo command       # Run command as superuser
sudo -u user cmd   # Run as a specific user
sudo -b cmd        # Run in background
sudo -i            # Interactive shell as root
sudo -l            # List allowed commands
