Linux Command: update-locale

Description

The update-locale command is used to update or set the system's locale settings in /etc/default/locale . It helps manage language and regional settings for applications.

Common Options

Option Description
LANG=<locale> Sets the default system locale.
LANGUAGE=<language> Sets the preference for language messages.
LC_*=<locale> Sets specific locale variables like LC_TIME .

Usage Examples

Set Default Locale to U.S. English 

sudo update-locale LANG=en_US.UTF-8

Set Language Preference to French 

sudo update-locale LANGUAGE=fr_FR

Set Date/Time Locale to British English 

sudo update-locale LC_TIME=en_GB.UTF-8

Cheat Sheet 


# Set full system locale
sudo update-locale LANG=

# Set message language priority
sudo update-locale LANGUAGE=

# Set specific locale variables
sudo update-locale LC_=
