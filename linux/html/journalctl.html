Linux Command: journalctl

Description

The journalctl command is used to query and display messages from the journald logging system.

Common Options

Option Description
-e Jump to the end of the journal.
-n <number> Show the last n lines of the journal.
-f Follow journal logs in real-time.
-u <unit> Show logs for a specific systemd unit.
--since "time" Show logs since a certain time.
--until "time" Show logs until a certain time.

Usage Examples 

# View the latest 10 log entries
journalctl -n 10

# Follow log updates in real-time
journalctl -f

# View logs for a specific service
journalctl -u apache2

# View logs since yesterday
journalctl --since "yesterday"

# View logs for a specific period
journalctl --since "2023-10-01" --until "2023-10-10"

Cheat Sheet 

journalctl -e            # Jump to end
journalctl -n 100        # Last 100 lines
journalctl -f            # Follow logs
journalctl -u nginx      # Logs for nginx
journalctl --since "9am" # Logs since 9am
