rkt Command Overview

Description

The rkt command is a container runtime designed for executing App Container (ACI) images.

Common Options

Option Description
--insecure-options=all Disables all security features for testing purposes.
--trust-keys-from-https Automatically trust image keys from secure URLs.
--interactive Runs the container interactively, similar to a shell.
--memory Limits the container's memory usage.
--cpu Limits the container's CPU usage.

Usage Examples 

# Run a container interactively with all security features disabled
rkt run --interactive --insecure-options=all example.com/myapp

# Run a container with trusted keys and limited resources
rkt run example.com/myapp \
    --trust-keys-from-https \
    --memory=512M \
    --cpu=2

# List all running pods
rkt list

Cheat Sheet 

# Interactive run
rkt run --interactive

# Disable security
rkt run --insecure-options=all

# Auto-trust HTTPS keys
rkt run --trust-keys-from-https

# List running containers
rkt list
