Linux Command: vigr

Description

The vigr command is used to safely edit the /etc/group and /etc/gshadow files with a text editor, ensuring file integrity during the editing process.

Common Options

Option Description
-d Do not edit, but perform data validation of the files.
-s Edit the /etc/gshadow file instead of /etc/group .
-h Display help message and exit.

Usage Examples 

# Edit the /etc/group file
sudo vigr

# Edit the /etc/gshadow file
sudo vigr -s

# Validate group and gshadow files without editing
sudo vigr -d

Cheat Sheet 

vigr          # Edit /etc/group
vigr -s       # Edit /etc/gshadow
vigr -d       # Validate files without editing
