gsettings Command

Description

The gsettings command is used to interact with the dconf database, allowing users to get, set, and list application preference settings in GNOME-based environments.

Common Options

Option Description
list-schemas List all available schemas.
list-keys List all keys in a specified schema.
get Get the value of a specified key.
set Set the value of a key.
reset Reset a key to its default value.

Usage Examples

Get the current wallpaper URI 

gsettings get org.gnome.desktop.background picture-uri

Set a new wallpaper 

gsettings set org.gnome.desktop.background picture-uri 'file:///home/user/Pictures/wallpaper.jpg'

List all keys in a schema 

gsettings list-keys org.gnome.desktop.interface

Reset the theme to default 

gsettings reset org.gnome.desktop.interface gtk-theme

Cheat Sheet 


    List schemas:   gsettings list-schemas
    List keys:      gsettings list-keys 
    Get value:      gsettings get  
    Set value:      gsettings set   
    Reset key:      gsettings reset
