iptables: Linux Firewall Utility

Description

iptables is a user-space utility program that allows a system administrator to configure the IP packet filter rules of the Linux kernel firewall, implemented as different Netfilter modules.

Common Options

Option Description
-A Append a rule to a chain.
-I Insert a rule at a specific position in a chain.
-D Delete a specific rule from a chain.
-L List all rules in a chain.
-F Flush all rules in a chain.
-P Set the policy for a chain (ACCEPT, DROP).
-t Specify the table to use (e.g., filter, nat).

Usage Examples 


# List all rules in the filter table
iptables -t filter -L

# Append a rule to drop all incoming traffic on port 22
iptables -A INPUT -p tcp --dport 22 -j DROP

# Allow outgoing HTTP traffic
iptables -A OUTPUT -p tcp --dport 80 -j ACCEPT

# Set default policy to drop for input chain
iptables -P INPUT DROP

# Flush all rules in the NAT table
iptables -t nat -F
