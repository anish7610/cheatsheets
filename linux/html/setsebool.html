setsebool Command

The setsebool command is used to enable or disable SELinux boolean values, modifying SELinux policies temporarily or persistently.

Common Options

Option Description
-P Make changes persistent across reboots.

Usage Examples 

# Enable the httpd_can_network_connect boolean temporarily
setsebool httpd_can_network_connect on

# Enable the boolean persistently
setsebool -P httpd_can_network_connect on

# Disable a boolean temporarily
setsebool httpd_can_sendmail off

Cheat Sheet 

# Enable or disable SELinux boolean
setsebool  

# Persistent change
setsebool -P
