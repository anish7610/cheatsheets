Understanding the Linux ss Command

The ss command is used to view network socket information in Linux. It is a powerful tool for network troubleshooting and monitoring, providing details about TCP, UDP, and other socket connections.

Common Options

Option Description
-t Show TCP sockets.
-u Show UDP sockets.
-l Show listening sockets.
-p Show process using socket.
-a Show all sockets.
-n Do not resolve service names.
-s Show summary statistics.

Usage Examples 

# Display all listening TCP sockets
ss -tl

# Display all listening UDP sockets
ss -ul

# Display all connections with process info
ss -p

# Show detailed statistics for each protocol
ss -s

# Show all TCP and UDP connections without resolving service names
ss -tan
