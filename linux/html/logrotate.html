Logrotate Command Overview

Description

The logrotate command is used to manage automatic rotation and compression of log files in Linux, preventing them from consuming excessive disk space.

Common Options

Option Description
daily Rotate logs daily.
weekly Rotate logs weekly.
monthly Rotate logs monthly.
rotate N Keep N old log files.
compress Compress rotated logs.
missingok Do not issue an error if the log file is missing.
notifempty Do not rotate empty log files.
create Create new empty log files after rotation.

Examples 


# Rotate the log file daily and keep 7 days of logs
/var/log/myapp.log {
    daily
    rotate 7
    compress
    notifempty
    create
    missingok
}
    

# Rotate weekly, compress old logs, and keep 4 weeks of logs
/var/log/weekly.log {
    weekly
    rotate 4
    compress
    create
}

Cheat Sheet 


# Basic logrotate config
/path/to/logfile {
    daily | weekly | monthly
    rotate 
    compress
    create
    missingok
    notifempty
}
