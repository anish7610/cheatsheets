Linux Command: terraform

terraform is an open-source infrastructure as code software tool that provides a consistent CLI workflow to manage hundreds of cloud services. It allows users to define and provision data center infrastructure using a declarative configuration language.

Common Options

Option Description
init Initializes a new or existing Terraform configuration.
plan Creates an execution plan, showing what actions Terraform will take.
apply Applies the changes required to reach the desired state of the configuration.
destroy Removes infrastructure managed by Terraform.
validate Validates the Terraform files without applying any changes.
output Reads an output variable from a Terraform state file.

Usage Examples 

# Initialize Terraform configuration
terraform init

# Create an execution plan
terraform plan

# Apply the changes to achieve desired state
terraform apply

# Destroy infrastructure managed by Terraform
terraform destroy

# Validate Terraform configuration
terraform validate

# Output a specific variable
terraform output <variable_name>

Cheat Sheet 

terraform init - Reinitialize your working directory.
terraform plan - Preview changes before applying.
terraform apply - Enact configuration changes.
terraform destroy - Tear down infrastructure.
terraform validate - Check configuration syntax and validity.
terraform output - Retrieve value from state.
