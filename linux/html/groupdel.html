Linux groupdel Command

Description

The groupdel command is used to delete a group from the system in Linux.

Common Options

Option Description
N/A groupdel has no options. Simply specify the group name to delete.

Usage Examples

Delete a Group 

sudo groupdel examplegroup

This deletes the group named examplegroup from the system.

Error Handling 

sudo groupdel nonexistinggroup

Attempting to delete a non-existent group will result in an error message indicating the group does not exist.

Check Group Exists Before Deletion 

getent group examplegroup && sudo groupdel examplegroup

This checks if the group examplegroup exists before attempting to delete it.

Cheat Sheet 


# Delete a group
sudo groupdel groupname

# Check if group exists then delete
getent group groupname && sudo groupdel groupname
