hdparm Command

hdparm is a command-line utility for Linux to set and view hardware parameters of hard disk drives.

Common Options

Option Description
-i Display the drive's identification information.
-I More detailed identification info from the drive.
-t Benchmark device read speed (timed read).
-T Test cached reads of the device.
-S Set standby (spindown) timeout.
-y Put drive into standby mode.
-Y Put drive into sleep mode.

Usage Examples 

# Display drive identification
sudo hdparm -i /dev/sda

# Perform a read speed test
sudo hdparm -t /dev/sda

# Set drive to standby mode
sudo hdparm -y /dev/sda

# Read detailed drive info
sudo hdparm -I /dev/sda

Cheat Sheet 


-i  : Basic info
-I  : Detailed info
-t  : Timed read
-T  : Cached read
-S  : Set spindown
-y  : Standby mode
-Y  : Sleep mode
