Linux `tar` Command Guide

Description

The tar command is used for archiving and compressing files. It creates and extracts .tar archives, and can handle special file types and attributes.

Common Options

Option Description
-c Create a new archive
-x Extract files from an archive
-v Display progress in the terminal
-f Use archive file or device (specifies filename)
-z Compress the archive with gzip
-j Compress the archive with bzip2
-t List the contents of an archive

Usage Examples

Create a tar.gz archive 

tar -cvzf archive.tar.gz /path/to/directory

Extract a tar.gz archive 

tar -xvzf archive.tar.gz

List contents of a tar file 

tar -tvf archive.tar

Create a tar.bz2 archive 

tar -cvjf archive.tar.bz2 /path/to/directory

Cheat Sheet 


Create:   tar -cvf archive.tar /path
Extract:  tar -xvf archive.tar
Gzip:     tar -cvzf archive.tar.gz /path
List:     tar -tvf archive.tar
