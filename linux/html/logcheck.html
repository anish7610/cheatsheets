Logcheck Command Overview

Description

logcheck is a command-line utility for scanning system log files for unusual activity, sending reports via email based on predefined filters.

Common Options

Option Description
-r Run in "service" mode (daemon).
-R Debug runs, showing why lines are included or ignored.
-t Test the alert filtering.
-a Include all lines in the report.
-s Silent mode, minimal output.

Usage Examples 

# Run logcheck in normal mode and output the report
logcheck

# Test alert filtering rules without sending email
logcheck -t

# Run logcheck and include all log lines
logcheck -a

# Debug mode to see why certain lines are processed
logcheck -R

Cheat Sheet 

logcheck          # Run normally
logcheck -r       # Daemon mode
logcheck -t       # Test filtering
