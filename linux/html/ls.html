Linux `ls` Command

The `ls` command lists directory contents, providing information about files and directories.

Common Options

Option Description
-a Show all files, including hidden files
-l Use a long listing format
-h Human-readable sizes (with -l)
-R Recursively list subdirectories
-t Sort by modification time

Usage Examples 

# List files in current directory
ls

# Detailed list, including hidden files
ls -la

# List files with human-readable sizes
ls -lh

# List files sorted by modification time
ls -lt

# Recursively list all files in directory and subdirectories
ls -R

Cheat Sheet 

ls                      # List files
ls -la                   # Detailed list including hidden
ls -lh                   # List with human-readable sizes
