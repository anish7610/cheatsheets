Linux Command: ifconfig

Description

The ifconfig command is used to configure and manage network interfaces in Unix-based systems.

Common Options

Option Description
interface Display configuration for a specific interface.
up Activate the specified network interface.
down Deactivate the specified network interface.
add Add an IP address to an interface.
del Remove an IP address from an interface.
netmask Set a network mask for an interface.

Usage Examples 

# Display all network interfaces
ifconfig

# Display information for eth0
ifconfig eth0

# Activate the interface eth0
ifconfig eth0 up

# Deactivate the interface eth0
ifconfig eth0 down

# Assign an IP address to eth0
ifconfig eth0 192.168.1.10

# Set the netmask for eth0
ifconfig eth0 netmask 255.255.255.0

# Add a second IP to eth0
ifconfig eth0:0 192.168.1.11

# Remove a specific IP address from eth0
ifconfig eth0:0 del 192.168.1.11
