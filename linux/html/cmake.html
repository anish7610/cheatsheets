CMake Command Overview

Description

CMake is a cross-platform build-system generator used to manage the build process of software projects. It generates native build files for different IDEs and compilers.

Common Options

Option Description
-S <path> Specifies the source directory.
-B <path> Specifies the build directory.
-G <generator> Selects the build system generator.
-D <var>=<value> Defines a variable to be used in the CMake process.
--build <dir> Builds a project using the specified directory.
--install <dir> Installs a project from the specified directory.

Usage Examples

Basic Project Setup 

cmake -S . -B build/

Configures the current directory as the source and build/ as the build directory.

Building a Project 

cmake --build build/

Compiles the project in the build/ directory.

Specifying a Build Generator 

cmake -G "Unix Makefiles" -S . -B build/

Uses "Unix Makefiles" as the build generator.

Defining Variables 

cmake -D CMAKE_BUILD_TYPE=Release -S . -B build/

Sets the CMAKE_BUILD_TYPE to Release .

Quick Reference Cheat Sheet 


# Configure project
cmake -S . -B build/

# Build project
cmake --build build/

# Install project
cmake --install build/

# Set build type
cmake -D CMAKE_BUILD_TYPE=Release -S . -B build/
