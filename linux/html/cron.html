Cron Command in Linux

Description

The cron command is a time-based job scheduler in Unix-like operating systems. It is used to schedule scripts or commands to run automatically at specified intervals.

Common Options

Option Description
-e Edit the current user's crontab file.
-l List the current user's crontab entries.
-r Remove the current user's crontab .

Usage Examples

1. Schedule a Backup Script to Run Daily 


# Open crontab editor
crontab -e

# Add this line to run backup.sh every day at 2 AM
0 2 * * * /path/to/backup.sh

2. Run a Cleanup Script Every Sunday 


# Open crontab editor
crontab -e

# Add this line to run cleanup.sh every Sunday at 3 AM
0 3 * * 0 /path/to/cleanup.sh

3. List All Scheduled Jobs 


crontab -l

Cron Cheat Sheet 


# Edit crontab:
crontab -e

# List crontab:
crontab -l

# Remove crontab:
crontab -r
