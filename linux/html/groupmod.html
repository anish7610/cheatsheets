Linux groupmod Command

groupmod is used to modify existing group definitions on Linux systems.

Common Options

Option Description
-g, --gid Change the Group ID (GID) to the specified value.
-n, --new-name Rename the group to a new group name.
-o, --non-unique Allow the GID to be non-unique.

Usage Examples 

# Change the GID of the group "developers" to 1001
groupmod -g 1001 developers

# Rename the group "staff" to "team"
groupmod -n team staff

# Allow a duplicate GID for the group "admins" to 1002
groupmod -o -g 1002 admins

Cheat Sheet 

groupmod -g GID groupName          # Change GID
groupmod -n newGroupName groupName  # Rename group
groupmod -o -g GID groupName        # Allow non-unique GID
