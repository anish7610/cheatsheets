hostnamectl Command Guide

Description

The hostnamectl command is used to query and change the system hostname and related settings on a Linux system.

Common Options

Option Description
status Display current hostname and related system information.
set-hostname [name] Set the system's transient and static hostnames.
set-icon-name [name] Set an icon name for the system.
set-chassis [type] Set the chassis type (e.g., desktop, laptop).
set-deployment [name] Set the deployment environment (e.g., development, production).
set-location [name] Set the location for the system.

Examples 

# Display current hostname and system info
hostnamectl status

# Change the system hostname to "mynewhost"
hostnamectl set-hostname mynewhost

# Set the chassis type to "laptop"
hostnamectl set-chassis laptop

# Set the deployment environment to "production"
hostnamectl set-deployment production
