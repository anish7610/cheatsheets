Linux umount Command

Description

The umount command is used to unmount filesystems in Linux. It detaches the filesystem from the file hierarchy.

Common Options

Option Description
-a Unmount all filesystems mentioned in /etc/mtab
-f Force unmount (useful for unreachable NFS systems)
-l Lazy unmount (detach the filesystem now, clean up after)
-v Enable verbose mode

Usage Examples 

# Unmount a filesystem by specifying the directory
umount /mnt/external_drive

# Force unmount if the usual unmount fails
umount -f /mnt/external_drive

# Lazy unmount a remote NFS mount
umount -l /mnt/remote_nfs

# Unmount all filesystems from /etc/mtab
umount -a

Cheat Sheet 

umount [options] 
-a Unmount all
-f Force
-l Lazy
-v Verbose
