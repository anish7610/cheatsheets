CloudFormation Command

The cloudformation command is part of the AWS CLI, used for managing AWS CloudFormation stacks, which allows you to model and set up your Amazon Web Services resources.

Common Options

Option Description
create-stack Creates a CloudFormation stack.
update-stack Updates an existing stack.
delete-stack Deletes a specified stack.
describe-stacks Describes one or more stacks.
list-stacks Lists stacks with a specific status.

Usage Examples 

# Create a new stack
aws cloudformation create-stack --stack-name my-stack --template-body file://template.json

# Update an existing stack
aws cloudformation update-stack --stack-name my-stack --template-body file://updated-template.json

# Delete a stack
aws cloudformation delete-stack --stack-name my-stack

# Describe specific stacks
aws cloudformation describe-stacks --stack-name my-stack

# List all stacks with their status
aws cloudformation list-stacks --stack-status-filter CREATE_COMPLETE

Cheat Sheet 

Create: aws cloudformation create-stack --stack-name [name] --template-body [file://path]
Update: aws cloudformation update-stack --stack-name [name] --template-body [file://path]
Delete: aws cloudformation delete-stack --stack-name [name]
Describe: aws cloudformation describe-stacks --stack-name [name]
List: aws cloudformation list-stacks --stack-status-filter [status]
