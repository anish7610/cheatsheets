inxi Command Overview

Description

The inxi command provides detailed system information. It's user-friendly and displays data about hardware, system resources, and other key aspects of the operating environment.

Common Options

Option Description
-S Show system information.
-C Display CPU information.
-M Memory details.
-G Graphics information.
-D Hard drives and storage details.

Usage Examples

Example 1: Display full system information 

inxi -F

Example 2: Show only CPU details 

inxi -C

Example 3: Check memory status 

inxi -M

Example 4: Get graphics card information 

inxi -G

Example 5: View hard drive details 

inxi -D

Cheat Sheet 

inxi -F   # Full system info
inxi -S   # System
inxi -C   # CPU
inxi -M   # Memory
inxi -G   # Graphics
