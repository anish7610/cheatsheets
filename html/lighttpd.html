Lighttpd Command Guide

Description

The lighttpd command is used to start and manage the Lighttpd web server, known for its speed and small footprint.

Common Options

Option Description
-f Specifies an alternative configuration file to use instead of the default.
-m Sets the base directory for modules.
-p Prints the configuration file after reading it.
-t Tests the configuration file for syntax errors.
-D Runs in the foreground; useful for debugging.
-v Displays version information.

Examples 

# Start Lighttpd using a custom configuration
lighttpd -f /path/to/config.conf

# Test the configuration file for errors
lighttpd -t -f /etc/lighttpd/lighttpd.conf

# Run in the foreground for debugging
lighttpd -D -f /etc/lighttpd/lighttpd.conf

# Print the configuration file after processing
lighttpd -p -f /etc/lighttpd/lighttpd.conf

Cheat Sheet 

Start    : lighttpd -f config.conf
Test     : lighttpd -t -f config.conf
Debug    : lighttpd -D -f config.conf
Version  : lighttpd -v
Print    : lighttpd -p -f config.conf
