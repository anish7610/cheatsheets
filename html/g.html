g++ Command Overview

The g++ command is a part of the GNU Compiler Collection (GCC). It is used to compile C++ programs.

Common Options

Option Description
-o <file> Specify the output file name.
-c Compile source files to object files without linking.
-Wall Enable all standard warnings.
-g Generate debug information.
-O2 Optimize the code for speed.
-std=<standard> Select the C++ language standard (e.g., C++11, C++14).

Usage Examples 

# Compile a single file to an executable
g++ main.cpp -o main

# Compile with debugging information
g++ main.cpp -g -o main

# Compile with optimizations and warnings
g++ main.cpp -O2 -Wall -o main

# Compile multiple files
g++ main.cpp utils.cpp -o program

# Compile with a specific C++ standard
g++ main.cpp -std=c++11 -o main

Cheat Sheet 

# Basic Compilation
g++ file.cpp -o output

# Debugging
g++ -g file.cpp -o output

# Optimization
g++ -O2 file.cpp -o output

# All Warnings
g++ -Wall file.cpp -o output

# Object File
g++ -c file.cpp
