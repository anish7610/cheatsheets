Linux Snapshot Command

Description

The snapshot command is used to create and manage snapshots of file systems or volumes. These snapshots capture the state of a file system at a particular point in time, allowing users to revert to this state if needed.

Common Options

Option Description
-c Create a new snapshot.
-d Delete an existing snapshot.
-l List all available snapshots.
-r Restore a snapshot to revert the file system.

Usage Examples 


# Create a snapshot of the /home directory
snapshot -c /home

# List all snapshots
snapshot -l

# Restore a specific snapshot
snapshot -r snapshot-name

# Delete a specific snapshot
snapshot -d snapshot-name

Cheat Sheet 


# Create:  snapshot -c 
# List:   snapshot -l
# Restore: snapshot -r 
# Delete: snapshot -d
