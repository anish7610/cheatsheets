nload Command

Description

nload is a command-line tool for monitoring network traffic and bandwidth usage in real-time, displaying incoming and outgoing traffic separately.

Common Options

Option Description
-a [time] Sets the length of the average calculation period to [time] seconds.
-i [interval] Determines the refresh interval for the display in milliseconds.
-u [unit] Sets the display unit (H for auto, K for kBytes, M for MBytes).
-m Monitors all network devices instead of specifying one.
--help Displays help information.

Usage Examples 


# Monitor default network interface
nload

# Monitor a specific network interface, e.g., eth0
nload eth0

# Set the average calculation period to 5 seconds
nload -a 5

# Refresh display every 500 milliseconds
nload -i 500

# Display bandwidth in kilobytes
nload -u K

Cheat Sheet 


nload [interface]            # Monitor default or specified interface
-a [time]                    # Set average period
-i [interval]                # Set refresh interval
-u [unit]                    # Set display unit
-m                           # Monitor all devices
