Linux Command: grub

Description

The grub command is used to manage the GRand Unified Bootloader (GRUB), which is the boot loader responsible for loading the Linux kernel and managing the boot process.

Common Options

Option Description
install Installs GRUB on a specified device.
update Updates the GRUB configuration file.
reboot Reboots the machine using GRUB.
set-default Sets the default boot entry for GRUB.

Usage Examples 

# Install GRUB on /dev/sda
grub-install /dev/sda

# Update GRUB configuration
grub-mkconfig -o /boot/grub/grub.cfg

# Reboot system using GRUB
grub-reboot 2

# Set default boot entry
grub-set-default 0

Cheat Sheet 

# Install GRUB
grub-install [device]

# Update config
grub-mkconfig -o /boot/grub/grub.cfg

# Reboot
grub-reboot [entry]

# Set default entry
grub-set-default [entry]
