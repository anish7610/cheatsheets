kubectl Command Overview

kubectl is the command-line tool for controlling Kubernetes clusters. It allows you to deploy applications, inspect and manage cluster resources, and view logs.

Common Options

Option Description
get List resources
describe Show detailed information about a resource
create Create a resource from a file or stdin
delete Delete resources by filenames, stdin, resources and names, or by resources and label selector
apply Apply a configuration to a resource by filename or stdin
logs Print the logs for a container in a pod
-n Specify a namespace

Usage Examples 

kubectl get pods
kubectl describe pod mypod
kubectl create -f pod.yaml
kubectl delete pod mypod
kubectl apply -f deployment.yaml
kubectl logs mypod
kubectl get services -n mynamespace

Cheat Sheet 

kubectl get [resource]
kubectl describe [resource] [name]
kubectl create -f [file]
kubectl delete [resource] [name]
kubectl apply -f [file]
kubectl logs [podname]
kubectl [command] -n [namespace]
