Linux modprobe Command

Description

The modprobe command is used to add or remove modules from the Linux kernel. It effectively manages dependencies between modules.

Common Options

Option Description
-v , --verbose Run in verbose mode, showing more detail about what the command does.
-r , --remove Remove a module from the kernel.
-n , --dry-run Simulate an action; do not actually load or unload a module.
--first-time Only load the module if it is not already loaded.
--show-depends List the dependencies of a module without loading it.

Realistic Usage Examples 

# Load a module (e.g., foo)
modprobe foo

# Remove a module (e.g., foo)
modprobe -r foo

# Verbosely load a module
modprobe -v foo

# Show dependencies of a module
modprobe --show-depends foo

# Simulate loading a module
modprobe -n foo

Cheat Sheet 

# Load: modprobe [module]
# Remove: modprobe -r [module]
# Verbose: modprobe -v [module]
# Show dependencies: modprobe --show-depends [module]
