sftp Command in Linux

The sftp command is used for secure file transfer over SSH. It supports various operations like file transfer, navigation, and renaming files.

Common Options

Option Description
-b batchfile Run commands from a batchfile on start up.
-C Enable compression.
-i identity_file Specifies the file from which the identity (private key) for public key authentication is read.
-o option Passes an option to SSH in the format used in ssh_config .
-P port Port to connect to on the remote host.
-q Quiet mode; suppresses the progress meter and non-error messages.
-r Recursively copy entire directories.

Usage Examples 

# Connect to a remote host
sftp user@hostname

# Connect using a specific port
sftp -P 2222 user@hostname

# Upload a file to the remote host
put localfile.txt

# Download a file from the remote host
get remotefile.txt

# Use a specific identity file for authentication
sftp -i ~/.ssh/id_rsa user@hostname

# Download an entire directory recursively
sftp -r user@hostname:/path/to/remote_dir /path/to/local_dir
