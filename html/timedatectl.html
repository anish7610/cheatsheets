timedatectl Command Overview

The timedatectl command is used to query and change the system clock and its settings in Linux systems.

Common Options

Option Description
status Displays the current time settings and status.
set-time <time> Sets the system time. Format: YYYY-MM-DD HH:MM:SS.
set-timezone <zone> Sets the system timezone. Example: America/New_York .
list-timezones Lists all available timezones.
set-ntp <bool> Enables or disables Network Time Protocol (NTP) synchronization.

Usage Examples 

        # Display the current time settings
        timedatectl status

        # Set the system time to a specific date and time
        timedatectl set-time "2023-10-01 12:00:00"

        # Change the time zone to New York
        timedatectl set-timezone America/New_York

        # List all available time zones
        timedatectl list-timezones

        # Enable NTP synchronization
        timedatectl set-ntp true
