Linux Command: locale

Description

The locale command in Linux is used to display or set the current locale settings, which define regional settings such as language, currency, and format for dates and numbers.

Common Options

Option Description
-a Display all available locales.
-m Display all available charmaps.
-k Display key names and values.
-c Check for permanent output capabilities.

Usage Examples 

# Display all available locales
locale -a

# Display current locale settings
locale

# Display specific locale category (e.g., LC_TIME)
locale LC_TIME

# Display all available charmaps
locale -m

# Display key names and values for the current locale
locale -k

Cheat Sheet 

# List all locales
locale -a
