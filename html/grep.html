Linux Command: grep

Description

The grep command is used in Linux to search for a specific pattern of text within files. It stands for "global regular expression print" and is commonly used to filter output or find specific information.

Common Options

Option Description
-i Ignore case distinctions in patterns and data.
-r / -R Recursively search directories.
-n Prefix each line of output with the line number.
-v Invert the match to select non-matching lines.
-l List file names with matching lines (once per file).
-c Count the number of matching lines per file.
--color Highlight the matching text.

Usage Examples 

# Search for 'error' in a file, ignoring case
grep -i 'error' file.txt

# Recursively search for 'TODO' in the current directory
grep -r 'TODO' .

# Show line numbers for matches
grep -n 'function' file.c

# Find files not containing 'success'
grep -v 'success' file.txt

# List files that contain 'main'
grep -l 'main' *.c

# Count occurrences of 'function'
grep -c 'function' file.c

# Search with highlighted matches
grep --color 'pattern' file.txt

Cheat Sheet 

grep -i 'pattern' file        # Ignore case
grep -r 'pattern' dir           # Recursive search
grep -n 'pattern' file          # Show line numbers
grep -v 'pattern' file          # Invert match
grep -l 'pattern' files         # List matching files
grep -c 'pattern' file          # Count matches
grep --color 'pattern' file     # Highlight matches
