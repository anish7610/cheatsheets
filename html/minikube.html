Minikube Command

Minikube is a tool that enables running Kubernetes clusters locally, providing a local development environment for Kubernetes.

Common Options

Option Description
-p <name> Specify the profile name to use for the cluster.
start Start a local Kubernetes cluster.
stop Stop the local Kubernetes cluster.
delete Delete the local Kubernetes cluster.
status Get the status of the local Kubernetes cluster.
dashboard Access the Kubernetes dashboard running within the cluster.

Usage Examples 

# Start a Minikube cluster
minikube start

# Stop the Minikube cluster
minikube stop

# Delete the Minikube cluster
minikube delete

# Check the status of the Minikube cluster
minikube status

# Open the Kubernetes dashboard
minikube dashboard

# Start a Minikube cluster with a specific profile
minikube start -p mycluster

Cheat Sheet
