fdisk Command Overview

Description

The fdisk command is used to view and manage disk partitions on Linux systems. It allows you to create, delete, and modify partitions on a hard disk.

Common Options

Option Description
-l List partition tables for specified devices.
-n Create a new partition.
-d Delete a partition.
-t Change the partition type.
-v Display version information.
-c Switch to DOS compatibility mode.

Usage Examples 

# List all partitions on all devices
fdisk -l

# Create a new partition on /dev/sda
fdisk /dev/sda
# (Interactive menu will prompt for changes)

# Delete a partition on /dev/sda
fdisk /dev/sda
# (Interactive menu will prompt, use 'd' for delete)

Cheat Sheet 

fdisk -l              # List partitions
fdisk /dev/sda        # Start fdisk on /dev/sda
Use 'n'               # Create new partition
Use 'd'               # Delete partition
Use 't'               # Change partition type
Use 'w'               # Write changes
