mkdir Command

The mkdir command is used to create new directories in a Linux file system.

Common Options

Option Description
-p Create parent directories as needed.
-v Print a message for each directory created.
-m mode Set file mode (permissions) for the new directory.

Usage Examples 

mkdir new_directory
mkdir -p /parent/directory/new_directory
mkdir -m 755 new_directory
mkdir -v new_directory

Cheat Sheet 

mkdir <directory>
mkdir -p <path/to/directory>
mkdir -m <mode> <directory>
mkdir -v <directory>
