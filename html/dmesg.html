Linux dmesg Command

The dmesg command is used to examine or control the kernel ring buffer, primarily for troubleshooting hardware and system-related issues.

Common Options

Option Description
-C Clear the kernel ring buffer contents.
-c Clear the buffer after printing.
-d Show only messages regarding booting issues.
-f facility Show messages from a specific facility (e.g., kern ).
-l level Filter messages by level (e.g., err , warn ).
-n level Set the level at which logging is done to the console.
--color Highlight messages by their level.

Usage Examples 

# Display recent kernel messages
dmesg

# Clear the kernel ring buffer
dmesg -C

# Show only error messages
dmesg -l err

# Print messages and clear the buffer
dmesg -c

# Highlight messages by severity
dmesg --color

Cheat Sheet 

dmesg            # Show all messages
dmesg -C         # Clear buffer
dmesg -l warn    # Show warnings
dmesg -c         # Print & clear
dmesg --color    # Highlight by level
