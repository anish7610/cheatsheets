Linux `unzip` Command

The `unzip` command is used to extract files from a ZIP archive.

Common Options

Option Description
-l List contents of ZIP archive without extracting.
-d directory Extract files into the specified directory.
-n Never overwrite existing files.
-o Overwrite existing files without prompting.
-q Quiet mode; minimal output.
-x pattern Exclude files that match the pattern.

Usage Examples 

# Extract all files from archive.zip
unzip archive.zip

# List contents without extracting
unzip -l archive.zip

# Extract to a specific directory
unzip archive.zip -d /path/to/directory

# Overwrite files without confirmation
unzip -o archive.zip

# Extract while excluding certain files
unzip archive.zip -x "*.txt"

Cheat Sheet 

unzip file.zip          # Extract all
unzip -l file.zip      # List contents
unzip -d dir file.zip  # Extract to dir
unzip -n file.zip      # Never overwrite
unzip -o file.zip      # Overwrite without prompt
unzip -q file.zip      # Quiet mode
