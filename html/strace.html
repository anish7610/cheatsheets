strace Command Overview

The strace command in Linux is used to trace system calls and signals made by a process. It helps in debugging and understanding how a program interacts with the operating system.

Common Options

Option Description
-c Count time, calls, and errors for each system call and report a summary.
-e Filter which system calls are traced.
-p Attach to a process with a specified PID.
-f Trace child processes created by fork .
-o Write trace output to a file.
-s Specify the maximum string length to print.

Usage Examples 

# Trace all system calls made by ls
strace ls

# Trace all open syscalls made by ls
strace -e open ls

# Attach to a running process with PID 1234
strace -p 1234

# Trace a process and its child processes
strace -f -o trace_output.txt my_program

# Summarize time and errors for all syscalls in ls
strace -c ls

Cheat Sheet 

# Basic usage
strace 

# Specific syscall
strace -e syscall 

# Attach to PID
strace -p 

# Output to file
strace -o
