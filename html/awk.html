AWK Command in Linux

awk is a powerful text processing tool used for pattern scanning and processing. It operates on each line of input, making it ideal for extracting and analyzing data from files or streams.

Common Options

Option Description
-F Sets the input field separator.
-f Reads the AWK program from a file.
-v Assigns a variable before program execution.
--help Displays a help message and exits.

Examples 

awk '{print $1}' filename.txt
# Prints the first column of each line in the file.

awk -F: '{print $1, $3}' /etc/passwd
# Uses colon as a separator and prints the first and third fields.

echo "Hello World" | awk '{print toupper($0)}'
# Converts the input text to uppercase.

ps aux | awk '/username/ {print $1, $2, $11}'
# Filters lines containing 'username' and prints specific columns.

awk -v var="value" 'BEGIN {print var}'
# Sets and prints a variable value.

Cheat Sheet 

# Field separator
awk -F, '{print $1}'

# Use a script file
awk -f script.awk filename

# Set a variable
awk -v var=value '{print var}'

# Execute an action for lines matching a pattern
awk '/pattern/ {action}' file

# Calculate sum of a column
awk '{sum += $1} END {print sum}' file
