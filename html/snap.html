Snap Command Overview

The snap command is used to manage and install packages from the Snap Store, a modern application package format for Linux.

Common Options

Option Description
install <package> Installs a package from the Snap Store.
remove <package> Removes a package.
refresh Updates all installed snaps.
list Lists all installed snaps.
info <package> Displays details about the specified snap package.

Usage Examples 

# Install a package
sudo snap install vlc

# Remove a package
sudo snap remove vlc

# Update all installed packages
sudo snap refresh

# List installed packages
snap list

# Get information about a specific package
snap info vlc
