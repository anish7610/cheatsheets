Linux Command: nslookup

Description

nslookup is a command-line tool used to query Domain Name System (DNS) to obtain domain name or IP address mapping, or other DNS records.

Common Options

Option Description
-type= Specify the type of DNS record to query (e.g., A, MX, TXT).
-timeout= Set the time to wait for a response.
-retry= Set the number of retries to make before giving up.
-debug Print debugging information.
-debug Enable verbose debugging mode.

Usage Examples 


# Lookup an IP address for a domain
nslookup example.com


# Query a specific DNS record type
nslookup -type=MX example.com


# Use a specific DNS server for the query
nslookup example.com 8.8.8.8


# Enable debugging information
nslookup -debug example.com

Cheat Sheet 


nslookup : Simple query
nslookup -type= : Query specific record
nslookup  : Use specific DNS server
nslookup -debug : Debug mode
