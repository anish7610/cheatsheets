Linux `jobs` Command

Description

The jobs command displays a list of all current jobs along with their statuses in the current shell session.

Common Options

Option Description
-l Displays the process ID (PID) of each job.
-n Lists only jobs that have stopped or exited since the last notification.
-p Displays only the process ID of the jobs.

Usage Examples 

# List all current jobs
jobs

# Include PIDs in the output
jobs -l

# Display jobs with updated states
jobs -n

# Show only the PIDs of the jobs
jobs -p

Quick Reference Cheat Sheet 

jobs       # List all jobs
jobs -l    # Include PIDs
