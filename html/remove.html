Linux `rm` Command

The `rm` command is used to remove files or directories from the filesystem.

Common Options

Option Description
-f, --force Ignore nonexistent files and do not prompt.
-i Prompt before every removal.
-r, -R, --recursive Remove directories and their contents recursively.
-d, --dir Remove empty directories.
--help Display help information about the `rm` command.

Usage Examples 


# Remove a file
rm file.txt

# Remove multiple files
rm file1.txt file2.txt

# Remove a directory and its contents recursively
rm -r directory_name

# Force remove a file without prompting
rm -f file.txt

# Prompt before removing each file
rm -i file.txt

# Remove an empty directory
rm -d empty_directory

Cheat Sheet 


rm [OPTION]... FILE...
rm -f filename          # Force remove
rm -i filename          # Interactive remove
rm -r directory         # Recursive remove
rm -d directory         # Remove empty directory
