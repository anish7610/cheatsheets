localectl Command

localectl is a command-line utility for querying and setting system locale and keyboard layout settings.

Common Options

Option Description
status Show current locale and keyboard settings.
set-locale LANG=... Set system locale to a specified language.
set-keymap MAP Set console keyboard layout to a specified keymap.
list-locales List available locales on the system.
list-keymaps List available keymaps on the system.

Usage Examples

Check Current Locale and Keymap 

localectl status

Set System Locale

Set the system locale to US English: 

localectl set-locale LANG=en_US.UTF-8

Set Keyboard Layout

Set the keyboard layout to German: 

localectl set-keymap de

List Available Locales 

localectl list-locales

List Available Keymaps 

localectl list-keymaps

Quick Reference Cheat Sheet
