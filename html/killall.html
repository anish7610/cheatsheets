Linux Command: killall

The killall command is used to terminate processes by name. It sends signals to all processes running any of the specified commands.

Common Options

Option Description
-e Exact match with the process name.
-i Interactive mode; prompts for confirmation before killing each process.
-v Verbose; reports if the signal has been successfully sent.
-u Restrict to processes from a specified user.
-s Specify a signal to send (e.g., SIGTERM , SIGKILL ).

Usage Examples 

# Terminate all instances of a program
killall firefox

# Terminate processes with confirmation
killall -i firefox

# Use verbose mode to confirm actions
killall -v firefox

# Send a specific signal (e.g., SIGKILL) to all instances
killall -s SIGKILL firefox

# Target processes of a specific user
killall -u username firefox

# Use exact match for the process name
killall -e firefox

Cheat Sheet 

# Kill all processes by name
killall 

# Kill with confirmation
killall -i 

# Verbose mode
killall -v 

# Send specific signal
killall -s
