Linux `msgmerge` Command

The `msgmerge` command is used to update message catalogs by merging the textual translations from a portable object file (.po) with the new strings from a portable object template file (.pot).

Common Options

Option Description
-U, --update Update the existing PO file directly.
-o FILE, --output-file=FILE Write output to specified file instead of standard output.
--backup=CONTROL Make a backup before overwriting, using the specified method.
--previous Keep previous strings of changed messages.
--no-wrap Do not wrap long message lines.

Usage Examples 

msgmerge -U translations.po template.pot
msgmerge -o new_translations.po translations.po template.pot
msgmerge --backup=numbered translations.po template.pot
msgmerge --previous translations.po template.pot

Cheat Sheet 

msgmerge -U [PO_FILE] [POT_FILE]
msgmerge -o [OUT_FILE] [PO_FILE] [POT_FILE]
msgmerge --backup=none [PO_FILE] [POT_FILE]
msgmerge --no-wrap [PO_FILE] [POT_FILE]
