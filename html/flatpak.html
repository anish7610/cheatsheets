Flatpak Command Overview

Description

Flatpak is a software utility for software deployment, application virtualization, and package management. It provides a sandbox environment in which users can run application software in isolation from the rest of the system.

Common Options

Option Description
install Install a Flatpak application or runtime.
uninstall Remove an installed application or runtime.
list List installed apps and/or runtimes.
run Run an installed application.
search Search for an app in the repositories.
update Update one or more applications or runtimes.
info Show information about an app or runtime.

Usage Examples 

# Install an application
flatpak install flathub org.mozilla.firefox

# Uninstall an application
flatpak uninstall org.mozilla.firefox

# List all installed applications
flatpak list

# Run an application
flatpak run org.mozilla.firefox

# Search for an application in the repository
flatpak search vlc

# Update all installed applications
flatpak update

# Show information about an installed application
flatpak info org.mozilla.firefox

Cheat Sheet 

flatpak install [repo] [package]  # Install app
flatpak uninstall [package]      # Uninstall app
flatpak list                     # List apps
flatpak run [package]            # Run app
flatpak search [term]            # Search app
flatpak update                   # Update apps
flatpak info [package]           # App info
