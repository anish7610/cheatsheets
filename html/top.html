Linux top Command

Description

The top command provides a dynamic, real-time view of system processes, showing key performance statistics such as CPU and memory usage.

Common Options

Option Description
-d seconds Set the delay between updates (default is 3 seconds).
-b Run in batch mode for easier parsing by other programs or scripts.
-n number Limit the number of iterations; useful with -b option.
-u user Show only processes for a specific user.

Usage Examples 

# Start top with a 5-second delay between updates
top -d 5

# Run top in batch mode, displaying only two iterations
top -b -n 2

# Display processes for a specific user
top -u alice

Cheat Sheet 

# Start top with default settings
top

# Set refresh delay
top -d 3

# Batch mode
top -b

# Limit iterations
top -n 5

# Filter by user
top -u username
