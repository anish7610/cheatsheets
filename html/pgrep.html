Linux `pgrep` Command

Description

`pgrep` is a command-line utility that searches for processes running on a system based on a given pattern. It returns the process IDs (PIDs) of those processes.

Common Options

Option Description
-f Match against the full argument lists of processes.
-l Display the process name alongside the PID.
-u Match only processes whose effective user ID is listed.
-v Negate the match. Select non-matching processes.
-x Match exactly the pattern.

Usage Examples 

# Find PIDs of all `ssh` processes
pgrep ssh

# Find PIDs and names of processes matching the full command line `java`
pgrep -lf java

# Find PIDs of processes belonging to the user `john`
pgrep -u john

# Get PIDs of processes not matching `nginx`
pgrep -v nginx

# Match exact process name `bash`
pgrep -x bash

Quick Reference 

pgrep -f [pattern]       # Match full command line
pgrep -l [pattern]       # Display PID and name
pgrep -u [user]          # Match by user
pgrep -v [pattern]       # Negate match
pgrep -x [pattern]       # Exact match
