Linux `install` Command

The install command is used to copy files and set attributes. It's commonly used to copy files to system directories, set file permissions, and attributes.

Common Options

Option Description
-d Create directories instead of copying files.
-m Set file mode (permissions), e.g., -m 755 .
-o Set owner of the files.
-g Set group ownership.
-t Specify target directory.

Usage Examples 

# Copy a file with specific permissions
install -m 644 example.txt /usr/local/share/

# Create a directory and set permissions
install -d -m 755 /usr/local/myapp/

# Copy multiple files to a target directory
install -t /usr/local/bin/ script1.sh script2.sh

# Copy a file and set owner and group
install -o user -g group example.txt /home/user/

Cheat Sheet 

install -m [mode] [source] [destination]
install -d -m [mode] [directory]
install -t [target] [source...]
