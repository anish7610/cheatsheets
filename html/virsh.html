Virsh Command Guide

Description

Virsh is a command-line interface tool for managing virtual machines and interacting with the virtualization layer of Linux systems. It primarily works with QEMU/KVM and libvirt.

Common Options

Option Description
list List all running virtual machines.
start <vm-name> Start a specified virtual machine.
shutdown <vm-name> Shutdown a specified virtual machine gracefully.
destroy <vm-name> Forcefully stop a specified virtual machine.
autostart <vm-name> Set a virtual machine to start automatically at boot.
undefine <vm-name> Remove a virtual machine definition.

Usage Examples 

# List running VMs
virsh list

# Start a VM named 'my-vm'
virsh start my-vm

# Shutdown a VM named 'my-vm'
virsh shutdown my-vm

# Force stop a VM named 'my-vm'
virsh destroy my-vm

# Set 'my-vm' to autostart
virsh autostart my-vm

# Remove the VM definition of 'my-vm'
virsh undefine my-vm
