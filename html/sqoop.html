Sqoop Command

Description

Sqoop is a tool designed for efficiently transferring bulk data between Apache Hadoop and structured datastores such as relational databases.

Common Options

Option Description
--connect JDBC connection string for the database to connect.
--username Username to connect to the database.
--password Password to connect to the database.
--table Table to read from the database.
--target-dir HDFS destination directory for importing data.
--export-dir HDFS source directory for exporting data.
--incremental Mode for incremental imports (append, lastmodified).
--split-by Column of the table used to split work units.

Usage Examples 

sqoop import \
--connect jdbc:mysql://db.example.com/testdb \
--username testuser \
--password testpass \
--table employees \
--target-dir /user/hadoop/employees

sqoop export \
--connect jdbc:mysql://db.example.com/testdb \
--username testuser \
--password testpass \
--table employees \
--export-dir /user/hadoop/employees_export \
--input-fields-terminated-by ','

Cheat Sheet 

Import Data:
sqoop import --connect  --username  --password  --table  --target-dir 

Export Data:
sqoop export --connect  --username  --password  --table 
 --export-dir 

Incremental Import:
sqoop import --incremental append --check-column  --last-value
