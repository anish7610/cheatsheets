pidstat Command in Linux

The pidstat command is used to monitor individual tasks managed by the Linux kernel. It provides statistics on CPU, memory usage, and I/O operations for specific processes.

Common Options

Option Description
-u Reports CPU usage.
-r Reports page faults and memory usage.
-d Reports I/O statistics.
-p [PID] Monitors a specific process by PID.
-t Includes thread statistics.
-h Displays the help information.
-V Displays version information.

Usage Examples 

# Monitor CPU usage for all processes
pidstat -u

# Monitor CPU usage for a specific PID
pidstat -u -p 1234

# Monitor memory usage
pidstat -r

# Monitor disk I/O statistics for a specific PID
pidstat -d -p 1234

# Display help information
pidstat -h

Cheat Sheet 

pidstat -u        # CPU usage
pidstat -r        # Memory usage
pidstat -d        # I/O statistics
