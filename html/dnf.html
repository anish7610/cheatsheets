DNF Command Overview

DNF (Dandified YUM) is a package manager for RPM-based distributions. It automatically resolves dependencies and maintains packages.

Common Options

Option Description
install Installs a package.
remove Uninstalls a package.
update Updates all packages or a specific package.
search Searches for a package by name.
list Lists available packages or installed packages.
info Displays information about a package.
clean Cleans cached metadata and packages.

Usage Examples

Install a Package 

# dnf install httpd

Remove a Package 

# dnf remove httpd

Update All Packages 

# dnf update

Search for a Package 

# dnf search nginx

List Installed Packages 

# dnf list installed

Get Package Info 

# dnf info python

Clean Metadata 

# dnf clean all

Cheat Sheet 


# Install a package
dnf install 

# Remove a package
dnf remove 

# Update packages
dnf update []

# Search for a package
dnf search 

# List available/installed packages
dnf list [installed]

# Package information
dnf info 

# Clean cache
dnf clean all
