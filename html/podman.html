Podman Command Overview

Podman is a tool for managing containers and images, offering a command-line interface similar to Docker.

Common Podman Options

Option Description
-d, --detach Run container in background and print container ID.
--name Assign a name to the container.
-p, --publish Publish a container's port to the host.
-v, --volume Bind mount a volume.
--rm Automatically remove the container when it exits.

Usage Examples

Run a Container 

      podman run --name my-container -d my-image

List Running Containers 

      podman ps

Stop a Container 

      podman stop my-container

Remove a Container 

      podman rm my-container

Build an Image 

      podman build -t my-image .

Pull an Image 

      podman pull nginx:latest

Podman Cheat Sheet 

      
Run: podman run [options] IMAGE
Ps: podman ps
Stop: podman stop CONTAINER
Rm: podman rm CONTAINER
Build: podman build -t IMAGE_NAME .
Pull: podman pull IMAGE
