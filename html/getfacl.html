Understanding the Linux Command: getfacl

Description

The getfacl command is used to display the Access Control Lists (ACLs) of files and directories in Linux, providing detailed permissions beyond the traditional owner-group-others model.

Common Options

Option Description
-a Display the access ACL.
-d Display the default ACL, if present.
-e Display output in a more easily parsed format.
-R Recursively display ACLs of all files and directories.
--version Output version information and exit.

Usage Examples 

# Display ACLs for a file
getfacl filename.txt

# Display ACLs for a directory
getfacl /path/to/directory

# Display default ACLs (if any) for a directory
getfacl -d /path/to/directory

# Recursively display ACLs for all files in a directory
getfacl -R /path/to/directory

# Use in a script with a parsable output
getfacl -e filename.txt

Cheat Sheet 

getfacl [options] 
-a    # Access ACL
-d    # Default ACL
-e    # Parsable output
-R    # Recursive
