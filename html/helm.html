Helm Command Overview

The helm command is a package manager for Kubernetes, used to streamline installing and managing Kubernetes applications.

Common Options

Option Description
install Install a chart.
upgrade Upgrade a release to a new version of a chart.
uninstall Uninstall a release.
list List all deployed releases.
repo add Add a new Helm chart repository.
repo update Update information on available charts.
get values Download the values file for a given release.

Usage Examples 

helm repo add stable https://charts.helm.sh/stable
helm repo update
helm install my-release stable/mysql
helm list
helm upgrade my-release stable/mysql
helm uninstall my-release
helm get values my-release

Cheat Sheet 

# Add repo and update
helm repo add [name] [url]
helm repo update

# Install and manage releases
helm install [release-name] [chart]
helm list
helm upgrade [release-name] [chart]
helm uninstall [release-name]
