Getenforce Command

The getenforce command is used to check the current status of SELinux (Security-Enhanced Linux) mode on your system.

Common Options

Option Description
N/A The getenforce command does not have any specific options. It simply outputs the current mode of SELinux.

Usage Examples 

# Check the current SELinux mode
getenforce

Cheat Sheet 

# Display current SELinux mode
getenforce
