Ping Command Overview

Description

The ping command is used to test the reachability of a host on a network and measure the round-trip time for messages sent from the source to the destination.

Common Options

Option Description
-c <count> Send a specific number of packets and then stop.
-i <interval> Wait a specified interval (in seconds) between sending each packet.
-s <size> Specify the number of data bytes to be sent.
-t <ttl> Set the Time To Live for packets.
-4 Use IPv4 only.
-6 Use IPv6 only.

Usage Examples 

$ ping google.com
$ ping -c 4 example.com
$ ping -i 2 -c 5 192.168.1.1
$ ping -s 1000 -c 3 localhost
$ ping -t 64 example.net

Cheat Sheet 

ping <host>
ping -c <count> <host>
ping -i <interval> <host>
ping -s <size> <host>
ping -t <ttl> <host>
