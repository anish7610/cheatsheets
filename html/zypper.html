Zypper Command Guide

Description

Zypper is a command-line tool for managing packages in openSUSE and SUSE Linux Enterprise. It allows you to install, update, and remove packages, as well as manage repositories.

Common Options

Option Description
install package_name Install a package.
remove package_name Remove a package.
update Update all installed packages.
list-updates List available package updates.
search keyword Search for packages matching a keyword.
clean Clean the local cache.
refresh Refresh the repository metadata.

Usage Examples 

# Install a package
zypper install vim

# Remove a package
zypper remove vim

# Update all packages
zypper update

# List updates
zypper list-updates

# Search for a package
zypper search curl

# Clean cache
zypper clean

# Refresh repositories
zypper refresh

Cheat Sheet 

Install:  zypper in package
Remove:   zypper rm package
Update:   zypper up
Search:   zypper se keyword
Clean:    zypper clean
Refresh:  zypper ref
