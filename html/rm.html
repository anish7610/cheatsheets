Linux Command: rm

Description

The rm command is used to remove files or directories in a Linux system.

Common Options

Option Description
-i Prompt before every removal.
-r Remove directories and their contents recursively.
-f Force removal without prompting for confirmation.
-v Verbose output showing what is being done.

Usage Examples 

# Remove a single file
rm filename.txt

# Remove a file with confirmation
rm -i filename.txt

# Remove a directory and its contents
rm -r dirname/

# Force remove a file without prompt
rm -f importantfile.txt

# Verbose removal of multiple files
rm -v file1.txt file2.txt

Cheat Sheet 

rm file          # Remove file
rm -i file       # Confirm remove
rm -r dir        # Recursively remove directory
rm -f file       # Force remove
rm -v file       # Verbose remove
