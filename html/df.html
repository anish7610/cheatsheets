Linux Command: df

Description

The df command displays the amount of disk space available on file systems, along with other relevant information.

Common Options

Option Description
-h Displays sizes in a human-readable format (e.g., KB, MB).
-a Includes dummy file systems.
-T Shows the file system type.
-i Displays inode usage instead of block usage.
--total Produces a grand total at the end of the output.

Usage Examples

Basic Disk Usage 

df -h

Displays disk space in a human-readable format.

Include All File Systems 

df -a

Shows all file systems, including dummy ones.

File System Types 

df -T

Includes the type of each file system in the output.

Inode Usage Information 

df -i

Displays inode usage for each file system.

Cheat Sheet 


df -h          # Human-readable disk usage
df -a          # All file systems
df -T          # Show file system types
df -i          # Inode usage
df --total     # grand total
