Borg Command Guide

Description

borg is a powerful deduplicating backup tool that provides efficient and secure data backup solutions.

Common Options

Option Description
init Initialize a new backup repository.
create Create a new backup archive.
extract Restore files from an archive.
list List archives in a repository.
prune Delete old archives to free space.
key Manage repository keys.

Usage Examples

Initialize a Repository 

borg init /path/to/repo

Create a Backup Archive 

borg create /path/to/repo::archive-name /path/to/data

Extract an Archive 

borg extract /path/to/repo::archive-name

List Archives 

borg list /path/to/repo

Prune Old Archives 

borg prune -r /path/to/repo --keep-daily=7 --keep-weekly=4 --keep-monthly=6

Cheat Sheet 


# Initialize repository
borg init /path/to/repo

# Create backup
borg create /repo::archive /data

# Extract archive
borg extract /repo::archive

# List archives
borg list /repo

# Prune archives
borg prune -r /repo --keep-daily=7 --keep-weekly=4 --keep-monthly=6
