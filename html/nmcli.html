nmcli Command Overview

nmcli is a command-line tool for interacting with NetworkManager to manage network connections on Linux systems. It can control network interfaces, connections, and report on network status.

Common Options

Option Description
nmcli general status Show overall status of NetworkManager.
nmcli device status List all network interfaces and their status.
nmcli connection show Display active network connections.
nmcli connection up <name> Activate a network connection by name.
nmcli connection down <name> Deactivate a network connection by name.

Usage Examples

Check Network Status 

nmcli general status

List All Devices 

nmcli device status

Show Active Connections 

nmcli connection show --active

Connect to a Wi-Fi Network 

nmcli device wifi connect "SSID" password "password"

Bring Down an Active Connection 

nmcli connection down <connection-name>
