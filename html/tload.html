Linux Command: tload

The tload command displays a graph of system load average over time in the terminal.

Common Options

Option Description
-d, --delay <seconds> Set delay between updates (default is 1 second).
-s, --scale <value> Specify scaling value for the graph (auto if not provided).
-V, --version Display version information and exit.
--help Display help message and exit.

Usage Examples 

# Display load average graph with default settings
tload

# Update every 2 seconds
tload -d 2

# Set scale value to 5
tload -s 5

Cheat Sheet 

# tload: Display system load average
tload [-d seconds] [-s value]
