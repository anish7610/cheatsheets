Linux `tail` Command

Description

The tail command is used to display the end of a file or stream.

Common Options

Option Description
-n number Show the last number of lines.
-f Output appended data as the file grows.
-c number Show the last number of bytes.
--help Display help and exit.

Usage Examples 

# Display the last 10 lines of a file
tail example.txt

# Display the last 20 lines of a file
tail -n 20 example.txt

# Follow a log file as it grows
tail -f /var/log/syslog

# Display the last 50 bytes of a file
tail -c 50 example.txt

Cheat Sheet 

tail [options] 
-n   # lines
-c   # bytes
-f           # follow
