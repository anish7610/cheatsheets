Linux Command: vipw

The vipw command is used to safely edit the /etc/passwd and /etc/shadow files, which store user account information on Linux systems.

Common Options

Option Description
-g Invoke the editor on the /etc/group file instead.
-s Invoke the editor on the /etc/shadow file instead.
-q Quiet mode; fewer messages.

Usage Examples

Example 1: Edit Password File 

vipw

Example 2: Edit Shadow File 

vipw -s

Example 3: Edit Group File 

vipw -g

Cheat Sheet 


vipw      # Edit /etc/passwd
vipw -s   # Edit /etc/shadow
vipw -g   # Edit /etc/group
