Linux `az` Command Guide

The `az` command is the Azure CLI tool used to manage Azure resources from the command line.

Common Options

Option Description
-h, --help Show help information for commands and subcommands.
--version Display the version of the Azure CLI.
--output <format> Specify output format: json, table, tsv, yaml.
--verbose Increase logging verbosity for debug purposes.
--query <JMESPath> Filter results using JMESPath query.

Usage Examples 

az login
# Authenticate with Azure

az group create --name myResourceGroup --location eastus
# Create a new resource group

az vm create --resource-group myResourceGroup --name myVM --image UbuntuLTS
# Create a new virtual machine with Ubuntu

az resource list --output table
# List all resources in the current subscription in a table format

az account list --query "[].{Name:name, State:state}" --output table
# List all accounts with name and state
