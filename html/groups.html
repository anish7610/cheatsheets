Linux `groups` Command

Description

The groups command displays the groups a specified user or the current user belongs to.

Common Options

Option Description
No Option Display groups for the current user.
username Display groups for the specified user.

Usage Examples 

# Display groups for the current user
groups

# Display groups for a specific user, e.g., "alice"
groups alice

Cheat Sheet 

# Display groups for current user
groups

# Display groups for a specific user
groups [username]
