mongosh Command Overview

The mongosh command provides an interactive JavaScript shell for MongoDB, allowing users to interact with their MongoDB databases.

Common Options

Option Description
--host Specifies the hostname of the MongoDB server to connect to.
--port Specifies the port number to connect to on the MongoDB server.
-u or --username Provides the username for authentication.
-p or --password Provides the password for authentication.
--authenticationDatabase Specifies the database that holds the user's credentials.

Usage Examples 

mongosh --host localhost --port 27017

Connect to a MongoDB instance running locally on the default port. 

mongosh --host example.com --username myUser --password myPassword

Connect to a remote MongoDB server with authentication. 

mongosh -u admin -p secret --authenticationDatabase admin

Authenticate as an admin user using a specific authentication database.

Cheat Sheet 


mongosh [options] []
--host          # MongoDB server hostname
--port          # MongoDB server port
-u, --username  # Username
-p, --password   # Password
--authenticationDatabase  # Auth DB
