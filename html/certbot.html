Certbot Command Guide

Description

Certbot is a tool to automate the process of obtaining and renewing SSL certificates from Let's Encrypt, enhancing security and ensuring encrypted communication.

Common Options

Option Description
renew Automatically renew all installed certificates that are near expiry.
certonly Obtain a certificate without installing it.
--apache Use the Apache plugin for certificate installation.
--nginx Use the Nginx plugin for certificate installation.
--dry-run Simulate obtaining a certificate, useful for testing.
--standalone Run a standalone web server for obtaining a certificate.
--webroot Use webroot to obtain a certificate.

Usage Examples 

sudo certbot --nginx -d example.com -d www.example.com
sudo certbot certonly --webroot -w /var/www/html -d example.com -d www.example.com
sudo certbot renew
sudo certbot renew --dry-run

Cheat Sheet 

certbot renew
certbot certonly --nginx -d example.com
certbot --apache -d example.com
certbot --dry-run
