Visudo Command Overview

Description

The visudo command is a utility to safely edit the /etc/sudoers file. It locks the file and performs a syntax check to prevent mistakes.

Common Options

Option Description
-c Check the syntax of the /etc/sudoers file.
-s Enable strict checking of the /etc/sudoers file syntax.
-V Display version information of visudo .
-f <file> Edit the specified sudoers file instead of the default.

Usage Examples 

# Edit the default sudoers file with safety checks
visudo

# Check the syntax of the sudoers file
visudo -c

# Edit a custom sudoers file
visudo -f /etc/custom_sudoers

# Enable strict syntax checking
visudo -s
