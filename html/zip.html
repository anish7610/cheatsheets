Linux Command: zip

Description

The zip command is used to create compressed archives, combining multiple files into a single file with compression to save space.

Common Options

Option Description
-r Recursively include files in subdirectories.
-d Delete files from the archive.
-u Update existing files in the archive and add new ones.
-l Show the archive contents without extracting.
-e Encrypt the zip file with a password.
-q Quiet mode. Suppress output.

Usage Examples 



zip -r archive.zip /path/to/directory


zip archive.zip file1.txt file2.txt


zip -u archive.zip file1.txt


zip -d archive.zip file2.txt


zip -e secure.zip /path/to/files
