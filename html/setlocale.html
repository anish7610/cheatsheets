setlocale Command

The setlocale command in Linux is used to modify or retrieve the current locale settings of an environment. It helps in setting language, currency, date format, and more.

Common Options

Option Description
LC_CTYPE Character classification and case conversion.
LC_COLLATE String comparison rules.
LC_TIME Date and time formatting.
LC_NUMERIC Numeric formatting (e.g., decimal point).
LC_MONETARY Currency formatting.
LC_MESSAGES Format and language of system messages.

Usage Examples 

# Set locale for all categories
export LANG=en_US.UTF-8

# Set locale only for numeric formatting
export LC_NUMERIC=de_DE.UTF-8

# Display current locale settings
locale

Cheat Sheet 

# Set locale
export VARIABLE=locale_value

# Check current locales
locale
