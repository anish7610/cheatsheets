Docker Compose Command

Description

docker-compose is a tool for defining and running multi-container Docker applications. It uses a YAML file to configure services, networks, and volumes needed for your application.

Common Options

Option Description
up Builds, (re)creates, starts, and attaches to containers for a service.
down Stops containers and removes containers, networks, volumes, and images created by up .
build Build or rebuild services.
start Starts existing containers for a service.
stop Stops running containers without removing them.
restart Restarts all stopped and running services.
logs View output from containers.
ps Lists containers.

Usage Examples 

# Start all services defined in docker-compose.yml
docker-compose up

# Start services in the background (detached mode)
docker-compose up -d

# Stop and remove resources started with up
docker-compose down

# Rebuild services
docker-compose build

# View logs from all services
docker-compose logs

# List all running containers
docker-compose ps

Cheat Sheet 

docker-compose up       # Start services
docker-compose down     # Stop and remove services
docker-compose build    # Build services
docker-compose start    # Start existing services
docker-compose stop     # Stop services
docker-compose restart  # Restart services
docker-compose logs     # View container logs
docker-compose ps       # List containers
