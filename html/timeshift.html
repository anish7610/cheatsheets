Timeshift Command Overview

Timeshift is a Linux command-line utility designed to create and manage system snapshots. It allows users to easily restore their systems to a previous state, helping recover from system failures or unstable configurations.

Common Options

Option Description
--create Create a new snapshot.
--list List all available snapshots.
--restore Restore the system to a specified snapshot.
--delete Delete a specified snapshot.
--check Check for errors in snapshots.
--help Display help information.

Usage Examples 

# Create a snapshot
sudo timeshift --create

# List all snapshots
sudo timeshift --list

# Restore a specific snapshot
sudo timeshift --restore --snapshot '2023-10-05_12-00-00'

# Delete a snapshot
sudo timeshift --delete --snapshot '2023-10-01_09-00-00'
