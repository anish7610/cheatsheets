apache2 Command Overview

Description

The apache2 command is used to control the Apache HTTP Server. It allows starting, stopping, and managing the server's operations.

Common Options

Option Description
-k start Start the Apache server.
-k stop Stop the Apache server.
-k restart Restart the Apache server.
-k graceful Gracefully restart the server without dropping connections.
-v Show the version number and exit.
-S Show the settings and virtual host configuration.
-t Test the configuration for syntax errors.

Usage Examples 

# Start the Apache server
sudo apache2 -k start

# Stop the Apache server
sudo apache2 -k stop

# Restart the Apache server
sudo apache2 -k restart

# Gracefully restart the server
sudo apache2 -k graceful

# Check Apache version
apache2 -v

# Show configuration and virtual hosts
sudo apache2 -S

# Test configuration for syntax errors
sudo apache2 -t

Cheat Sheet
