VBoxManage Command

vboxmanage is a command-line tool for managing VirtualBox virtual machines (VMs). It allows users to create, modify, start, stop, and control VMs and their settings.

Common Options

Option Description
createvm Create a new VM.
startvm Start a VM.
controlvm Control running VMs (e.g., pause, reset).
modifyvm Modify VM settings.
list List VMs and their details.
showvminfo Display detailed information about a VM.

Usage Examples

Create a New VM 

vboxmanage createvm --name "MyVM" --register

Start a VM 

vboxmanage startvm "MyVM" --type headless

Modify VM Settings 

vboxmanage modifyvm "MyVM" --memory 2048 --cpus 2

List All Registered VMs 

vboxmanage list vms

Show VM Information 

vboxmanage showvminfo "MyVM"
