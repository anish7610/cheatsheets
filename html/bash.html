Bash Command Guide

Description

Bash is a Unix shell and command language, often used as the default login shell for Linux. It provides a command-line interface for running scripts and managing system tasks.

Common Options

Option Description
-c "string" Run the commands in "string" .
--version Display version information and exit.
-x Print commands and their arguments as they are executed.
--login Start a login shell.
-i Run an interactive shell session.

Usage Examples 

# Run a simple command
bash -c "echo Hello World"

# Start an interactive session
bash -i

# Debug a script
bash -x script.sh

# Get version info
bash --version

Cheat Sheet 

-c "string" : Run command
--version     : Show version
-x            : Debug
--login       : Login shell
-i            : Interactive shell
