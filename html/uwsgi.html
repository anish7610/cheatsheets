uwsgi Command Overview

uwsgi is a versatile application server for deploying web applications. It supports WSGI, FastCGI, and HTTP protocols, and is commonly used to host Python web applications.

Common Options

Option Description
--http Start a HTTP server on the specified address.
--wsgi-file Load a WSGI file.
--module Load a WSGI module.
--master Enable the master process.
--processes Number of worker processes to spawn.
--threads Number of threads per worker.
--socket Bind to a given UNIX/TCP socket.
--plugin Load a specified plugin.

Usage Examples 


# Start a simple HTTP server on port 8080 with a specific WSGI file
uwsgi --http :8080 --wsgi-file myapp.py

# Run a WSGI application with 4 processes and 2 threads per process
uwsgi --socket :3031 --wsgi-file myapp.py --processes 4 --threads 2

# Use a WSGI module with a master process and 8 workers
uwsgi --http :9090 --module myapp:app --master --processes 8

Cheat Sheet 


# Basic Structure: uwsgi [options]
uwsgi --http :port --wsgi-file file.py
uwsgi --socket :port --module module:app
uwsgi --http :port --master --processes n --threads m --wsgi-file file.py
