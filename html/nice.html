Linux Command: nice

Description

The nice command in Linux sets the scheduling priority of a command, allowing you to run it with a modified niceness value. This affects the process's priority in using CPU resources.

Common Options

Option Description
-n, --adjustment=N Sets the niceness to N . Positive values decrease priority, while negative ones increase it (superuser only).
-h, --help Displays help information about the nice command.
-V, --version Shows the version of the nice package.

Usage Examples 

# Run a command with a niceness of 10
nice -n 10 myscript.sh

# Increase priority of a command (superuser)
sudo nice -n -5 myprogram

Cheat Sheet 

# Syntax
nice [OPTION] [COMMAND]

# Example: Lower priority
nice -n 10 [COMMAND]

# Example: Increase priority (root)
sudo nice -n -5 [COMMAND]
