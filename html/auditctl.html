auditctl Command Guide

Description

The auditctl command is used to control the Linux Audit daemon, allowing you to manage audit rules and settings.

Common Options

Option Description
-a Add a rule to the end of a rule list.
-A Add a rule to the beginning of a rule list.
-d Delete a rule from the list.
-l List all loaded audit rules.
-s Show the status of the audit daemon.

Usage Examples 

# Add a file watch for changes to /etc/passwd
auditctl -a always,exit -F path=/etc/passwd -F perm=wa

# List all current audit rules
auditctl -l

# Delete all audit rules
auditctl -D

# Add a system call audit rule
auditctl -a always,exit -S open -F success=0

Cheat Sheet 

# Add to end: auditctl -a [type]
# Add to begin: auditctl -A [type]
# Delete rule: auditctl -d [rule]
# List rules: auditctl -l
# Show status: auditctl -s
