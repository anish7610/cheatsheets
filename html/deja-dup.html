Deja Dup Command Guide

Description

deja-dup is a simple backup tool in Linux that provides a user-friendly way to back up and restore files. It supports scheduling, incremental backups, encryption, and cloud storage.

Common Options

Option Description
--backup Start a backup process immediately.
--restore Begin the restore process for files.
--prompt Show a prompt for further actions if necessary.
--display=DISPLAY Set the X display to use.
--version Display the version of Deja Dup.

Usage Examples 

# Start a backup immediately
deja-dup --backup

# Restore files from the latest backup
deja-dup --restore

# Start a backup with prompts for user input
deja-dup --backup --prompt

Cheat Sheet 

deja-dup --backup         # Start immediate backup
