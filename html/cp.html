Linux `cp` Command

Description

The cp command is used to copy files and directories in Linux.

Common Options

Option Description
-r or --recursive Copy directories recursively
-i or --interactive Prompt before overwriting
-u or --update Copy only when the SOURCE file is newer
-v or --verbose Explain what is being done
-a or --archive Preserve attributes and copy recursively (same as -dR --preserve=all)

Usage Examples 

# Copy a file to another directory
cp file.txt /path/to/destination/

# Copy a directory recursively
cp -r /source/directory /destination/directory

# Copy a file interactively (prompt before overwrite)
cp -i file.txt /path/to/destination/

# Copy a file and show detailed output
cp -v file.txt /path/to/destination/

# Copy only if the destination file is older
cp -u file.txt /path/to/destination/

Cheat Sheet 

cp source_file destination_file
cp -r source_dir destination_dir
cp -i file destination
cp -v file destination
cp -u file destination
