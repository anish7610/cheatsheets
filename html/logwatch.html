Logwatch Command Overview

Logwatch is a log parsing tool that analyzes and summarizes system log files, providing detailed reports about various system activities.

Common Options

Option Description
--detail Specifies the level of detail in the report (e.g., Low, Med, High).
--service Define specific services to include (or "All" for every service).
--range Set the time range for the logs (e.g., Yesterday, Today).
--output Choose the report output method (e.g., stdout, file).
--logfile Specify a particular log file to analyze.

Usage Examples 


# Generate a detailed report for all services from yesterday
logwatch --detail High --service All --range Yesterday

# Get a summary report of sshd logs only
logwatch --service sshd --detail Med

# Output a report to a specific file
logwatch --range Today --output file > /var/log/logwatch_report.txt
