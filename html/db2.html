Linux `db2` Command

Description

The db2 command is used for interacting with IBM Db2 databases, allowing users to execute database commands and queries.

Common Options

Option Description
-c Continue processing after an error occurs.
-t Terminate commands with a semicolon.
-v Display command output verbosely.
-x Generate XML output for the commands.
-r file Redirect output to the specified file.

Usage Examples 

# Connect to a database
db2 connect to sample

# Run a simple query
db2 "SELECT * FROM employees"

# Execute commands from a file, terminate with semicolons
db2 -tvf script.sql

# Redirect output to a file
db2 -r output.log "SELECT * FROM employees"

Cheat Sheet 

# Connect to a database
db2 connect to dbname

# Execute commands with continuation on error
db2 -c "command"

# Query with semicolon termination
db2 -t "query"

# Run a script file
db2 -tvf file.sql
