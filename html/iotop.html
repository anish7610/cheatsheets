Linux Command: iotop

Description

iotop is a command-line tool for monitoring I/O usage by processes on Linux, providing a real-time view similar to top .

Common Options

Option Description
-o Show only processes or threads actually doing I/O.
-b Run in batch mode, suitable for logging.
-n NUM Set the number of iterations before quitting (used with -b ).
-d SEC Delay between iterations in seconds.
-p PID Monitor specific process by PID.

Usage Examples 

# Monitor all I/O operations in real-time
iotop

# Show only processes performing I/O
iotop -o

# Run 10 iterations in batch mode with 1-second intervals
iotop -b -n 10 -d 1

# Monitor I/O for a specific process with PID 1234
iotop -p 1234

Cheat Sheet 

iotop -o      # Active I/O only
iotop -b -n 5  # Batch mode, 5 iterations
iotop -d 2     # 2-second delay
iotop -p 1234  # Specific PID
