Linux Command: npm

Description

npm (Node Package Manager) is a package manager for JavaScript, enabling the installation, upgrade, configuration, and removal of packages from the Node.js ecosystem.

Common Options

Option Description
install Install a package and add to package.json as a dependency.
update Update all or specified packages to their latest version.
uninstall Remove a package from the project's dependencies.
init Create a package.json file interactively.
list Display installed packages in the current project.

Usage Examples

Install a Package 

npm install express

Update a Package 

npm update lodash

Uninstall a Package 

npm uninstall express

Initialize a New Project 

npm init

List Installed Packages 

npm list

Cheat Sheet 


npm install [package]   # Install package
npm update [package]    # Update package
npm uninstall [package] # Remove package
npm init                # Create package.json
npm list                # List installed packages
