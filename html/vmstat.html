vmstat Command Guide

Description

The vmstat command reports virtual memory statistics, providing information about system processes, memory, paging, block IO, traps, and CPU activity.

Common Options

Option Description
-a Display active and inactive memory.
-s Display a table of various event counters and memory statistics.
-d Report disk statistics.
-p Report disk statistics for a specified device.
-w Display a table of system swap activity.
-f Report the number of forks since boot.

Usage Examples 

# Display system statistics every 2 seconds
vmstat 2

# Display memory statistics including active and inactive memory
vmstat -a

# Display the number of forks since system boot
vmstat -f

# Display disk statistics
vmstat -d

# Display statistics for a specific disk
vmstat -p sda

Cheat Sheet 

vmstat [options] [delay [count]]
-a : Active/inactive memory
-s : Memory statistics
-d : Disk statistics
-p : Disk statistics for device
-w : Swap activity
-f : Fork count
