pgAdmin Command Overview

pgAdmin is a popular open-source management tool for administering PostgreSQL and its related databases. It provides a graphical interface for database management and query execution.

Common Options

Option Description
-h Specify the host of the PostgreSQL server
-p Specify the port number (default is 5432)
-U Specify the username to connect with
-d Specify the database name
--version Show the version number of pgAdmin

Usage Examples 


# Connect to a PostgreSQL server on localhost
pgAdmin -h localhost -p 5432 -U myusername -d mydatabase

# Display pgAdmin version
pgAdmin --version

Cheat Sheet 


# Connect: pgAdmin -h  -p  -U  -d
