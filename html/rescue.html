Linux `rescue` Command

Description

The `rescue` command is used to initiate a rescue operation on a Linux system, often used to recover a system that cannot boot normally.

Common Options

Option Description
-n, --new-root Specify a new root filesystem for the rescue operation.
-r, --readonly Mount the rescue filesystem as read-only.
-v, --verbose Enable verbose output for debugging purposes.

Usage Examples 


# Rescue operation with a new root filesystem
rescue -n /dev/sda1

# Perform a rescue operation with the filesystem mounted read-only
rescue -n /dev/sda1 -r

# Verbose output for more details during rescue
rescue -v

Cheat Sheet 


rescue -n /dev/sdX    # Specify new root
rescue -r             # Read-only
rescue -v             # Verbose
