DigitalOcean Command-Line Interface (doctl)

Description

The doctl command is a CLI for managing resources on DigitalOcean. It allows users to interact with droplets, images, networking, and more.

Common Options

Option Description
account get View information about the authenticated userâ€™s account.
droplet list List all droplets in your account.
droplet create Create a new droplet with specified configurations.
droplet delete Delete a droplet by ID.
kubernetes cluster list List all Kubernetes clusters.

Usage Examples 

      
# List all droplets
doctl compute droplet list

# Create a droplet with a specific name and size
doctl compute droplet create example-droplet --size s-1vcpu-1gb

# Delete a droplet by ID
doctl compute droplet delete 12345678

# Check account details
doctl account get

# List all Kubernetes clusters
doctl kubernetes cluster list
