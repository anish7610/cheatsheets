Ausearch

The ausearch command is used to search Linux audit logs generated by the audit daemon. It helps in locating specific events based on various criteria like time, user, and event type.

Common Options

Option Description
-m Search for specific audit message types.
-ts Specify the start time for the search.
-te Specify the end time for the search.
-ui Search for events by user ID.
-ua Search for events by user name.

Usage Examples 

# Search for all events by a specific user
ausearch -ua username

# Search for authentication-related events
ausearch -m USER_AUTH

# Search for events between specific times
ausearch -ts 09:00:00 -te 17:00:00

# Find all events for a specific user ID
ausearch -ui 1001

Cheat Sheet 

Ausearch options:
-m: message type
-ts, -te: start and end time
-ui, -ua: user ID and user name
