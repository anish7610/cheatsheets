Linux `at` Command Overview

The at command is used to schedule one-time execution of a command or script at a later time.

Common Options

Option Description
-f file Read the job from a file instead of standard input.
-m Send mail to the user when the job is done, even if there was no output.
-l List the user's pending jobs. Equivalent to atq .
-d or -r Delete a job, identified by its job number. Equivalent to atrm .

Usage Examples 


# Schedule a job to run at a specific time
echo "backup.sh" | at 02:00

# Using a file to schedule multiple commands
at -f backup_tasks.txt now + 1 day

# Listing pending jobs
at -l

# Removing a specific job by job number
at -r 5

Compact Cheat Sheet 


at TIME             # Schedule a command
at -f FILE TIME     # Schedule commands from a file
at -l               # List jobs
at -r JOB_NUMBER    # Remove a job
