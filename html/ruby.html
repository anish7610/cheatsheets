Linux `ruby` Command

Description

The ruby command runs Ruby scripts and provides an interactive Ruby environment.

Common Options

Option Description
-e 'command' Execute the provided Ruby code.
-v Display the Ruby version.
-w Enable warnings.
-c Check syntax only without executing.
--help Show help information.

Usage Examples 

# Run a Ruby script
ruby my_script.rb

# Execute Ruby code from the command line
ruby -e 'puts "Hello, World!"'

# Check syntax of a Ruby file
ruby -c my_script.rb

# Run Ruby with warnings enabled
ruby -w my_script.rb

# Display the Ruby version
ruby -v

Cheat Sheet 

Run script: ruby script.rb
Eval code: ruby -e 'code'
Version: ruby -v
Warnings: ruby -w script.rb
Syntax check: ruby -c script.rb
