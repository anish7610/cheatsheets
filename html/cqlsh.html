cqlsh Command Guide

Description

The cqlsh command is a Python-based command-line interface for interacting with Apache Cassandra using CQL (Cassandra Query Language).

Common Options

Option Description
-u, --username Specify a username for authenticating with Cassandra.
-p, --password Specify a password for authenticating with Cassandra.
-k, --keyspace Connect to a specific keyspace automatically.
--ssl Enable SSL for connecting to Cassandra.
-f, --file Execute commands from a CQL script file.
--debug Enable debugging mode for detailed logs.

Usage Examples 

# Start cqlsh and connect to the default host and port
cqlsh

# Connect to a specific host on a specific port
cqlsh 192.168.1.100 9042

# Connect using username and password
cqlsh -u myuser -p mypass

# Connect directly to a specific keyspace
cqlsh -k mykeyspace

# Execute CQL commands from a script file
cqlsh -f script.cql

Cheat Sheet 

cqlsh [host [port]] [-u user] [-p pass] [-k keyspace] [--ssl] [-f file]
