Linux Command: setfacl

Description

The setfacl command is used to set file access control lists (ACLs) on files and directories, allowing more granular permission settings compared to traditional Unix file permissions.

Common Options

Option Description
-m Modify an existing ACL or add a new one.
-x Remove a specific ACL entry.
-b Remove all ACL entries.
-k Remove the default ACL.
-R Apply changes recursively.

Usage Examples 

# Add or modify an ACL entry for user 'john' granting read and write permissions.
setfacl -m u:john:rw file.txt

# Remove ACL entry for user 'john'.
setfacl -x u:john file.txt

# Remove all ACL entries.
setfacl -b file.txt

# Set a default ACL recursively on a directory for group 'staff'.
setfacl -R -m d:g:staff:rx /path/to/directory

Cheat Sheet 

setfacl -m u::    # Modify/add ACL for a user
setfacl -x u:          # Remove ACL for a user
setfacl -b                  # Remove all ACLs
setfacl -R -m d:g::   # Recursively set default ACL on directory
