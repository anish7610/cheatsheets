Linux Command: rpm

The rpm command is used for managing packages on RPM-based Linux distributions. It can install, update, remove, and query packages.

Common Options

Option Description
-i, --install Install a package
-U, --upgrade Upgrade a package
-e, --erase Uninstall a package
-q, --query Query package information
-V, --verify Verify a package
--rebuilddb Rebuild the RPM database

Examples 

# Install a package
rpm -i package.rpm

# Upgrade a package
rpm -U package.rpm

# Uninstall a package
rpm -e package_name

# Query if a package is installed
rpm -q package_name

# Verify a package
rpm -V package_name
