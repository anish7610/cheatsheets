Linux `cassandra` Command

The cassandra command is used to start the Apache Cassandra database server, a distributed NoSQL database management system.

Common Options

Option Description
-f Runs Cassandra in the foreground.
-p <file> Records the process ID in the specified file.
-R Forces recovery on the commit logs.
-Dcassandra.config=<url> Specifies the path to the configuration file.

Usage Examples 

# Start Cassandra in the foreground
cassandra -f

# Start Cassandra and log the process ID
cassandra -p /var/run/cassandra.pid

# Start Cassandra with a specific configuration file
cassandra -Dcassandra.config=file:///path/to/cassandra.yaml
