Understanding the `apt-get` Command

`apt-get` is a command-line tool for handling packages in Debian-based Linux distributions. It is used for installing, upgrading, and removing software packages.

Common Options

Option Description
update Update package index files from their sources.
upgrade Install the latest versions of all installed packages.
install [package] Install the specified package(s).
remove [package] Remove the specified package(s), but not configuration files.
purge [package] Remove the specified package(s), including configuration files.
autoremove Remove automatically installed packages that are no longer needed.
clean Clean out the local repository of retrieved package files.

Usage Examples 

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install vim
sudo apt-get remove firefox
sudo apt-get purge apache2
sudo apt-get autoremove
sudo apt-get clean

Quick Cheat Sheet 

update     - Refresh package lists
upgrade    - Update all packages
install    - Add new packages
remove     - Remove packages
purge      - Remove packages and configs
autoremove - Clean up unused packages
clean      - Clear cache
