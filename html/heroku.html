Heroku Command in Linux

Description

The heroku command is used to interact with Heroku's cloud platform for deploying, managing, and scaling applications.

Common Options

Option Description
login Log in to your Heroku account.
create Create a new Heroku app.
push Push code to Heroku for deployment.
logs --tail Stream the live logs of an app.
open Open the app in your default web browser.
config:set Set a configuration variable for the app.
ps:scale Scale the number of dynos for an app.

Usage Examples 

heroku login
heroku create my-app
git push heroku main
heroku logs --tail
heroku open
heroku config:set DEBUG=True
heroku ps:scale web=2

Cheat Sheet 

heroku login
heroku create <app-name>
git push heroku main
heroku logs --tail
heroku open
heroku config:set <KEY>=<VALUE>
heroku ps:scale web=<N>
