Linux `sar` Command

The sar command (System Activity Report) is used to collect, report, or save system activity information. It helps monitor system performance and provides insights into various system metrics.

Common Options

Option Description
-u Report CPU utilization.
-r Report memory usage.
-b Report I/O and transfer rates.
-n ALL Report network statistics.
-P ALL Report statistics for all processors.
-f file Read data from a specified file.
-s time Set start time for data collection.
-e time Set end time for data collection.

Usage Examples 


1. Check CPU utilization every 5 seconds for 3 reports:
   sar -u 5 3

2. Monitor memory usage over a period:
   sar -r 5 3

3. Get network statistics:
   sar -n ALL 5 3

4. Save data to a file and read it later:
   sar -o datafile 5 3
   sar -f datafile

Cheat Sheet 


CPU      : sar -u
Memory   : sar -r
Network  : sar -n ALL
Disk I/O : sar -b
All CPUs : sar -P ALL
