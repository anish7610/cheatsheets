localedef Command Overview

Description

The localedef command is used to compile locale information for use by the system, creating binary locale data from text-based source files.

Common Options

Option Description
-f charmap Specify the character map file.
-i source Specify the input locale source file.
-c Suppress warnings and proceed.
--verbose Provide additional information during processing.
-A aliasfile Specify an alias file.

Usage Examples 

# Create a locale for US English with UTF-8 encoding
localedef -i en_US -f UTF-8 en_US.UTF-8

# Suppress warnings while creating German locale
localedef -c -i de_DE -f UTF-8 de_DE.UTF-8

# Create a verbose French locale
localedef --verbose -i fr_FR -f UTF-8 fr_FR.UTF-8
