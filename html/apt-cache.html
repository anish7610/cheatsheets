apt-cache Command

The apt-cache command is used in Debian-based systems to query and display information from the APT's package cache.

Common Options

Option Description
search <package> Searches for packages matching the specified term.
show <package> Displays detailed information about a specific package.
depends <package> Lists dependencies of a specified package.
rdepends <package> Lists reverse dependencies of a specified package.
policy <package> Shows priority and pinning details for a specified package.

Usage Examples 

# Search for a package related to 'nginx'
apt-cache search nginx

# Show information about the 'curl' package
apt-cache show curl

# List dependencies for 'vim'
apt-cache depends vim

# List packages that depend on 'libc6'
apt-cache rdepends libc6

# Show policy details for 'apache2'
apt-cache policy apache2

Cheat Sheet 

# Search packages
apt-cache search <package>

# Show package details
apt-cache show <package>

# List dependencies
apt-cache depends <package>

# List reverse dependencies
apt-cache rdepends <package>

# Show package policy
apt-cache policy <package>
