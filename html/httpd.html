Linux `httpd` Command Guide

Description

The `httpd` command starts the Apache HTTP Server, an open-source web server used to serve web content. It is a key component of web hosting services.

Common Options

Option Description
-k start Starts the Apache HTTP server.
-k stop Stops the Apache HTTP server.
-k restart Restarts the Apache HTTP server.
-k graceful Gracefully restarts the server, allowing ongoing requests to finish.
-f <file> Uses the specified configuration file.
-v Displays the version information.

Usage Examples 

# Start the Apache HTTP server
httpd -k start

# Stop the server
httpd -k stop

# Restart the server
httpd -k restart

# Gracefully restart with a specific config file
httpd -k graceful -f /path/to/httpd.conf

# Check version
httpd -v

Cheat Sheet 

Start: httpd -k start
Stop: httpd -k stop
Restart: httpd -k restart
Graceful restart: httpd -k graceful
Version: httpd -v
Config: httpd -f <file>
