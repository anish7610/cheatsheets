iperf Command Guide

Description

iperf is a tool for measuring and tuning network performance, by monitoring data transfer rates between hosts.

Common Options

Option Description
-s Run in server mode.
-c <host> Run in client mode, connecting to specified host.
-p <port> Specify server port (default is 5201).
-t <time> Time in seconds for which to transmit (default is 10 seconds).
-u Use UDP instead of TCP.

Usage Examples 

# Start iperf in server mode
iperf -s

# Run iperf client to connect to server at 192.168.1.1
iperf -c 192.168.1.1

# Run iperf client with UDP, connecting to server at 192.168.1.1
iperf -c 192.168.1.1 -u

# Run iperf client for 30 seconds duration
iperf -c 192.168.1.1 -t 30
