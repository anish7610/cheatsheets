fsck Command Overview

fsck is a command-line utility used to check and repair filesystems in Linux. It stands for "File System Consistency Check".

Common Options

Option Description
-A Check all filesystems listed in /etc/fstab .
-C Display a progress bar while checking.
-N Perform a dry run without making any changes.
-R Skip root filesystem when using with -A .
-V Verbose mode; show more detailed output.
-y Automatically answer "yes" to prompts.
-n Automatically answer "no" to prompts.

Usage Examples 

# Check and repair all filesystems in /etc/fstab
sudo fsck -A

# Check specific filesystem by device
sudo fsck /dev/sda1

# Dry run to see what would be changed
sudo fsck -N /dev/sda1

# Automate repair process without prompts
sudo fsck -y /dev/sda1

Cheat Sheet 


# Check all
fsck -A

# Check specific device
fsck /dev/sdX

# Dry run
fsck -N /dev/sdX

# Auto-repair
fsck -y /dev/sdX
