Understanding the `free` Command

The `free` command is used to display the amount of free and used memory in a Linux system. It provides a quick overview of RAM and swap usage.

Common Options

Option Description
-h Display the output in human-readable format (e.g., GB, MB).
-m Show memory usage in megabytes.
-g Show memory usage in gigabytes.
-s [interval] Continuously display memory usage updates every [interval] seconds.

Usage Examples 

# Display memory usage in human-readable format
free -h

# Show memory usage in megabytes
free -m

# Continuously display memory usage every 5 seconds
free -s 5

Cheat Sheet 

free -h   # Human-readable output
free -m   # Display in MB
free -s X # Update every X seconds
