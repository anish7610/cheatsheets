Linux Command: useradd

Description

The useradd command is used to create a new user account on a Linux system. It modifies system files and options related to user management.

Common Options

Option Description
-m Create the user's home directory.
-d Specify the home directory path.
-s Set the user's login shell.
-g Assign the user to a group by name or GID.
-c Add a comment (usually the user's full name).
-G Specify additional groups for the user to join.

Usage Examples 

# Create a user with a home directory
useradd -m username

# Create a user with a specific home directory
useradd -m -d /custom/home username

# Create a user with a specific shell and comment
useradd -m -s /bin/bash -c "John Doe" username

# Create a user and add to specific group
useradd -m -g groupname username

# Create a user with additional group memberships
useradd -m -G group1,group2 username

Cheat Sheet 

# Add user with home, shell, and comment
useradd -m -s /bin/bash -c "Full Name" username

# Add user to primary and secondary groups
useradd -m -g primary_group -G group1,group2 username
