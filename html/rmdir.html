Linux Command: rmdir

Description

The rmdir command is used to remove empty directories in Linux.

Common Options

Option Description
--ignore-fail-on-non-empty Ignore errors for non-empty directories.
-p --parents Remove directory and its parent directories if they are empty.
--verbose Output a message for each directory that is removed.

Examples 

# Remove a single empty directory
rmdir mydirectory

# Remove a directory and its parent if empty
rmdir -p parentdir/mydirectory

# Remove directory and show verbose output
rmdir --verbose mydirectory

# Attempt to remove multiple directories
rmdir dir1 dir2 dir3

Cheat Sheet 

rmdir [OPTION]... DIRECTORY...
-p : Remove parents
--ignore-fail-on-non-empty : Ignore non-empty errors
--verbose : Verbose output
