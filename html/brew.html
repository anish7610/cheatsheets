Brew Command in Linux

Description

The brew command is part of Homebrew, a package manager for Linux and macOS, used to install and manage software packages easily.

Common Options

Option Description
install [package] Installs the specified package.
uninstall [package] Uninstalls the specified package.
update Updates Homebrew itself and all installed formulae.
upgrade Upgrades all outdated packages.
list Lists all installed packages.
search [package] Searches for the specified package.
info [package] Shows information about a specific package.

Usage Examples 

brew install wget
brew uninstall wget
brew update
brew upgrade
brew list
brew search python
brew info node

Cheat Sheet 

brew install [package]    # Install a package
brew uninstall [package]  # Uninstall a package
brew update               # Update Homebrew and formulae
brew upgrade              # Upgrade all packages
brew list                 # List installed packages
brew search [package]     # Search for a package
brew info [package]       # Get info about a package
