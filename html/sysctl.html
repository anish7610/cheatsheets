Linux Command: sysctl

Description

The sysctl command is used to view and manage kernel parameters at runtime. It helps in configuring kernel behavior without rebooting the system.

Common Options

Option Description
-a Display all kernel parameters and their values.
-p [file] Load values from a specified file (default is /etc/sysctl.conf ).
-w Write a parameter value in format variable=value .
-n Display only the value of a parameter.

Usage Examples 

# Display all kernel parameters
sysctl -a

# Set a kernel parameter (e.g., enable IP forwarding)
sysctl -w net.ipv4.ip_forward=1

# Load kernel parameters from a custom file
sysctl -p /path/to/custom.conf

# Display the value of a specific parameter
sysctl -n net.ipv4.ip_forward

Cheat Sheet 

sysctl -a            # Show all parameters
sysctl -w key=val   # Set parameter
sysctl -p [file]    # Load from file
sysctl -n key       # Get parameter value
