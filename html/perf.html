Linux `perf` Command

Description

The perf command is a powerful tool for performance monitoring and analysis on Linux systems. It provides insight into system performance and can help diagnose performance-related issues.

Common Options

Option Description
-e, --event Specify the event to monitor, such as CPU cycles or cache misses.
stat Gather performance statistics on the specified command.
record Record performance data for later analysis.
report Generate a report from recorded performance data.
top Display a real-time view of performance counters.

Usage Examples 

perf stat ls
# Collect statistics for the 'ls' command.

perf record -e cpu-clock ls
# Record 'ls' command with CPU clock event.

perf report
# Display a report of collected data.

perf top
# Show real-time system performance.

Cheat Sheet 

perf stat <command>
perf record -e <event> <command>
perf report
perf top
