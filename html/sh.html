Linux Command: sh

Description

The sh command invokes the Bourne shell, a command interpreter that executes commands read from a command line string, file, or standard input.

Common Options

Option Description
-c Read commands from the given string.
-s Read commands from standard input.
-x Print commands and their arguments as they are executed.
-v Print shell input lines as they are read.
-n Read commands but do not execute them (syntax check).

Examples 

# Execute a script file
sh script.sh

# Run a command using a string
sh -c "echo 'Hello, World!'"

# Debugging: print commands as they execute
sh -x script.sh

# Check syntax without execution
sh -n script.sh

Cheat Sheet 

sh script.sh     # Execute script
sh -c "cmd"       # Run command
sh -x script.sh   # Debug mode
sh -n script.sh   # Syntax check only
sh -s             # Read from stdin
