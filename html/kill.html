Linux Kill Command

Description

The kill command is used to send signals to processes. Most commonly, it's used to terminate a process.

Common Options

Option Description
-l List all signal names.
-s <signal> Send a specified signal instead of the default SIGTERM.
-9 Send the SIGKILL signal, forcibly terminating the process.
-15 Send the SIGTERM signal, gracefully terminating the process.

Usage Examples 

# Terminate a process with PID 1234 using default signal (SIGTERM)
kill 1234

# Forcefully terminate a process with PID 1234
kill -9 1234

# Gracefully terminate a process with PID 1234
kill -15 1234

# List all available signals
kill -l

# Send a specific signal (e.g., SIGHUP) to process with PID 1234
kill -s SIGHUP 1234

Cheat Sheet
