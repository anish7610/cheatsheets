Understanding the Linux Command: scp

Short Description

The scp (Secure Copy Protocol) command is used to copy files and directories securely between hosts on a network. It uses SSH for data transfer, providing the same level of security and authentication.

Common Options

Option Description
-r Recursively copy entire directories.
-P Specify the port to connect to on the remote host.
-i Specify identity file (private key) for authentication.
-C Enable compression.
-v Enable verbose mode to display debugging messages.

Usage Examples 

scp file.txt user@remotehost:/path/to/destination
scp -r /local/dir user@remotehost:/remote/dir
scp -P 2222 file.txt user@remotehost:/path/to/destination
scp -i /path/to/key.pem file.txt user@remotehost:/path/to/destination

Compact Cheat Sheet 


scp source_file user@remote:/destination
scp -r local_dir user@remote:/remote_dir
scp -P port file user@remote:/path
scp -i key.pem file user@remote:/path
