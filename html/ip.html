Linux 'ip' Command

Description

The ip command is used for network configuration. It allows manipulation of routing, devices, policy routing, and tunnels.

Common Options

Option Description
addr Display or manipulate IP addresses and their properties.
link Show and manipulate network devices and their properties.
route Show and manipulate the IP routing table.
netns Manage network namespaces.
neigh Show and manipulate neighbor objects (ARP table).

Usage Examples

Display IP Addresses 

ip addr show

Add an IP Address 

ip addr add 192.168.1.10/24 dev eth0

Delete an IP Address 

ip addr del 192.168.1.10/24 dev eth0

Show Network Interfaces 

ip link show

Bring Interface Up 

ip link set dev eth0 up

Bring Interface Down 

ip link set dev eth0 down

Display Routing Table 

ip route show

Add a Default Route 

ip route add default via 192.168.1.1

Cheat Sheet 


ip addr show
ip addr add [IP]/[CIDR] dev [interface]
ip addr del [IP]/[CIDR] dev [interface]
ip link show
ip link set dev [interface] up|down
ip route show
ip route add default via [gateway]
