mydumper Command Overview

Description

mydumper is a high-performance MySQL backup tool designed to produce consistent snapshots while performing the backup in parallel, efficiently handling large datasets.

Common Options

Option Description
--user Specifies the MySQL user name.
--password Password for the specified MySQL user.
--host Target host for the MySQL server.
--port Port number for the MySQL server connection.
--outputdir Directory to store the backup files.
--threads Number of threads to use for backup.
--compress Compress backup files using gzip.
--verbose Set verbosity level for logging (e.g., 3 for debugging).

Usage Examples

Basic Backup Command 

mydumper --user=root --password=example --host=localhost --outputdir=/backups/mydb

Backup with Compression and Multiple Threads 

mydumper --user=root --password=example --host=localhost --outputdir=/backups/mydb --threads=4 --compress

Backup with Specific Port and Verbosity 

mydumper --user=root --password=example --host=localhost --port=3307 --outputdir=/backups/mydb --verbose=3

Cheat Sheet 


mydumper --user= --password= --host= \ 
--outputdir= [--threads=] [--compress] [--verbose=]
