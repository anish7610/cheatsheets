gcloud Command Overview

The gcloud command is a powerful CLI tool used to manage resources on Google Cloud Platform (GCP), such as Compute Engine, Cloud Storage, and Kubernetes Engine.

Common Options

Option Description
--project Specify the Google Cloud project to use.
--zone Specify the zone for regional resources.
--quiet Disable all interactive prompts.
--format Set the output format (e.g., json, yaml, text).
--help Display help information about the command.

Usage Examples 

# List all projects
gcloud projects list

# Set a default project
gcloud config set project [PROJECT_ID]

# Create a new Compute Engine instance
gcloud compute instances create [INSTANCE_NAME] --zone=[ZONE]

# Deploy an app to App Engine
gcloud app deploy

# List all Compute Engine instances
gcloud compute instances list

# Delete a specific Compute Engine instance
gcloud compute instances delete [INSTANCE_NAME] --zone=[ZONE] --quiet
