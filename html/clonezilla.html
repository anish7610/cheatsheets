Clonezilla Command Overview

Clonezilla is a partition and disk imaging/cloning program for data recovery and deployment.

Common Options

Option Explanation
-q Enable quiet mode to minimize output.
-y Auto confirm prompts during the process.
-r Resizes file systems when restoring image.
--no-restore-mbr Do not restore the Master Boot Record.
-j2 Use multicore compression. Faster compression but high CPU usage.
-n Skip checking the image before restoring.

Usage Examples 

# Clone a disk to an image
sudo clonezilla -q -y -r -j2 -s /dev/sda -o /backup/image.img

# Restore an image to a disk
sudo clonezilla --no-restore-mbr -r -q -y -f /backup/image.img -t /dev/sdb

# Clone one disk to another directly
sudo clonezilla -q -y -r -s /dev/sda -t /dev/sdb

Cheat Sheet 

Clone to Image: -q -y -r -s [source] -o [output.img]
Restore Image: --no-restore-mbr -r -q -y -f [image.img] -t [target]
Direct Clone: -q -y -r -s [source] -t [target]
