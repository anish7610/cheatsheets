Linux Command: syslog

Description

The syslog command is used for logging system messages. It outputs to the system log files managed by the syslogd or rsyslogd service.

Common Options

Option Description
-s Log a message to the system logger.
-p Specifies the priority of the message (e.g., user.info ).
-t Tag the message with a specific identifier.

Usage Examples 

# Log a simple message
logger -s "System update completed"

# Log a message with a specific priority
logger -p local0.notice "Backup completed successfully"

# Log a message with a specific tag
logger -t BACKUP "Backup finished at $(date)"

Cheat Sheet 

# Simple log
logger -s "Message"

# Log with priority
logger -p [facility.level] "Message"

# Log with a tag
logger -t [tag] "Message"
