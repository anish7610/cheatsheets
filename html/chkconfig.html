Linux `chkconfig` Command

Description

The chkconfig command is used to manage the runlevel settings of system services in Linux. It allows you to enable or disable services to start automatically at different runlevels.

Common Options

Option Description
--list Lists all services and their current runlevel status.
--level Specifies the runlevel to configure (e.g., --level 345 ).
--add Adds a new service to be managed by chkconfig .
--del Removes a service from chkconfig management.

Usage Examples

List all services and their runlevel settings 

chkconfig --list

Enable a service to start at runlevels 3, 4, and 5 

chkconfig --level 345 httpd on

Disable a service at all runlevels 

chkconfig httpd off

Add a new service to chkconfig 

chkconfig --add customservice

Remove a service from chkconfig 

chkconfig --del customservice

Cheat Sheet 


chkconfig --list
chkconfig --level   on|off
chkconfig --add 
chkconfig --del
