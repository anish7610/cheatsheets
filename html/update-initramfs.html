update-initramfs

The update-initramfs command is used to generate an initial RAM filesystem (initramfs) image. This image helps in the system boot process by pre-loading necessary drivers and modules.

Common Options

Option Description
-c Create a new initramfs.
-u Update an existing initramfs.
-d Delete an existing initramfs.
-k Specify the kernel version (e.g., -k 5.4.0-26-generic ).
-v Enable verbose mode.
-b Set an alternate boot directory.

Usage Examples 

# Create a new initramfs for the current kernel
sudo update-initramfs -c -k $(uname -r)

# Update the initramfs for a specific kernel version
sudo update-initramfs -u -k 5.4.0-26-generic

# Delete an existing initramfs for a specific kernel
sudo update-initramfs -d -k 4.15.0-20-generic

Cheat Sheet
