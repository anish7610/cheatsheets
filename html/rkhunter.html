rkhunter Command Overview

Description

rkhunter (Rootkit Hunter) is a Unix-based tool that scans for rootkits, backdoors, and possible local exploits.

Common Options

Option Description
-c , --check Performs a check of the local system.
--update Updates the database files used by rkhunter .
--versioncheck Checks if a new version of rkhunter is available.
--report-warnings-only Shows only warning messages in the output.
--propupd Updates the file properties database.

Examples 

# Perform a full system check
sudo rkhunter -c

# Update rkhunter's database files
sudo rkhunter --update

# Check only for warnings to reduce noise
sudo rkhunter --check --report-warnings-only
