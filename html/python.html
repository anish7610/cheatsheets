Linux Command: python

The python command is used to execute Python scripts and open an interactive Python shell.

Common Options

Option Explanation
-c <command> Execute the Python code passed as a string.
-m <module> Run a library module as a script.
-i Inspect interactively after running a script.
-V or --version Show Python version and exit.
--help Show a help message and exit.

Usage Examples 

# Run a Python script
python script.py

# Execute a one-liner command
python -c "print('Hello World')"

# Run a module
python -m http.server

# Enter interactive mode after running a script
python -i script.py

# Check Python version
python --version

Cheat Sheet 

python script.py            # Run script
python -c "cmd"              # Execute command
python -m module             # Run module
python -i script.py          # Interactive mode
python --version             # Python version
