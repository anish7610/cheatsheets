Linux Command: renice

The renice command is used to alter the priority of running processes in Linux. A lower nice value means higher priority (more CPU time).

Common Options

Option Description
-n Specifies the increment to adjust the nice value.
-p Specifies a process ID (PID) to change the nice value for.
-u Specifies a user whose processes' nice values are to be changed.
-g Specifies a process group ID to change the nice value for processes in that group.

Usage Examples 

# Renice a process with PID 1234 to higher priority
renice -n -5 -p 1234

# Lower the priority of all processes by user 'john'
renice -n 5 -u john

# Adjust priority of a process group with GID 5678
renice -n 10 -g 5678

Cheat Sheet 

# Renice process by PID
renice -n NICE_VALUE -p PID

# Renice all processes by user
renice -n NICE_VALUE -u USER

# Renice a process group
renice -n NICE_VALUE -g GID
