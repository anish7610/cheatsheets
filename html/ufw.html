Understanding the Linux Command: ufw

Description

ufw (Uncomplicated Firewall) is a user-friendly interface to manage a Linux firewall. It streamlines the process of configuring firewall rules.

Common Options

Option Description
enable Activate the firewall.
disable Deactivate the firewall.
status Show the status of the firewall.
allow [SERVICE|PORT] Permit incoming connections on a specified service or port.
deny [SERVICE|PORT] Block incoming connections on a specified service or port.
delete [RULE] Remove a specific rule.
reset Revert to default settings.

Examples 

# Enable the firewall
sudo ufw enable

# Allow SSH connections
sudo ufw allow ssh

# Allow HTTP traffic on port 80
sudo ufw allow 80

# Deny all incoming traffic on port 25
sudo ufw deny 25

# Show current firewall status and rules
sudo ufw status

# Disable the firewall
sudo ufw disable

# Delete a rule allowing traffic on port 80
sudo ufw delete allow 80

Quick Reference Cheat Sheet 


sudo ufw enable           # Start firewall
