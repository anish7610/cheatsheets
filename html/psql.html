psql Command Guide

Description

The psql command is a terminal-based front-end to PostgreSQL. It allows you to interact with the PostgreSQL database server using SQL queries and commands.

Common Options

Option Description
-d dbname Specifies the name of the database to connect to.
-U username Connects using the specified username.
-h host Specifies the database server host.
-p port Specifies the port on which the server is listening.
-c command Executes a single command and exits.
-f filename Executes commands from a file.
-l Lists all available databases.

Usage Examples

Connect to a Database 

psql -d mydatabase -U myuser

Execute a Command 

psql -d mydatabase -c "SELECT * FROM mytable;"

Execute Commands from a File 

psql -d mydatabase -U myuser -f commands.sql

List All Databases 

psql -l

Cheat Sheet 


# Connect to a database:
psql -d  -U 

# Execute SQL command:
psql -d  -c ""

# Execute commands from file:
psql -d  -f 

# List all databases:
psql -l
