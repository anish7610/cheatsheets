Hadoop Command Overview

The hadoop command is a tool for interacting with Hadoop, a framework for distributed storage and processing of large data sets. It provides various subcommands for different aspects of Hadoop ecosystem.

Common Options

Option Description
fs Interacts with the Hadoop filesystem (HDFS).
jar Executes Hadoop jobs packaged as JAR files.
version Displays the Hadoop version.
help Provides help for a specific command.

Usage Examples 

hadoop fs -ls /user/hadoop/

Lists files and directories in the specified HDFS path. 

hadoop jar myapp.jar com.example.MyApp input output

Runs a Hadoop job defined in myapp.jar with the main class com.example.MyApp , taking input and output as arguments. 

hadoop version

Displays the current Hadoop version installed.

Cheat Sheet 


hadoop fs -ls /path        # List directory contents
hadoop fs -mkdir /path     # Create directory in HDFS
hadoop fs -put local target # Upload local file to HDFS
hadoop fs -get source local # Download HDFS file to local
hadoop jar file.jar args   # Run Hadoop job JAR
hadoop version             # Show Hadoop version
hadoop help command        # Get help for command
