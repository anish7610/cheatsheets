Bacula Command Overview

Bacula is an open-source backup solution that allows for the management and automation of backup, recovery, and verification of data across a network of computers.

Common Options

Option Description
-c, --config Specify configuration file.
-d, --daemon Start the Bacula daemon.
-t, --test Test configuration for errors.
-u, --user Specify the user to run as.
-g, --group Specify the group to run as.
-s, --simulate Simulate the execution without actual data transfer.

Usage Examples 

# Start Bacula daemon with specified config
bacula -d -c /etc/bacula/bacula-dir.conf

# Test configuration file for errors
bacula -t -c /etc/bacula/bacula-dir.conf

# Simulate a backup job
bacula -s -c /etc/bacula/bacula-dir.conf
