Linux Command: lxc

Description

The lxc command is a part of LXD, a system container manager for running Linux systems in containers. It allows management and operations on LXC containers efficiently.

Common Options

Option Description
launch Start a new container from a given image.
start Start an existing container.
stop Stop a running container.
list List all containers with their statuses.
delete Delete an existing container.
exec Execute a command in a running container.
info Get details about a container.

Usage Examples 

# Launch a new Ubuntu container named "mycontainer"
lxc launch ubuntu:22.04 mycontainer

# Start an existing container
lxc start mycontainer

# Stop a running container
lxc stop mycontainer

# List all containers
lxc list

# Delete a container
lxc delete mycontainer

# Execute a command in a container
lxc exec mycontainer -- bash

# Get information about a container
lxc info mycontainer

Compact Cheat Sheet 

lxc launch IMAGE NAME  # Start NEW container
lxc start NAME          # Start existing container
lxc stop NAME           # Stop running container
lxc list                # List containers
lxc delete NAME         # Delete container
lxc exec NAME -- CMD    # Execute CMD in container
lxc info NAME           # Show container info
