Node Command in Linux

Description

The node command runs Node.js, a JavaScript runtime built on Chrome's V8 JavaScript engine.

Common Options

Option Description
-v , --version Outputs the Node.js version.
-e , --eval Evaluates JavaScript code from the command line.
-p , --print Evaluates and prints the result of JavaScript code.
--check Syntax check the script without executing.
--inspect Activates the inspector, for debugging.

Usage Examples 

node -v
# Outputs the current Node.js version.

node -e "console.log('Hello, World!')"
# Runs a simple script to print "Hello, World!".

node -p "1 + 1"
# Prints the result of the expression, outputting "2".

node script.js
# Executes the JavaScript file script.js.

node --check script.js
# Checks script.js for syntax errors.

Cheat Sheet 

node -v             # Show version
node -e "code"     # Eval code
node script.js     # Run script
node --check file  # Syntax check file
