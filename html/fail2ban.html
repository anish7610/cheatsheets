fail2ban Command

Fail2ban is a security tool that monitors server logs for suspicious activity and bans offending IPs to protect systems from brute-force attacks.

Common Options

Option Description
-h, --help Display help information.
-b, --background Run in the background.
-d, --debug Enable debug mode for diagnostics.
-v, --verbose Increase output verbosity.
-f, --force Force action that may fail if unsuccessful.
-s, --start Start the fail2ban server/process.
-u, --stop Stop the fail2ban server/process.
-r, --restart Restart the fail2ban server/process.

Usage Examples 

# Start fail2ban in the background
fail2ban-client start

# Stop fail2ban service
fail2ban-client stop

# Restart fail2ban service with verbose output
fail2ban-client -v restart

# Check the status of the jail
fail2ban-client status sshd

# Unban an IP address
fail2ban-client set sshd unbanip 192.168.1.100
