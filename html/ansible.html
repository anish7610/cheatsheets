Ansible Command Guide

Description

The ansible command is a tool for managing and automating the configuration of remote systems. It uses SSH connections to communicate with hosts.

Common Options

Option Description
-i / --inventory Specify inventory host path or comma-separated host list.
-m / --module-name Specify the module to run on host.
-a / --args Arguments for the module.
-u / --user Specify the SSH user name.
-k Prompt for SSH password.

Usage Examples 

# Ping all hosts in the inventory
ansible all -i inventory.ini -m ping

# Run a command on a specific host
ansible myserver -i inventory.ini -m shell -a "uptime"

# Copy a file to all hosts
ansible all -i inventory.ini -m copy -a "src=/path/to/local/file dest=/path/to/remote/"

# Install a package on all hosts
ansible all -i inventory.ini -m apt -a "name=htop state=latest" -u root

Cheat Sheet 

# Inventory: -i
# Module: -m
# Args: -a
# User: -u
# Password: -k
