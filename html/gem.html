Linux Command: gem

Description

gem is a command-line tool for managing Ruby gems, which are packages or libraries used to extend the functionality of Ruby applications.

Common Options

Option Description
install Installs a gem into the local repository.
uninstall Removes a gem from the local repository.
update Updates installed gems to the latest version.
list Displays a list of installed gems.
search Searches for gems by name.
help Shows help for gem commands.

Usage Examples 

# Install a specific gem
gem install rails

# Uninstall a specific gem
gem uninstall rails

# Update all installed gems
gem update

# List all installed gems
gem list

# Search for gems with a specific keyword
gem search json

# Display help for gem commands
gem help
