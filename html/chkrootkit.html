Linux Command: chkrootkit

Description

chkrootkit is a tool to locally check for signs of a rootkit infection on a Linux system. It scans system binaries and searches for known rootkit signatures.

Common Options

Option Description
-h Display help information.
-q Run in quiet mode (minimal output).
-r dir Specify an alternate root directory to scan.
-x Perform expert mode tests.
-V Display the version of chkrootkit.

Usage Examples 

# Check the system for rootkits
sudo chkrootkit

# Run chkrootkit with quiet output
sudo chkrootkit -q

# Scan an alternate root directory
sudo chkrootkit -r /mnt/other_system

# Expert mode scan
sudo chkrootkit -x
