HAProxy Command Overview

Description

HAProxy is a high-performance TCP/HTTP load balancer used to distribute workloads across multiple servers.

Common Options

Option Description
-f <file> Specifies the configuration file to use.
-v Displays the version and exits.
-c Checks the configuration file for syntax errors.
-d Starts in debug mode.
-q Quiet mode, suppresses output at startup.

Usage Examples 


# Start HAProxy with a specific configuration file
haproxy -f /etc/haproxy/haproxy.cfg

# Check HAProxy configuration for errors
haproxy -c -f /etc/haproxy/haproxy.cfg

# Start HAProxy in debug mode
haproxy -d -f /etc/haproxy/haproxy.cfg

# Display HAProxy version
haproxy -v

Cheat Sheet 


# Start with config:
haproxy -f <config_file>

# Check config:
haproxy -c -f <config_file>

# Debug mode:
haproxy -d -f <config_file>

# Version:
haproxy -v
