Packer Command Overview

Packer is a tool for creating machine images for multiple platforms from a single source configuration.

Common Options

Option Description
-version Display the version of Packer.
build Build an image from a template file.
-debug Enable debug mode for verbose output.
validate Validate a template file for errors.
inspect Show components of a template file.

Usage Examples

Build an Image 

packer build template.json

Debug Mode 

PACKER_LOG=1 packer build template.json

Validate a Template 

packer validate template.json

Inspect a Template 

packer inspect template.json

Cheat Sheet
