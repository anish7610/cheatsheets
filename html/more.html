Linux Command: more

Description

The more command is used to view the contents of a file one screen at a time, allowing easy navigation through large files.

Common Options

Option Description
-num Lines per page. Displays num lines per screen.
+num Start displaying at line num .
+/string Start displaying at the first occurrence of string .
-d Prompt for next page with instructions.
-c Clear the screen before displaying the next page.

Usage Examples 

# View a file one screen at a time
more filename.txt

# Display 20 lines per page
more -20 filename.txt

# Start displaying from line 5
more +5 filename.txt

# Start displaying from the first occurrence of "error"
more +/error filename.txt

# Use a clearer display with prompts
more -d -c filename.txt

Cheat Sheet 


more filename.txt      # View file page by page
more -num filename.txt # Set custom lines per page
more +num filename.txt # Start at specific line
more +/string filename.txt # Start at search string
