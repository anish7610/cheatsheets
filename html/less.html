Understanding the less Command

The less command in Linux is a text file viewer that allows you to view the contents of a file one screen at a time. It is useful for reading long files without opening an editor and supports both forward and backward navigation.

Common Options

Option Description
-N Show line numbers
-S Chop long lines instead of wrapping
+/search Start at the first occurrence of "search"
-F Exit if the entire file fits on one screen
-R Display raw control characters

Usage Examples 

# View a file with line numbers
less -N filename.txt

# Search for a term and start at its first occurrence
less +/search-term filename.txt

# View a file with lines chopped (horizontal scrolling)
less -S filename.txt

# Quickly view a file and exit if it fits on one screen
less -F filename.txt

# View a file with raw control characters
less -R filename.txt

Cheat Sheet 


less -N filename     # Show line numbers
less -S filename     # No wrap, chop lines
less +/term filename # Search for 'term'
less -F filename     # Auto-exit if fit
less -R filename     # Raw chars
