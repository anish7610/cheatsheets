Linux Command: sqlite3

sqlite3 is a command-line interface for interacting with SQLite databases, allowing users to execute SQL commands directly on database files.

Common Options

Option Description
-init file Read and execute commands from a specified initialization file
-batch Stop interactive I/O and limit output for batch processing
-echo Display each command before execution
-column Format output in columns
-csv Output results in CSV format
-separator sep Specify output field separator (default is '|')
-version Display the SQLite version

Usage Examples 

Create a new database and table:
sqlite3 my_database.db "CREATE TABLE users (id INTEGER PRIMARY KEY, name TEXT, age INTEGER);"

Insert data into the table:
sqlite3 my_database.db "INSERT INTO users (name, age) VALUES ('Alice', 30);"

Display data in column format:
sqlite3 -column -header my_database.db "SELECT * FROM users;"

Export table data to CSV:
sqlite3 -csv my_database.db "SELECT * FROM users;" > users.csv

Cheat Sheet 


sqlite3 database.db - Execute commands on a database file
-init file         - Run commands from a file
-batch             - Non-interactive mode
-column            - Column output format
-csv               - CSV output format
-separator sep     - Set output field separator
-version           - Show SQLite version
