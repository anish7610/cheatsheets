Linux `wget` Command

Description

The `wget` command is a free utility for downloading files from the web using HTTP, HTTPS, and FTP protocols. It is non-interactive, meaning it can work in the background, making it excellent for scheduled and automated tasks.

Common Options

Option Description
-O <file> Save downloaded file to the specified name.
-c Continue getting a partially downloaded file.
-b Run in the background.
-q Turn off wget's output (quiet mode).
-r Recursive download.
-l <depth> Specify recursion level depth.
--limit-rate=<rate> Limit download speed.
--no-check-certificate Do not validate the server's certificate.

Usage Examples 

# Download a single file
wget http://example.com/file.zip

# Download a file and save it as a different name
wget -O newname.zip http://example.com/file.zip

# Resume a partially downloaded file
wget -c http://example.com/largefile.zip

# Download files in the background
wget -b http://example.com/largefile.zip

# Download an entire website
wget -r http://example.com

# Download files at a limited rate of 200kB/s
wget --limit-rate=200k http://example.com/largefile.zip

# Skip certificate check for an HTTPS download
wget --no-check-certificate https://example.com/file.zip
