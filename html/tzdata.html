Linux Command: tzdata

Description

The tzdata command is not typically executed directly but refers to the time zone database in Linux systems. It's often used in associated tools for setting or displaying time zone information.

Common Options

Option Description
dpkg-reconfigure tzdata Reconfigure the time zone on Debian-based systems.
timedatectl list-timezones List available time zones to use with tzdata .
timedatectl set-timezone [Zone/City] Set the system time zone.

Usage Examples

Reconfigure Time Zone (Debian) 

sudo dpkg-reconfigure tzdata

List Available Time Zones 

timedatectl list-timezones

Set Time Zone to New York 

sudo timedatectl set-timezone America/New_York

Cheat Sheet 


# Reconfigure timezone on Debian
sudo dpkg-reconfigure tzdata

# List all time zones
timedatectl list-timezones

# Set timezone (example: New York)
sudo timedatectl set-timezone America/New_York
