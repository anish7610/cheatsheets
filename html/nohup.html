nohup Command

Description

The nohup command allows a process to continue running in the background even after the user logs out. By default, it redirects the output to a file named nohup.out .

Common Options

Option Description
nohup <command> & Run a command in the background.
nohup <command> > output.log & Redirect output to a specific file.
nohup <command> >/dev/null 2>&1 & Ignore all output, including errors.

Usage Examples 

# Run a script in the background
nohup ./my_script.sh > script.log &

# Start a Python server without hanging up
nohup python3 -m http.server 8000 &

# Run a backup script and discard output
nohup ./backup.sh >/dev/null 2>&1 &

Cheat Sheet
