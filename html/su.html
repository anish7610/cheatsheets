Linux "su" Command

Description

The su command is used to switch the current user identity to another user or to become root user (superuser) in Linux systems.

Common Options

Option Description
- or -l Start a login shell with the environment of the target user.
-c Specify a command to execute as the target user.
-s Specify a shell to use.
-m or --preserve-environment Preserve the current environment when switching users.

Usage Examples 

# Switch to root user
su

# Switch to another user (e.g., alice)
su alice

# Execute a command as root
su -c 'ls /root'

# Switch to root and start a login shell
su -
    
# Use a specific shell (e.g., bash)
su -s /bin/bash alice

Cheat Sheet 

su       # Switch to root
su USER  # Switch to USER
su -     # Login shell as root
su -c CMD # Run CMD as root
su -s SHELL # Use SHELL
