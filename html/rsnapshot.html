Linux Command: rsnapshot

Description

rsnapshot is a command-line utility for taking filesystem snapshots, effectively creating backups using rsync and hard links, which makes efficient use of disk space.

Common Options

Option Description
-v Verbose output, useful for debugging.
-q Quiet mode, only output errors.
config_file Specify a custom configuration file.
sync_first Synchronize files before creating a snapshot.
-c Check the configuration file for syntax errors.

Usage Examples

Creating a Daily Backup 

rsnapshot daily

Using a Custom Configuration File 

rsnapshot -c /path/to/config.conf daily

Verbose Output for Debugging 

rsnapshot -v weekly

Cheat Sheet 


rsnapshot daily
