Linux `update-alternatives` Guide

Description

The update-alternatives command manages symbolic links determining default commands for system alternatives.

Common Options

Option Description
--install Install a new alternative for a command.
--config Select a specific alternative for a command.
--remove Remove an alternative link.
--display Show information about the alternatives.

Usage Examples

Install New Alternative 

sudo update-alternatives --install /usr/bin/editor editor /usr/bin/nano 10

Select Alternative 

sudo update-alternatives --config editor

Remove Alternative 

sudo update-alternatives --remove editor /usr/bin/nano

Display Alternatives 

update-alternatives --display editor

Cheat Sheet
