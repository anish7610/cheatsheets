Rsync Command

Description

The rsync command is a powerful utility for efficiently transferring and synchronizing files across computer systems by checking the differences in files.

Common Options

Option Description
-a Archive mode, preserves permissions, times, symbolic links, etc.
-v Verbose output, display files being processed.
-z Compress file data during transfer.
--delete Delete files in the destination that are not present in the source.
-P Show progress during transfer and keep partially transferred files.
-e Specify the remote shell to use, e.g., ssh .
--exclude Exclude files matching specified patterns.

Examples

Local Directory Synchronization 

rsync -avz /source/directory/ /destination/directory/

Remote Backup Over SSH 

rsync -avz -e ssh /local/directory/ user@host:/remote/directory/

Synchronize and Delete Extraneous Files 

rsync -av --delete /source/directory/ /destination/directory/

Exclude Specific Files 

rsync -av --exclude '*.log' /source/directory/ /destination/directory/

Cheat Sheet 


rsync -avz source/ destination/
rsync -avz -e ssh source/ user@host:/path/
rsync -av --delete source/ destination/
rsync -av --exclude 'pattern' source/ destination/
