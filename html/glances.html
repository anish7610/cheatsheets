Glances Command in Linux

Glances is a cross-platform system monitoring tool that provides a comprehensive view of your system resources, including CPU, memory, disk I/O, network, and more. It's designed to present data in a dynamic and real-time manner.

Common Options

Option Description
-h Display help message and exit.
-V Show version information and exit.
-c <file> Use a specific configuration file.
-t <seconds> Set the refresh time in seconds.
-B Disable network bandwidth.
-q Start Glances in quiet mode.

Usage Examples 


# Run Glances with default settings
glances

# Run Glances with a refresh time of 5 seconds
glances -t 5

# Run Glances using a specific configuration file
glances -c /path/to/config

# Run Glances in quiet mode
glances -q
