Cloud-Init Command

cloud-init is a command-line utility used to handle early initialization of cloud instances. It configures system settings, networking, and user data on first boot.

Common Options

Option Description
status Displays the status of cloud-init configuration.
init Initializes cloud-init stages: local, config, final.
modules Execute specific cloud-init modules.
clean Cleans cloud-init, removing instance data and logs.
single Runs a single cloud-init module.
collect-logs Collects cloud-init logs and system information.

Usage Examples 

# Check cloud-init status
cloud-init status

# Re-run cloud-init initialization
cloud-init init

# Run specific modules for network setup
cloud-init modules --mode=config

# Clean instance-specific data and logs
cloud-init clean

# Collect logs for troubleshooting
cloud-init collect-logs
