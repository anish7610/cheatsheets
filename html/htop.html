Understanding the htop Command

Description

htop is an interactive process viewer for Unix systems. It provides a dynamic, real-time view of system processes, usage, and performance metrics.

Common Options

Option Description
-d <delay> Set the delay between updates, in tenths of seconds.
-C Start htop in monochrome mode.
-u <username> Show only processes of a given user.
-s <column> Sort processes by the specified column.
-p <PIDs> Show only processes with specified PIDs.

Usage Examples 

# Start htop with common settings
htop

# Update every 2 seconds
htop -d 20

# Show processes of user 'john'
htop -u john

# Start in monochrome mode
htop -C

# Sort by CPU usage
htop -s PERCENT_CPU

Cheat Sheet 

/          : Search process
F4         : Filter processes
F6         : Sort by column
Up/Down    : Navigate processes
F9         : Kill process
q          : Quit htop
