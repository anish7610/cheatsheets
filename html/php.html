PHP Command Overview

The php command is used to execute PHP scripts from the command line.

Common Options

Option Description
-r Run PHP code without using script tags <?php ?>.
-f Parse and execute a file.
-l Check the syntax of a PHP file (lint).
-a Run the interactive mode shell.
-i Display PHP information (equivalent to phpinfo()).
-m Show compiled in PHP modules.

Usage Examples 

# Run a PHP script file
php -f script.php

# Execute PHP code directly
php -r 'echo "Hello, World!\n";'

# Syntax check for a PHP file
php -l script.php

# Interactive shell
php -a

# Display PHP configuration info
php -i

# List all loaded PHP modules
php -m

PHP Command Cheat Sheet 

php -f [file]    # Run script with file
php -r [code]    # Run code directly
php -l [file]    # Lint check for file
php -a           # Interactive mode
php -i           # PHP info
php -m           # List modules
