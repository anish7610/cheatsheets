Virt-Manager Command Overview

Description

virt-manager is a desktop user interface for managing virtual machines through libvirt . It supports the creation, configuration, and monitoring of virtual machines.

Common Options

Option Description
-h , --help Show help message and exit.
-c <URI> , --connect <URI> Specify the hypervisor connection URI.
--show-domain-console <name> Open the console for the specified domain.
--no-fork Run the application in the foreground.

Usage Examples 

# Open the virt-manager interface
virt-manager

# Connect to a remote hypervisor and open console for a specified domain
virt-manager -c qemu+ssh://user@remote/system --show-domain-console myvm

# Run virt-manager without forking to the background
virt-manager --no-fork

Cheat Sheet 

virt-manager               # Launch interface
virt-manager -c URI        # Connect to URI
virt-manager --show-domain-console NAME  # Open console for VM
virt-manager --no-fork     # Run in foreground
