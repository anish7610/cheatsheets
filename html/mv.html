Linux Command: mv

Description

The mv command in Linux is used to move or rename files and directories.

Common Options

Option Description
-i Prompts before overwriting files.
-f Forces overwriting without prompting.
-n Prevents overwriting of existing files.
-v Displays the steps being executed.

Usage Examples 


# Move a file to another directory
mv file.txt /path/to/directory/

# Rename a file
mv oldname.txt newname.txt

# Move multiple files to another directory
mv file1.txt file2.txt /path/to/directory/

# Use interactive mode to prevent accidental overwrites
mv -i file.txt /path/to/directory/

# Use verbose mode to see the operations being performed
mv -v file.txt /path/to/directory/

Cheat Sheet 


mv [options] source destination
mv -i    Prompt before overwrite
mv -f    Force overwrite without prompt
mv -n    No overwrite
mv -v    Verbose output
