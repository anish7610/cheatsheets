SVN Command Guide

Description

svn is a command line tool for interacting with Subversion, a version control system. It allows users to track changes, manage project versions, and collaborate effectively.

Common Options

Option Description
svn checkout [URL] Download a working copy from the repository.
svn update Bring changes from the repository into the working copy.
svn commit -m "message" Send changes from the working copy to the repository.
svn status Show the status of working copy files and directories.
svn log Display log messages from the repository.
svn add [file] Schedule a file or directory for addition to the repository.
svn delete [file] Schedule a file or directory for removal from the repository.
svn revert Undo changes in the working copy.

Examples 

# Check out a repository
svn checkout http://example.com/svn/myproject

# Update the working copy
svn update

# Commit changes with a message
svn commit -m "Fixed bugs and added features"

# Add a new file for version control
svn add newfile.txt

# View the status of the working copy
svn status

# Revert changes in the working copy
svn revert changedfile.txt

Cheat Sheet 

svn co [URL]        # Checkout
svn up            # Update
svn ci -m "msg"   # Commit
svn st            # Status
svn add [file]    # Add a file
svn del [file]    # Delete a file
svn revert        # Revert changes
