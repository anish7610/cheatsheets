Linux Command: cat

Description

The cat command is used to concatenate and display the content of files in the terminal.

Common Options

Option Description
-n Number all output lines.
-b Number non-blank output lines.
-s Squeeze multiple blank lines into one.
-E Display a $ at the end of each line.

Usage Examples 

# Display the contents of a file
cat filename.txt

# Concatenate multiple files into a single file
cat file1.txt file2.txt > combined.txt

# Number all lines in a file
cat -n filename.txt

# Number non-blank lines in a file
cat -b filename.txt

# Suppress repeated empty output lines
cat -s filename.txt

# Show ends of lines
cat -E filename.txt

Cheat Sheet 

cat [options] [files...]
-n: number all lines
-b: number non-blank lines
-s: squeeze blank lines
-E: show $ at end of lines
