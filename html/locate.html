Linux locate Command

Description

The locate command is used to find files by name quickly. It searches a pre-built database to perform fast file lookups.

Common Options

Option Description
-i Ignore case distinctions in patterns and data.
-c Display only the count of matching entries.
-r Interpret pattern as a regular expression.
-n NUM Limit the number of results to NUM.

Usage Examples 

# Find all files with "document" in the name
locate document

# Find all files with ".conf" extension, ignoring case
locate -i .conf

# Count files with "log" in the name
locate -c log

# Use a regular expression to find files starting with "test"
locate -r '^test'

# Limit results to 5 entries for "backup"
locate -n 5 backup

Cheat Sheet 

locate [OPTION]... PATTERN...
-i       # Ignore case
-c       # Count matches
-r       # Regex pattern
-n NUM   # Limit results
