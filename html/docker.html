Docker Command Overview

The docker command is used to manage containers in Linux, allowing you to create, deploy, and run applications in containers.

Common Options

Option Description
docker run Run a command in a new container.
docker ps List running containers.
docker pull Download an image from a registry.
docker build Build an image from a Dockerfile.
docker exec Run a command in a running container.
docker stop Stop a running container.

Usage Examples 

# Run a container from an image
docker run -d -p 80:80 nginx

# List all running containers
docker ps

# Pull the latest Ubuntu image
docker pull ubuntu:latest

# Build an image from a Dockerfile in the current directory
docker build -t myimage .

# Execute a command inside a running container
docker exec -it mycontainer /bin/bash

# Stop a running container
docker stop mycontainer

Cheat Sheet 

docker run IMAGE    # Run container
docker ps             # List containers
docker pull IMAGE     # Pull image
docker build PATH     # Build image
docker exec CONTAINER # Exec in container
docker stop CONTAINER # Stop container
