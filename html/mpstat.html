mpstat Command

The mpstat command is used to display CPU-related statistics. It provides information about each available processor or core, helping monitor CPU performance.

Common Options

Option Description
-P ALL Display statistics for all CPUs.
-u Show CPU utilization.
-I SUM Display interrupt information summary.
-V Display version information.

Usage Examples 

mpstat
mpstat -P ALL
mpstat -u 5 3
mpstat -I SUM 1 5
