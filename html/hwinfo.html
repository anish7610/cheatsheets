Linux Command: hwinfo

Description

The hwinfo command is used to probe for hardware components and display detailed information about them. It provides a comprehensive overview of your systemâ€™s hardware.

Common Options

Option Description
--short Displays a concise summary of the hardware information.
--cpu Shows detailed information about the CPU.
--disk Lists information about disk drives.
--network Provides details about network interfaces.
--gfxcard Displays information about graphics cards.
--help Shows all available options of the command.

Usage Examples 

# Display a short summary of all hardware
hwinfo --short

# Get detailed information about the CPU
hwinfo --cpu

# List details about all disk drives
hwinfo --disk

# Show information on network interfaces
hwinfo --network

# Display graphics card information
hwinfo --gfxcard
