GCC: GNU Compiler Collection

The gcc command is used to compile C and C++ programs. It is part of the GNU Compiler Collection, supporting multiple languages. Commonly used to turn source code into executable files.

Common Options

Option Description
-o <file> Specify the output file name for the executable.
-c Compile the source file without linking.
-Wall Enable most compiler warnings.
-O2 Optimize the code for speed.
-g Include debugging information in the compiled file.
-I<dir> Add directory to the list of include search paths.
-L<dir> Add directory to the list of library search paths.
-l<library> Link with the specified library.

Usage Examples 

# Compile a single C file
gcc -Wall -o hello hello.c

# Compile with debugging information
gcc -g -o debug_program program.c

# Compile without linking
gcc -c file.c

# Compile with optimization
gcc -O2 -o fast_program program.c

# Include additional header file directories
gcc -I/usr/local/include -o program program.c

# Link with math library
gcc -o calc calc.c -lm

Cheat Sheet 

gcc -o <output> <source.c>
gcc -Wall -o <output> <source.c>
gcc -g -o <output> <source.c>
gcc -O2 -o <output> <source.c>
gcc -I<dir> -L<dir> -l<lib> -o <output> <source.c>
