Get-WebApplication Command Guide

Description

The Get-WebApplication command retrieves information about web applications in IIS (Internet Information Services).

Common Options

Option Description
-Name Specifies the name of the web application to retrieve.
-Site Specifies the site under which the web application resides.
-Filter Applies a filter to the list of web applications.

Usage Examples 

# Retrieve all web applications on the default site
Get-WebApplication -Site "Default Web Site"

# Retrieve a specific web application by name
Get-WebApplication -Name "MyApp"

# Apply a filter to retrieve applications that contain 'shop' in their name
Get-WebApplication -Filter "*shop*"

Cheat Sheet 

# List all on default site
Get-WebApplication -Site "Default Web Site"

# Get specific app
Get-WebApplication -Name "AppName"

# Filter applications
Get-WebApplication -Filter "*pattern*"
