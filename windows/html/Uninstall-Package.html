Uninstall-Package Command Overview

Description

The Uninstall-Package command is used in Windows PowerShell to remove a software package from the system that was installed by a package provider.

Common Options

Option Description
-Name Specifies the name of the package to uninstall.
-ProviderName Specifies the name of the package provider.
-Force Forces the command to run without asking for user confirmation.
-AllVersions Uninstalls all versions of the package.
-Source Specifies the source of the package.

Usage Examples 

# Uninstall a specific package by name
Uninstall-Package -Name "ExamplePackage"

# Force uninstall a package without confirmation
Uninstall-Package -Name "ExamplePackage" -Force

# Uninstall a package using a specific provider
Uninstall-Package -Name "ExamplePackage" -ProviderName "ProviderExample"

# Uninstall all versions of a package
Uninstall-Package -Name "ExamplePackage" -AllVersions

# Uninstall a package specifying the source
Uninstall-Package -Name "ExamplePackage" -Source "SourceExample"

Cheat Sheet 

# Basic uninstall
Uninstall-Package -Name "PackageName"

# Force uninstall
Uninstall-Package -Name "PackageName" -Force

# Specific provider
Uninstall-Package -Name "PackageName" -ProviderName "ProviderName"

# All versions
Uninstall-Package -Name "PackageName" -AllVersions
