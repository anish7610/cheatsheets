Get-PhysicalDisk Command

Description

The Get-PhysicalDisk cmdlet retrieves information about the physical disks on a Windows system.

Common Options

Option Description
No specific options Returns all physical disks with default properties.
-CimSession <CimSession> Runs the command in a remote session or on a remote computer.
-SerialNumber Filters the output based on a specified serial number.
-UniqueId Displays information for a disk with a specific unique ID.

Usage Examples 

# List all physical disks
Get-PhysicalDisk

# Filter by serial number
Get-PhysicalDisk -SerialNumber "123456789"

# Run the command on a remote computer
Get-PhysicalDisk -CimSession "RemoteServerName"
