Clear-EventLog Command

Description

The Clear-EventLog command in Windows PowerShell is used to clear all entries from specified event logs on the local or remote computers.

Common Options

Option Description
-LogName Specifies the names of the event logs to clear.
-ComputerName Specifies the target computer(s) from which to clear the event logs.

Usage Examples 

# Clear the Application event log on the local computer
Clear-EventLog -LogName Application

# Clear the System and Security event logs on the local computer
Clear-EventLog -LogName System, Security

# Clear the Application event log on a remote computer
Clear-EventLog -LogName Application -ComputerName Server01

Cheat Sheet 

Clear-EventLog -LogName [LogName]
Clear-EventLog -LogName [LogName1, LogName2] -ComputerName [ServerName]
