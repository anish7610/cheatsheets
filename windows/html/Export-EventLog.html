Export-EventLog Command

Export-EventLog is a PowerShell command used to export event logs from Windows systems to a file, making it easier to archive and analyze log files.

Common Options

Option Description
-LogName Specifies the name of the event log to be exported.
-FilePath Defines the full path where the exported log file will be saved.
-Newest Exports the newest entries up to the specified number.
-ComputerName Specifies the target computer from which the event log is exported.
-UserName Uses a specific user credential for connecting to a remote computer.

Usage Examples 

# Export the Application log to a file
Export-EventLog -LogName Application -FilePath "C:\Logs\AppLog.evt"
# Export the last 50 entries from the System log
Export-EventLog -LogName System -FilePath "C:\Logs\SysLog.evt" -Newest 50
# Export the Security log from a remote computer
Export-EventLog -LogName Security -FilePath "C:\Logs\SecLog.evt" -ComputerName "Server01"

Cheat Sheet 

# Basic format
Export-EventLog -LogName [Log] -FilePath [Path] -Newest [Number]
