Stop-Service Command

Description

The Stop-Service command in Windows PowerShell is used to stop a running service on a local or remote machine.

Common Options

Option Explanation
-Name Specifies the name of the service to stop.
-DisplayName Specifies the display name of the service to stop.
-InputObject Accepts a service object, typically obtained from Get-Service .
-Force Forces the service to stop even if it has dependent services.

Usage Examples 

# Stop a service by name
Stop-Service -Name "Spooler"

# Stop a service by display name
Stop-Service -DisplayName "Print Spooler"

# Stop multiple services using a pipeline
Get-Service -Name "Spooler", "wuauserv" | Stop-Service

# Force stop a service
Stop-Service -Name "Spooler" -Force

Cheat Sheet 

Stop-Service -Name ""
Stop-Service -DisplayName ""
Get-Service | Stop-Service
Stop-Service -Name "" -Force
