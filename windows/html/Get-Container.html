Get-Container Command

Description

The Get-Container command is used in Windows PowerShell to retrieve information about containers running on a system.

Common Options

Option Description
-Name Specifies the name of the container to get.
-Id Specifies the ID of the container to get.
-Status Filters containers by their status (e.g., Running, Stopped).

Usage Examples 

# Retrieve all containers
Get-Container

# Retrieve a container by name
Get-Container -Name "myContainer"

# Retrieve a container by ID
Get-Container -Id "container123"

# Retrieve only running containers
Get-Container | Where-Object {$_.Status -eq "Running"}

Cheat Sheet 

Get-Container               # List all containers
Get-Container -Name     # Get container by name
