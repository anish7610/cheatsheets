Windows SDK Overview

Description

The Windows SDK includes tools, libraries, headers, and drivers needed to develop applications for Windows. It is used to create, compile, and debug Windows applications.

Common Options

Option Description
/install Install the Windows SDK components.
/uninstall Remove the Windows SDK components.
/quiet Run the installation in silent mode.
/help Display help information for the SDK commands.

Examples 

sdksetup.exe /install /quiet

Silently install the Windows SDK without user interaction. 

sdksetup.exe /uninstall

Uninstall the Windows SDK components. 

sdksetup.exe /help

Display help information for available options.

Cheat Sheet 

/install - Install SDK
/uninstall - Remove SDK
/quiet - Silent mode
/help - List options
