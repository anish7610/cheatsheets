Batch Files (.bat) in Windows

Batch files are scripts that contain a sequence of commands to be executed by the Windows command interpreter. They are commonly used for automating routine tasks and configuring system settings.

Common Options and Explanations

Option Description
echo Displays messages or turns command echoing on or off.
pause Suspends the batch process and prompts the user to press any key to continue.
rem Adds a comment line in the batch file.
if Performs conditional processing in batch programs.
goto Directs batch file processing to a labeled line.
set Defines and manages environment variables.
call Calls another batch file and returns to the original script afterward.
exit Exits the command processor or batch file.

Usage Examples 

@echo off
echo Hello, World!
pause

@echo off
set /p name=Enter your name: 
echo Hello, %name%!
pause

@echo off
if exist "file.txt" (
    echo File exists.
) else (
    echo File does not exist.
)
pause

Cheat Sheet 

@echo off        :: Disable command echoing
echo text         :: Display text
pause             :: Wait for user input
rem comment       :: Add comment
if condition      :: Conditional execution
goto label        :: Jump to label
set var=value     :: Set variable
call script.bat   :: Call another batch file
exit              :: Exit script
