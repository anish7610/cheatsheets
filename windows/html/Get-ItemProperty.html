Get-ItemProperty Command

The Get-ItemProperty command retrieves properties of a specified item in Windows, often used to access registry keys and their values.

Common Options

Option Description
-Path Specifies the path to the item or items.
-Name Specifies the property name to retrieve.
-Filter Applies a filter to the property names.
-ExcludeProperty Excludes certain properties from the output.
-IncludeProperty Includes only specified properties in the output.

Usage Examples 

Get-ItemProperty -Path "HKLM:\Software\Microsoft\Windows\CurrentVersion" -Name "ProgramFilesDir"
Get-ItemProperty -Path "HKLM:\Software\Microsoft\Windows\CurrentVersion\Uninstall\*" -Filter "DisplayName" | Select-Object DisplayName, Publisher

Cheat Sheet 


Get-ItemProperty -Path "Path" -Name "PropertyName"
Get-ItemProperty -Path "Path" -Filter "FilterString"
