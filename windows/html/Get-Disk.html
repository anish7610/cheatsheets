Get-Disk Command Overview

Description

The Get-Disk command retrieves information about physical disk drives in a Windows system.

Common Options

Option Description
-Number Specifies the number of a particular disk to retrieve.
-CimSession Runs the command on a remote computer.
-PartitionStyle Filters disks by partition style (MBR or GPT).
-BusType Filters disks by bus type (e.g., USB, SATA).

Usage Examples 

# List all physical disks
Get-Disk

# Get information about disk number 0
Get-Disk -Number 0

# List all GPT partition style disks
Get-Disk -PartitionStyle GPT

# List all USB connected disks
Get-Disk -BusType USB

Cheat Sheet 

Get-Disk                  # List all disks
Get-Disk -Number 0       # Get details of disk 0
Get-Disk -PartitionStyle # Filter by partition (MBR/GPT)
Get-Disk -BusType        # Filter by bus type (USB, SATA)
