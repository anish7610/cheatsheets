Windows Set-Service Command

Description

The Set-Service command in PowerShell is used to modify the properties of a service on a local or remote computer.

Common Options

Option Description
-Name Specifies the name of the service to modify.
-Status Sets the status of the service. Values: Running, Stopped.
-StartupType Sets the startup type of the service. Values: Automatic, Manual, Disabled.
-PassThru Returns an object representing the service.
-ComputerName Specifies a remote computer to modify the service on.

Usage Examples 

# Start a service
Set-Service -Name "wuauserv" -Status Running

# Stop a service
Set-Service -Name "wuauserv" -Status Stopped

# Set a service to start automatically
Set-Service -Name "wuauserv" -StartupType Automatic

# Set a service to manual start
Set-Service -Name "wuauserv" -StartupType Manual

# Set a service to Disabled
Set-Service -Name "wuauserv" -StartupType Disabled

Cheat Sheet 


Start: Set-Service -Name <ServiceName> -Status Running
Stop: Set-Service -Name <ServiceName> -Status Stopped
Auto Start: Set-Service -Name <ServiceName> -StartupType Automatic
Manual Start: Set-Service -Name <ServiceName> -StartupType Manual
Disable: Set-Service -Name <ServiceName> -StartupType Disabled
