Get-Command

Description

The Get-Command cmdlet is used in PowerShell to retrieve information about cmdlets, functions, workflows, aliases, and applications. It's a powerful tool for discovering system capabilities and available commands.

Common Options

Option Description
-Name Specifies the name of the command to get.
-Noun Filters commands based on the noun part of their names.
-Verb Filters commands based on the verb part of their names.
-Module Filters commands from a specific module.
-CommandType Limits the results to a specified command type (e.g., cmdlet, function).

Usage Examples 


# Get information about a specific command
Get-Command -Name Get-Process

# List all cmdlets with the verb 'Get'
Get-Command -Verb Get

# List all commands from a specific module
Get-Command -Module Microsoft.PowerShell.Management

# Find all functions with a specific noun
Get-Command -Noun Item

Cheat Sheet 


# Find by name
Get-Command -Name 

# Filter by verb
Get-Command -Verb 

# Filter by noun
Get-Command -Noun 

# Filter by module
Get-Command -Module 

# Specify command type
Get-Command -CommandType
