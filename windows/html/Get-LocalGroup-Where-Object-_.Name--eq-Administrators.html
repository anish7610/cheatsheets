Get-LocalGroup Command Guide

Description

The `Get-LocalGroup` command is used in Windows PowerShell to retrieve local group information. Combined with `Where-Object`, it filters groups based on specific criteria, such as finding the "Administrators" group.

Common Options

Option Description
-Name Specifies the name of the group to retrieve.
-SID Returns the security identifier of the group.
-Description Provides the description of the group.

Usage Examples 


# Retrieve information about the "Administrators" group
Get-LocalGroup | Where-Object { $_.Name -eq "Administrators" }

# List all local groups
Get-LocalGroup

# Find a group by name
Get-LocalGroup -Name "Users"

Cheat Sheet 


# List all groups
Get-LocalGroup

# Find the Administrators group
