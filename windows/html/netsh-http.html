netsh http Command Overview

Description

The netsh http command allows for managing HTTP configurations on Windows, providing control over settings such as URL reservations, SSL certificates, and more.

Common Options

Option Description
add urlacl Reserves a URL namespace for a particular user or group.
delete urlacl Deletes a previously reserved URL namespace.
show urlacl Displays all reserved URL namespaces.
add sslcert Binds an SSL certificate to an IP address and port.
delete sslcert Removes an SSL certificate binding from an IP address and port.
show sslcert Displays all SSL certificate bindings.

Usage Examples 

# Reserve a URL for a specific user
netsh http add urlacl url=http://+:8080/ user=DOMAIN\username

# Delete a reserved URL
netsh http delete urlacl url=http://+:8080/

# Display all URL reservations
netsh http show urlacl

# Add an SSL certificate binding
netsh http add sslcert ipport=0.0.0.0:443 certhash=YOURCERTTHUMBPRINT appid={YOURAPPID}

# Remove an SSL certificate binding
netsh http delete sslcert ipport=0.0.0.0:443

# Display all SSL certificate bindings
netsh http show sslcert
