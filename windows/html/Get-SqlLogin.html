Get-SqlLogin Command

The Get-SqlLogin command retrieves login information from a SQL Server instance.

Common Options

Option Description
-ServerInstance Specifies the SQL Server instance.
-Credential Provides authentication credentials.
-Username Filters results by specific username.
-Databases Retrieves logins for specified databases.

Usage Examples 


Get-SqlLogin -ServerInstance "ServerName" -Credential (Get-Credential)
Get-SqlLogin -ServerInstance "ServerName" -Username "sa"
Get-SqlLogin -ServerInstance "ServerName" -Databases "Database1"

Cheat Sheet 


Get-SqlLogin -ServerInstance [Instance] -Credential [Get-Credential]
Get-SqlLogin -ServerInstance [Instance] -Username [User]
Get-SqlLogin -ServerInstance [Instance] -Databases [DB]
