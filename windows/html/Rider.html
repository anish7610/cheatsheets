Rider Command Overview

Description

The `rider` command is used to launch and control JetBrains Rider, a cross-platform .NET IDE, from the command line.

Common Options

Option Description
--line NUMBER Open a file at a specific line number.
--project PATH Specify the project or solution file to open.
--help Show help information and exit.
--version Show Rider version and exit.

Usage Examples 

# Open a specific file at line 10
rider --line 10 /path/to/file.cs

# Open a specific project or solution
rider --project /path/to/solution.sln

# Display Rider version
rider --version

Cheat Sheet 


rider --line [NUM] [FILE]     # Open file at line NUM
rider --project [PATH]        # Open project/solution
rider --version               # Show version
rider --help                  # Show help
