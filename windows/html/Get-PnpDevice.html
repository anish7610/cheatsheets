Get-PnpDevice Command

Description

The Get-PnpDevice command in Windows PowerShell retrieves information about Plug and Play (PnP) devices on a system.

Common Options

Option Description
-Status Filters devices by status (e.g., OK, Error).
-FriendlyName Specifies devices by their friendly name.
-Class Filters devices by their class (e.g., Display, USB).

Usage Examples 

Get-PnpDevice
Get-PnpDevice -Status OK
Get-PnpDevice -Class USB
Get-PnpDevice -FriendlyName "Intel(R) Wireless"

Cheat Sheet 

Get-PnpDevice                  # Lists all PnP devices
Get-PnpDevice -Status OK       # Lists devices with status OK
Get-PnpDevice -Class USB       # Lists USB devices
Get-PnpDevice -FriendlyName "Name" # Lists devices by friendly name
