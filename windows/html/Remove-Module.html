Remove-Module Command

The Remove-Module command in Windows PowerShell is used to remove modules from the current session, freeing up resources and unloading functions and cmdlets provided by the module.

Common Options

Option Description
-Name Specifies the name of the module to remove.
-Force Forcibly removes the module even if it is used by other modules.
-ErrorAction Determines how the command responds to an error.

Usage Examples 

# Example 1: Remove a module by name
Remove-Module -Name 'ModuleName'

# Example 2: Forcefully remove a module
Remove-Module -Name 'ModuleName' -Force

# Example 3: Remove a module and handle errors silently
Remove-Module -Name 'ModuleName' -ErrorAction SilentlyContinue

Cheat Sheet 

Remove-Module -Name 'ModuleName'
Remove-Module -Name 'ModuleName' -Force
Remove-Module -Name 'ModuleName' -ErrorAction SilentlyContinue
