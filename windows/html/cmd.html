cmd Command in Windows

Description

The cmd command is the command-line interpreter for Windows operating systems, providing a command shell and scripting abilities.

Common Options

Option Description
/c Carries out the command specified by the string and then terminates.
/k Carries out the command specified by the string but remains open.
/q Turns echo off (command output isn't displayed).
/d Disables execution of AutoRun commands.
/s Modifies the treatment of string after /c or /k.
/v:on|off Enables or disables delayed environment variable expansion.

Usage Examples 



cmd /c echo Hello, World!


cmd /k echo This window will stay open


cmd /c C:\path\to\script.bat


cmd /d /c dir


cmd /v:on /k set myvar=Hello && echo !myvar!

Cheat Sheet 


cmd /c      # Run and exit
cmd /k      # Run and stay
cmd /q               # Echo off
cmd /d               # Disable AutoRun
cmd /v:on|off        # Delayed expansion
