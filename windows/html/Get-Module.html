Get-Module Command Overview

The Get-Module command is used in Windows PowerShell to retrieve information about the modules that have been imported or are available to be imported.

Common Options

Option Description
-ListAvailable Shows all modules that are available to import.
-Name Filters modules based on their name.
-FullyQualifiedName Gets modules using a fully qualified name.
-All Gets all modules, not limited to the first one found.

Usage Examples 

# List all imported modules
Get-Module

# List all available modules on the system
Get-Module -ListAvailable

# Get information about a specific module
Get-Module -Name ModuleName

# Display all available versions of a specific module
Get-Module -Name ModuleName -ListAvailable -All

Cheat Sheet 

# Show all imported modules
Get-Module

Get-Module -ListAvailable
