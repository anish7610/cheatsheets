Introduction to Scoop

Scoop is a command-line installer for Windows that simplifies the process of installing, updating, and managing software packages.

Common Options

Option Description
install Install a specific application.
uninstall Remove an installed application.
update Update installed applications.
list List installed applications.
search Search for applications.
info Get detailed information about a package.

Usage Examples 


# Install an application
scoop install git

# Uninstall an application
scoop uninstall git

# Update all applications
scoop update *

# List installed applications
scoop list

# Search for an application by name
scoop search python

# Get information about a specific application
scoop info nodejs

Cheat Sheet 


scoop install [app]      # Install an app
scoop uninstall [app]    # Uninstall an app
scoop update *           # Update all apps
scoop list               # List installed apps
scoop search [keyword]   # Search for an app
scoop info [app]         # Get app info
