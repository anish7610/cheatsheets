WMIC Command in Windows

WMIC (Windows Management Instrumentation Command-line) is a command-line utility for interacting with Windows Management Instrumentation (WMI). It allows users to perform administrative tasks and query system information.

Common Options

Option Description
/node: Specify the target computer.
/user: Specify the user to use for the command.
/password: Specify the user's password.
process Query, terminate or display processes.
os Retrieve operating system information.
product List installed products and software.

Usage Examples 

wmic process list brief
wmic os get caption, version
wmic product get name, version
wmic process where "name='notepad.exe'" call terminate
