IntelliJ IDEA Command

Description

The idea command is used to open IntelliJ IDEA from the command line, allowing you to manage and develop projects efficiently.

Common Options

Option Description
-h , --help Show help message and exit.
--line <line_number> Open a file and move the caret to the specified line.
<file> Specify a file to open in IntelliJ IDEA.
<project_directory> Open a specific project directory.

Usage Examples 


# Open IntelliJ IDEA
idea

# Open a specific file in the editor
idea path/to/file.java

# Open a project directory
idea path/to/project/

# Open a file at a specific line number
idea --line 42 path/to/file.java
