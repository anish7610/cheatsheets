Windows Backup Command

Description

The Windows Backup command is used to create backups of files, folders, or entire systems, allowing users to restore data in case of loss or corruption.

Common Options

Option Description
/backup Initiates a backup operation.
/restore Restores data from a backup.
/quiet Runs the command without any interactive prompts.
/target: destination Specifies the backup location.
/include: items Specifies files or folders to include in the backup.
/exclude: items Specifies files or folders to exclude from the backup.

Usage Examples 



wbadmin start backup -backupTarget:D: -include:C:\Users\John -quiet


wbadmin start recovery -version:09/30/2023-10:00 -backupTarget:D: -quiet

Cheat Sheet 


# Start backup
wbadmin start backup -backupTarget: -include: -quiet

# Restore backup
wbadmin start recovery -version: -backupTarget: -quiet
