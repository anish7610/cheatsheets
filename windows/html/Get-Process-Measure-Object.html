Get-Process | Measure-Object

Description

The command Get-Process | Measure-Object retrieves a list of processes and computes statistics, such as count, using Measure-Object .

Common Options

Option Description
-Property Specifies the property to measure, such as CPU time.
-Sum Calculates the sum of the specified property values.
-Average Calculates the average of the specified property values.
-Maximum Finds the maximum value of the specified property.
-Minimum Finds the minimum value of the specified property.

Usage Examples 

# Count the total number of processes
Get-Process | Measure-Object

# Calculate the total CPU time used by all processes
Get-Process | Measure-Object -Property CPU -Sum

# Find the average CPU time of processes
Get-Process | Measure-Object -Property CPU -Average

# Get the highest memory usage of running processes
Get-Process | Measure-Object -Property WS -Maximum

Cheat Sheet 

# Count processes
Get-Process | Measure-Object

# Sum of CPU times
Get-Process | Measure-Object -Property CPU -Sum

# Average CPU usage
Get-Process | Measure-Object -Property CPU -Average

# Maximum memory usage
Get-Process | Measure-Object -Property WS -Maximum
