Get-SqlBackupHistory: Overview

The Get-SqlBackupHistory command retrieves the backup history of databases from a SQL Server instance.

Common Options

Option Description
-ServerInstance The name or network address of the SQL Server instance.
-Database The name of the database to get backup history for.
-Since Gets backups since the specified date.
-MaximumRows Limits the number of rows returned.

Usage Examples 

# Retrieve backup history for a specific database
Get-SqlBackupHistory -ServerInstance "ServerName" -Database "DatabaseName"

# Get all backup history since a specific date
Get-SqlBackupHistory -ServerInstance "ServerName" -Since "2023-01-01"

# Limit the results to the last 5 backups
Get-SqlBackupHistory -ServerInstance "ServerName" -MaximumRows 5
