Windows Command: rmdir

The rmdir command is used to delete directories from the file system. It can only remove empty directories unless specific options are used.

Common Options

Option Description
/S Removes all directories and files in the specified directory, including the specified directory itself.
/Q Quiet mode, does not prompt for confirmation when using /S .

Usage Examples 

rmdir /S /Q C:\example\directory
rmdir C:\empty\directory

Cheat Sheet 

rmdir /S /Q [directory]   # Remove directory and contents without confirmation
rmdir [directory]        # Remove empty directory
