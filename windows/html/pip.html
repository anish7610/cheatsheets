Windows Command: pip

pip is the package manager for Python. It allows you to install and manage packages from the Python Package Index and other repositories.

Common Options

Option Description
install <package> Installs the specified package.
uninstall <package> Uninstalls the specified package.
list Lists installed packages.
freeze Outputs installed packages in a format that can be used by pip install .
search <query> Searches PyPI for packages.
show <package> Displays information about the installed package.
help Shows help for commands and options.
upgrade <package> Upgrades the specified package to the latest version.

Usage Examples 

pip install requests
pip uninstall requests
pip list
pip freeze > requirements.txt
pip install -r requirements.txt
pip search django
pip show numpy
pip install --upgrade pip

Cheat Sheet 

pip install <package>
pip uninstall <package>
pip list
pip freeze
pip search <query>
pip show <package>
pip install --upgrade <package>
