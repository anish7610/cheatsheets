Get-WinEvent -LogName System

Description

The Get-WinEvent -LogName System command retrieves events from the System log on Windows. It's used for monitoring system events, troubleshooting, and auditing.

Common Options

Option Description
-MaxEvents Specifies the maximum number of events to return.
-FilterXPath Filters events using an XPath query.
-Oldest Retrieves the oldest events first.
-ComputerName Specifies the target computer; defaults to the local computer.
-Credential Uses specified credentials for remote access.

Usage Examples 


# Retrieve the last 50 system events
Get-WinEvent -LogName System -MaxEvents 50

# Get system events from the oldest to the newest
Get-WinEvent -LogName System -Oldest

# Filter events for a specific event ID (e.g., ID 7045)
Get-WinEvent -LogName System | Where-Object { $_.Id -eq 7045 }

# Retrieve system events from a remote machine
Get-WinEvent -LogName System -ComputerName "RemotePC"
