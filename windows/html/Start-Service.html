Start-Service Command

Description

The Start-Service command in Windows PowerShell is used to start a stopped service.

Common Options

Option Description
-Name Specifies the service name to start.
-DisplayName Starts the service with the specified display name.
-InputObject Specifies the service object to start.
-PassThru Returns an object representing the service.
-Verbose Provides detailed information about the command execution.

Usage Examples 

# Start a specific service by name
Start-Service -Name "wuauserv"

# Start a service using its display name
Start-Service -DisplayName "Windows Update"

# Start a service and return the service object
Start-Service -Name "wuauserv" -PassThru

Cheat Sheet 

Start-Service -Name <ServiceName>
Start-Service -DisplayName <DisplayName>
Start-Service -Name <ServiceName> -PassThru
