PowerShell Command: Set-ItemProperty

The Set-ItemProperty command in PowerShell is used to change the value of a property for a specified item, such as a registry key or a file system item.

Common Options

Option Description
-Path Specifies the path to the item where the property value is set.
-Name Specifies the name of the property to modify.
-Value Specifies the new value for the property.
-LiteralPath Specifies a path that is used exactly as entered.
-Force Overrides restrictions that might prevent command completion.

Examples

Change a Registry Key Value 

Set-ItemProperty -Path "HKLM:\Software\MyApp" -Name "Version" -Value "2.0"

Update File Attribute 

Set-ItemProperty -Path "C:\MyFolder\file.txt" -Name "IsReadOnly" -Value $false

Cheat Sheet 


Set-ItemProperty -Path "" -Name "" -Value ""
