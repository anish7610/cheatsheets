gcloud (Google Cloud SDK)

Description

The gcloud command is a part of the Google Cloud SDK used for managing Google Cloud resources and applications from the command line.

Common Options

Option Description
--project Specifies the Google Cloud project to use.
--quiet or -q Disables all interactive prompts.
--help Displays help information for a specific command.
--format Specifies the output format (e.g., json , yaml ).
--zone Specifies the zone of the resource.

Usage Examples 



gcloud auth login


gcloud config set project my-project-id


gcloud compute instances list --project=my-project-id


gcloud compute instances create instance-1 --zone=us-central1-a


gcloud compute instances delete instance-1 --zone=us-central1-a

Cheat Sheet 


# Authenticate
gcloud auth login

# Set project
gcloud config set project [PROJECT_ID]

# List instances
gcloud compute instances list

# Create instance
gcloud compute instances create [INSTANCE_NAME] --zone=[ZONE]

# Delete instance
gcloud compute instances delete [INSTANCE_NAME] --zone=[ZONE]
