Netstat Command Overview

Description

The netstat command displays network connections, routing tables, and interface statistics.

Common Options

Option Description
-a Display all connections and listening ports.
-n Show numerical addresses instead of resolving hosts.
-o Show the owning process ID associated with each connection.
-b Display the executable involved in creating each connection.
-s Show statistics by protocol.

Examples 

# Display all connections and listening ports
netstat -a

# Show all connections with numerical IPs and ports
netstat -an

# Find which process ID is using a specific port (e.g., port 80)
netstat -ao | findstr :80

# Show statistics for TCP protocol
netstat -s -p tcp

Cheat Sheet 

# All connections and ports
netstat -a

# Numerical output
netstat -n

# With process ID
netstat -o

# With executable details
netstat -b
