Get-Package Command Overview

The Get-Package command in Windows PowerShell retrieves a list of software packages that are installed on the system.

Common Options

Option Description
-Name Specify the name of the package to retrieve.
-ProviderName Filter results by the package provider, such as "Programs" or "msi".
-Source Specify the source of the packages, such as a repository name.
-AllVersions List all versions of the specified package.

Usage Examples 

Get-Package
Get-Package -Name "Notepad++"
Get-Package -ProviderName "Programs"
Get-Package -Name "Google Chrome" -AllVersions

Cheat Sheet 

Get-Package                      # List all packages
Get-Package -Name <name>        # Find a specific package
Get-Package -ProviderName <provider>  # Filter by provider
