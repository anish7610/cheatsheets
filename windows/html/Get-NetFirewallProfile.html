Get-NetFirewallProfile Command

Description

The Get-NetFirewallProfile command retrieves the current settings of the Windows Firewall profile, including domain, private, and public profiles.

Common Options

Option Description
-Profile Specifies which firewall profile(s) to retrieve (Domain, Private, Public).
-CimSession Runs the command on a remote session or computer.
-PolicyStore Specifies the policy store location from which to retrieve the profiles.

Usage Examples 

# Retrieve all firewall profiles
Get-NetFirewallProfile

# Retrieve only the private profile settings
Get-NetFirewallProfile -Profile Private

# Retrieve firewall profiles from a remote computer
Get-NetFirewallProfile -CimSession RemotePCName

# Retrieve specific profiles from a specified policy store
Get-NetFirewallProfile -Profile Domain -PolicyStore PersistentStore

Cheat Sheet 

# List all profiles
Get-NetFirewallProfile

# List a specific profile
Get-NetFirewallProfile -Profile [Domain|Private|Public]

# Remote session usage
Get-NetFirewallProfile -CimSession [ComputerName]
