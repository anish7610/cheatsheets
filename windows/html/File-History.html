File History Command Overview

Description

The File History command in Windows is a built-in tool that automatically backs up files stored in Libraries, Desktop, Contacts, and Favorites to an external drive or network location. It allows users to restore previous versions of files as needed.

Common Options

Option Description
/enable Turns on File History for the current drive.
/disable Turns off File History for the current drive.
/backupnow Performs an immediate backup of the files.
/restore Opens the File History restore interface to recover files.

Usage Examples 

FileHistory /enable
FileHistory /backupnow
FileHistory /restore

Cheat Sheet 

Enable:   FileHistory /enable
Disable:  FileHistory /disable
Backup:   FileHistory /backupnow
Restore:  FileHistory /restore
