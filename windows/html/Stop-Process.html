Stop-Process Command in Windows

The Stop-Process command is used to terminate processes on a Windows system. It can target processes by process ID (PID) or by name.

Common Options

Option Description
-Id Specifies the process ID (PID) of the process to stop.
-Name Specifies the name of the process to stop.
-Force Forces the termination of the process.
-PassThru Returns the process object for each stopped process.

Usage Examples 

# Stop a process by PID
Stop-Process -Id 1234

# Stop a process by name
Stop-Process -Name "notepad"

# Force stop a process by PID
Stop-Process -Id 5678 -Force

# Stop a process by name and return process object
Stop-Process -Name "explorer" -PassThru

Cheat Sheet 

# Stop by ID
Stop-Process -Id 

# Stop by Name
Stop-Process -Name ""

# Force Stop
Stop-Process -Id  -Force
