Start-VM Command in Windows

The Start-VM command is used to start a virtual machine in Windows using PowerShell.

Common Options

Option Description
-Name <String> Specifies the name of the VM to start.
-ComputerName <String> Specifies the Hyper-V host on which the VM is located.
-AsJob Runs the command as a background job.
-Passthru Returns an object representing the VM.

Usage Examples 

# Start a VM by name
Start-VM -Name "UbuntuServer01"

# Start multiple VMs
Start-VM -Name "VM1", "VM2", "VM3"

# Start a VM on a specific host
Start-VM -Name "UbuntuServer01" -ComputerName "HyperV-Host1"

# Start a VM and get the object
Start-VM -Name "UbuntuServer01" -Passthru

# Start a VM as a background job
Start-VM -Name "UbuntuServer01" -AsJob
