Remove-PSSession Command

Remove-PSSession is a PowerShell cmdlet used to close and remove one or more PowerShell sessions from a local or remote computer.

Common Options

Option Description
-Id Specifies the session ID to remove.
-Session Specifies the session objects to remove.
-InstanceId Specifies the instance ID of the session to remove.
-ComputerName Specifies the names of the computers where the sessions are stored.
-ThrottleLimit Limits the number of simultaneous operations.

Usage Examples 

# Remove a session by ID
Remove-PSSession -Id 2

# Remove multiple sessions by session object
$sessions = Get-PSSession | Where-Object { $_.State -eq "Open" }
Remove-PSSession -Session $sessions

# Remove a session from a specific computer
Remove-PSSession -Id 3 -ComputerName "Server01"

Cheat Sheet 

# Remove session by ID
Remove-PSSession -Id 

# Remove session by object
Remove-PSSession -Session 

# Remove session by instance ID
Remove-PSSession -InstanceId
