Add-LocalGroupMember Command

The Add-LocalGroupMember command is used to add users or groups to a local security group in Windows.

Common Options

Option Description
-Group Specifies the name of the local group.
-Member Identifies the users or groups to add to the group.
-Credential Specifies the user credentials to use for the command.
-Confirm Prompts for confirmation before executing the command.
-WhatIf Shows what would happen if the command runs, without executing it.

Usage Examples 

# Adds a user to the Administrators group
Add-LocalGroupMember -Group "Administrators" -Member "Username"
# Adds multiple users to the Users group
Add-LocalGroupMember -Group "Users" -Member "User1","User2"
# Using -WhatIf to simulate adding a user
Add-LocalGroupMember -Group "Remote Desktop Users" -Member "Username" -WhatIf

Cheat Sheet 

# Add user to group
Add-LocalGroupMember -Group "GroupName" -Member "Username"
# Simulate adding
Add-LocalGroupMember -Group "GroupName" -Member "Username" -WhatIf
