VS Code Command

Description

VS Code command is used to launch and control the Visual Studio Code editor from the terminal or command line interface.

Common Options

Option Description
code . Open the current directory in VS Code.
code [file] Open a specific file in VS Code.
--new-window Force opening a new instance of VS Code.
--reuse-window Force opening the file or folder in an already opened window.
--wait Wait for the files to be closed before returning.
--list-extensions List installed extensions.
--install-extension Install an extension.
--uninstall-extension Uninstall an extension.

Usage Examples 

# Open the current directory
code .

# Open a specific file
code example.txt

# Open a directory in a new window
code /path/to/folder --new-window

# Install an extension
code --install-extension ms-python.python

# Uninstall an extension
code --uninstall-extension ms-python.python

# List all installed extensions
code --list-extensions
