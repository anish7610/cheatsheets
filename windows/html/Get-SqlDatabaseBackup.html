Get-SqlDatabaseBackup Command Overview

The Get-SqlDatabaseBackup command retrieves information about SQL Server database backups.

Common Options

Option Description
-ServerInstance Specifies the SQL Server instance to connect to.
-Database Filters results for a specific database.
-LastBackup Shows the most recent backup.
-Credential Provides credentials for authentication.

Usage Examples 

# Retrieve all backups for a specific database
Get-SqlDatabaseBackup -ServerInstance "MyServer" -Database "MyDatabase"

# Show the last backup for all databases on an instance
Get-SqlDatabaseBackup -ServerInstance "MyServer" -LastBackup

# Use specific credentials to get backup information
Get-SqlDatabaseBackup -ServerInstance "MyServer" -Credential $MyCredential

Cheat Sheet 

Get-SqlDatabaseBackup -ServerInstance "Server" -Database "DB"
Get-SqlDatabaseBackup -ServerInstance "Server" -LastBackup
