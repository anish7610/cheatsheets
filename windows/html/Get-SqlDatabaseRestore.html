Get-SqlDatabaseRestore Command Overview

The Get-SqlDatabaseRestore command is used to retrieve information about SQL Server database restores. It provides details on restore history, including date, restore type, and the source of the restore.

Common Options

Option Description
-ServerInstance Specify the SQL Server instance to connect to.
-Database Specify the name of the database to retrieve restore information for.
-Credential Provide a SQL Server authentication credential.
-Top Limit the number of restore records returned.

Usage Examples 

# Retrieve restore information for a specific database
Get-SqlDatabaseRestore -ServerInstance "Server\Instance" -Database "MyDatabase"

# Get the top 5 restore records for a database
Get-SqlDatabaseRestore -ServerInstance "Server\Instance" -Database "MyDatabase" -Top 5

# Use a specific credential to connect to SQL Server
Get-SqlDatabaseRestore -ServerInstance "Server\Instance" -Database "MyDatabase" -Credential (Get-Credential)

Cheat Sheet
