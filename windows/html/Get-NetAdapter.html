Get-NetAdapter Command

The Get-NetAdapter command in Windows PowerShell retrieves the basic network adapter properties and status on a system.

Common Options

Option Description
-Name Specifies the name of the network adapter.
-InterfaceDescription Filters the output by the interface description.
-InterfaceIndex Filters the output by the interface index.
-Physical Displays only physical network adapters.
-Status Filters results by adapter status (Up, Down).

Usage Examples 

        
# List all network adapters
Get-NetAdapter

# Get a specific adapter by name
Get-NetAdapter -Name "Ethernet0"

# Get adapters with 'Up' status
Get-NetAdapter -Status Up

# Display only physical adapters
Get-NetAdapter -Physical

Cheat Sheet 

        
Get-NetAdapter                # List all adapters
Get-NetAdapter -Name "eth0"   # Specific adapter
Get-NetAdapter -Status Up     # Adapters Up
