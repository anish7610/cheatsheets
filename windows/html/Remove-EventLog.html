Remove-EventLog Command Overview

The Remove-EventLog command in Windows PowerShell is used to delete an event log or its entries.

Common Options

Option Description
-LogName Specifies the name of the event log to remove.
-ComputerName Indicates the target computer for the command.
-WhatIf Shows what would happen if the command was executed without actually performing the action.
-Confirm Prompts for confirmation before executing the command.

Usage Examples 

# Remove the 'Application' event log locally
Remove-EventLog -LogName "Application"

# Remove the 'Security' event log on a remote computer
Remove-EventLog -LogName "Security" -ComputerName "Server01"

# Check what would happen if removing the 'System' log without executing
Remove-EventLog -LogName "System" -WhatIf

# Remove the 'CustomLog' with confirmation prompt
Remove-EventLog -LogName "CustomLog" -Confirm

Cheat Sheet 


# Remove local log
Remove-EventLog -LogName "LogName"

# Remove remote log
Remove-EventLog -LogName "LogName" -ComputerName "ComputerName"

# Check without execution
Remove-EventLog -LogName "LogName" -WhatIf
