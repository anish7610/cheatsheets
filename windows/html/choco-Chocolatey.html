Chocolatey (`choco`) Command Guide

Description

Chocolatey is a package manager for Windows which allows users to install, update, and manage software packages from the command line.

Common Options

Option Description
install <package> Installs the specified package.
upgrade <package> Upgrades the specified package.
uninstall <package> Uninstalls the specified package.
list Lists installed packages.
search <package> Searches for a package.
help Displays help information.

Usage Examples 


# Install a package
choco install git

# Upgrade a package
choco upgrade git

# Uninstall a package
choco uninstall git

# List installed packages
choco list --local-only

# Search for a package
choco search git
