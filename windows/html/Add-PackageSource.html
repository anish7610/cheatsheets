Add-PackageSource Command

The Add-PackageSource command in Windows PowerShell is used to add a new package source to a specified package provider, allowing users to manage software packages from different repositories.

Common Options

Option Description
-Name The name of the package source.
-Location The URL or path to the package source.
-ProviderName The name of the package provider (e.g., NuGet).
-Trusted Indicates whether the package source is trusted.
-Credential The user credentials for accessing the package source.

Usage Examples

Example 1: Add a NuGet Package Source 


Add-PackageSource -Name "MyNuGet" -Location "https://api.nuget.org/v3/index.json" -ProviderName "NuGet" -Trusted

Example 2: Add an Internal Repository 


Add-PackageSource -Name "InternalRepo" -Location "\\Server\Repo" -ProviderName "NuGet" -Credential (Get-Credential)

Cheat Sheet 


Add-PackageSource -Name  -Location  -ProviderName  [-Trusted] [-Credential ]
