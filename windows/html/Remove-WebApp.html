Remove-WebApp Command Overview

The Remove-WebApp command is used in Windows to delete a web application from a specified Internet Information Services (IIS) site.

Common Options

Option Description
-Name Specifies the name of the web application to remove.
-Site Specifies the site from which the web application will be removed.
-WhatIf Shows what would happen if the command is run, without executing the removal.
-Confirm Prompts for confirmation before executing the removal.

Usage Examples 

# Remove a web application named 'BlogApp' from the 'Default Web Site'
Remove-WebApp -Name 'BlogApp' -Site 'Default Web Site'

# Simulate removal of 'ShopApp' from 'E-commerce Site' without executing
Remove-WebApp -Name 'ShopApp' -Site 'E-commerce Site' -WhatIf

# Remove 'AdminApp' from 'Corporate Site' with confirmation prompt
Remove-WebApp -Name 'AdminApp' -Site 'Corporate Site' -Confirm

Cheat Sheet 

Remove-WebApp -Name <AppName> -Site <SiteName>
Remove-WebApp -Name <AppName> -Site <SiteName> -WhatIf
Remove-WebApp -Name <AppName> -Site <SiteName> -Confirm
