Windows `move` Command

The move command is used to move files and directories from one location to another.

Common Options

Option Description
/Y Suppresses confirmation prompts when overwriting files.
/-Y Prompts for confirmation before overwriting files. This is the default option.

Usage Examples 


move C:\Source\File.txt D:\Destination\
move /Y C:\Source\File.txt D:\Destination\RenamedFile.txt
move C:\Source\*.txt D:\Destination\
move C:\Source\OldFolder D:\NewLocation

Cheat Sheet 


move [options] [source] [destination]
move C:\source.txt D:\dest\
move /Y C:\*.txt D:\dest\
