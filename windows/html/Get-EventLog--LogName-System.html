Get-EventLog -LogName System

The Get-EventLog command retrieves event log entries from the specified log on a Windows system. The -LogName System option targets the system log, providing information on hardware and system components.

Common Options

Option Description
-Newest Displays the most recent event log entries.
-EntryType Filters events by type (e.g., Error, Warning).
-Before Shows entries before a specific date/time.
-After Shows entries after a specific date/time.
-Source Filters events from a specific source.

Usage Examples 


# Retrieve the 10 most recent events
Get-EventLog -LogName System -Newest 10

# Get all error events from the system log
Get-EventLog -LogName System -EntryType Error

# List events after January 1, 2023
Get-EventLog -LogName System -After 01/01/2023

# Find events from a specific source
Get-EventLog -LogName System -Source "Service Control Manager"

Cheat Sheet 


# Latest 10 events
Get-EventLog -LogName System -Newest 10

# Error events
Get-EventLog -LogName System -EntryType Error

# Events after a date
Get-EventLog -LogName System -After [date]

# Events from a source
Get-EventLog -LogName System -Source [source]
