Find-Package Command Overview

The Find-Package command is used in Windows PowerShell to search for software packages from software package providers.

Common Options

Option Description
-Name Specifies the name of the package to find.
-ProviderName Specifies the provider to search in (e.g. NuGet ).
-Source Specifies the source repository to search.
-AllVersions Shows all available versions of the package.
-RequiredVersion Specifies an exact version of the package to be found.

Usage Examples 

# Find a specific package by name
Find-Package -Name "PackageName"

# Find a package using a specific provider
Find-Package -Name "PackageName" -ProviderName "NuGet"

# List all versions of a specific package
Find-Package -Name "PackageName" -AllVersions

# Find a package in a specific repository
Find-Package -Name "PackageName" -Source "RepositoryURL"
