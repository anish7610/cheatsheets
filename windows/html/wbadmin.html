Windows Command: wbadmin

wbadmin is a command-line utility in Windows for managing backup and restore operations, allowing users to create backups, restore files, and manage backup configurations.

Common Options

Option Description
start backup Initiates a backup operation.
get versions Lists details of backups available on the local computer.
start recovery Restores files from a specified backup.
enable backup Enables a scheduled daily backup.
disable backup Disables a scheduled daily backup.

Usage Examples 

wbadmin start backup -backupTarget:D: -include:C: -allCritical -quiet
wbadmin get versions -backuptarget:D:
wbadmin start recovery -version:03/15/2023-09:00 -itemType:File -items:C:\Users\Documents -recoveryTarget:C:\Recovery
