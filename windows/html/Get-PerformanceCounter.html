Get-PerformanceCounter Command

The Get-PerformanceCounter command in Windows PowerShell retrieves performance counter data, allowing users to monitor and diagnose system performance.

Common Options

Option Description
-Counter Specifies the performance counter(s) to query.
-SampleInterval Defines the time in seconds between samples.
-MaxSamples Limits the number of samples collected.

Usage Examples 

# Get CPU usage every second, 5 times
Get-PerformanceCounter -Counter "\Processor(_Total)\% Processor Time" -SampleInterval 1 -MaxSamples 5

# Monitor available memory
Get-PerformanceCounter -Counter "\Memory\Available MBytes"

# Collect multiple counters
Get-PerformanceCounter -Counter @("\Processor(_Total)\% Processor Time", "\Memory\Available MBytes") -SampleInterval 2 -MaxSamples 3

Cheat Sheet 


# Quick CPU Usage
gpc -Counter "\Processor(_Total)\% Processor Time"

# Monitor Memory
gpc -Counter "\Memory\Available MBytes"

# Multiple Counters
gpc -Counter @("\Processor\% Processor Time", "\Memory\Available MBytes")
