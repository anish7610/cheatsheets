Get-SqlAgentJob Command Overview

The Get-SqlAgentJob cmdlet retrieves SQL Server Agent jobs and their properties from an instance of SQL Server.

Common Options

Option Description
-ServerInstance Specifies the SQL Server instance to connect to.
-Name Filters to retrieve a specific job by name.
-JobId Filters to retrieve a specific job by job ID.
-AsJob Runs the command in the background as a job.

Usage Examples 

# Retrieve all jobs from a SQL Server instance
Get-SqlAgentJob -ServerInstance "YourSqlServerInstance"

# Retrieve jobs with the specific name
Get-SqlAgentJob -ServerInstance "YourSqlServerInstance" -Name "BackupDatabase"

# Retrieve jobs using job ID
Get-SqlAgentJob -ServerInstance "YourSqlServerInstance" -JobId "00000000-0000-0000-0000-000000000000"

Quick Reference 

# Get all jobs
Get-SqlAgentJob -ServerInstance "InstanceName"

# Get job by name
Get-SqlAgentJob -ServerInstance "InstanceName" -Name "JobName"

# Get job by ID
Get-SqlAgentJob -ServerInstance "InstanceName" -JobId "JobId"
