Start-Process Command

Description

The Start-Process command in PowerShell is used to start one or more processes on the local computer.

Common Options

Option Description
-FilePath Specifies the program or document to start.
-ArgumentList Specifies the parameters to pass to the executable.
-Wait Waits for the process to complete before returning.
-NoNewWindow Starts the process in the current console window.
-PassThru Returns a process object for each process started.

Examples

Open a Text File in Notepad 

Start-Process -FilePath "notepad.exe" -ArgumentList "C:\example.txt"

Run a Program and Wait for Completion 

Start-Process -FilePath "C:\Program Files\App\app.exe" -Wait

Open a URL in Default Browser 

Start-Process -FilePath "https://www.example.com"

Cheat Sheet 


Start-Process -FilePath "program.exe"
Start-Process -FilePath "program.exe" -ArgumentList "arg1 arg2"
Start-Process -FilePath "program.exe" -Wait
Start-Process -FilePath "program.exe" -NoNewWindow
