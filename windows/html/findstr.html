Findstr Command

Description

The findstr command in Windows is used to search for a specified text string in files. It supports regular expressions, making it powerful for pattern matching.

Common Options

Option Description
/I Ignore the case of the characters when searching for the string.
/S Search the specified directory and all subdirectories.
/R Use regular expressions in the search pattern.
/M Prints only the filename if a file contains a match.
/N Prints the line number before each matching line.
/C: Matches the specified string literally.

Usage Examples

Example 1: Search for a String in a File 

findstr "error" log.txt

Example 2: Case-insensitive Search Across Files 

findstr /I "warning" *.txt

Example 3: Recursive Search in Directory 

findstr /S /R "pattern" C:\Logs\*.log

Example 4: Display Line Numbers in Results 

findstr /N "critical" system.log

Cheat Sheet 


findstr /I "text" file.txt   - Case-insensitive search
findstr /S /C:"string" *     - Search string in all files
findstr /R "^pattern" *.txt  - Regex search
findstr /M "word" *.log      - Print matching files only
