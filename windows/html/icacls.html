Understanding the `icacls` Command

The `icacls` command is used in Windows to view and modify access control lists (ACLs) for files and directories.

Common Options

Option Description
/grant Grants specified user access rights.
/deny Denies specified user access rights.
/remove Removes access rights for specified user or group.
/setowner Sets the owner of a file or directory.
/inheritance Modifies inheritance settings for a file or directory.
/save Saves ACLs of specified files to a file.
/restore Restores ACLs from a file.

Usage Examples 

            
icacls C:\example.txt /grant UserName:F
icacls C:\example.txt /deny UserName:F
icacls C:\example.txt /remove UserName
icacls C:\example.txt /setowner Administrator
icacls C:\example /inheritance:d
icacls C:\example\* /save acls.txt
icacls C:\example\* /restore acls.txt
