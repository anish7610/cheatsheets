Get-SqlAgentJobHistory Command

The Get-SqlAgentJobHistory command retrieves the history of SQL Server Agent jobs, providing details on job execution, duration, and status.

Common Options

Option Description
-ServerInstance Specifies the SQL Server instance to connect to.
-JobName Filters results by the job name.
-StartRunDate Filters history to include entries from this date onward.
-EndRunDate Filters history to include entries up to this date.
-MaxRows Limits the number of history entries returned.

Usage Examples

Retrieve All Job History 

Get-SqlAgentJobHistory -ServerInstance "ServerName"

Filter by Job Name 

Get-SqlAgentJobHistory -ServerInstance "ServerName" -JobName "BackupJob"

Limit Returned Rows 

Get-SqlAgentJobHistory -ServerInstance "ServerName" -MaxRows 10

Filter by Date Range 

Get-SqlAgentJobHistory -ServerInstance "ServerName" -StartRunDate "2023-10-01" -EndRunDate "2023-10-31"

Cheat Sheet 


Get-SqlAgentJobHistory -ServerInstance "Name" [-JobName "Name"] 
                       [-StartRunDate "YYYY-MM-DD"] [-EndRunDate "YYYY-MM-DD"] 
                       [-MaxRows N]
