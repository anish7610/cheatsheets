New-EventLog Command

The New-EventLog command in Windows is used to create a new event log and configure event sources on a local or remote computer.

Common Options

Option Description
-LogName Specifies the name of the event log to create.
-Source Defines the event sources that can write to the event log.
-ComputerName Indicates the target computer where the event log will be created.

Usage Examples 

# Create a local event log named "ApplicationLog" with a source "MyApp"
New-EventLog -LogName "ApplicationLog" -Source "MyApp"

# Create an event log named "NetworkLog" on a remote computer
New-EventLog -LogName "NetworkLog" -Source "NetworkService" -ComputerName "RemotePC"

Cheat Sheet 

New-EventLog -LogName "LogName" -Source "SourceName" [-ComputerName "ComputerName"]
