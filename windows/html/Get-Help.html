Windows Get-Help Command

Description

The Get-Help command is used in Windows PowerShell to display help information about commands and concepts.

Common Options

Option Description
-Name Specifies the name of the command to get help for.
-Examples Displays examples of how the command can be used.
-Detailed Shows detailed information about the command.
-Full Provides the full help information, including parameters and examples.
-Online Opens the online version of the help in a web browser.

Usage Examples 

Get-Help Get-Process
Get-Help Get-Service -Examples
Get-Help Get-Item -Detailed
Get-Help Start-Process -Full
Get-Help Set-ExecutionPolicy -Online
