Start-Container Command

The Start-Container command in Windows is used to start an existing container that has been created but is currently stopped.

Common Options

Option Description
-Name Specifies the name of the container to start.
-Id Specifies the ID of the container to start.
-Wants Start containers that “want” a specified state.
-WhatIf Shows what would happen if the command runs without executing.

Usage Examples 

# Start a container by name
Start-Container -Name "MyContainer"

# Start a container by ID
Start-Container -Id "12345678"

# Check what would happen without actually starting
Start-Container -Name "MyContainer" -WhatIf

Cheat Sheet 

Start-Container -Name "ContainerName"
Start-Container -Id "ContainerID"
Start-Container -Name "ContainerName" -WhatIf
