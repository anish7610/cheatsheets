Description

The whoami /groups command in Windows displays group membership information for the current user, listing all the groups that the user belongs to and their attributes.

Common Options

Option Description
/groups Displays all the groups a user is a member of, along with group attributes.
/fo list Formats the output as a list.
/fo table Formats the output as a table.
/nh Omit the header row from the output.

Examples 

whoami /groups
whoami /groups /fo list
whoami /groups /fo table /nh

Cheat Sheet 

whoami /groups         # Show user group membership
whoami /groups /fo list  # List format
whoami /groups /fo table # Table format
whoami /groups /nh       # No header
