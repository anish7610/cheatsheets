IIS (Internet Information Services) Command Overview

IIS is a flexible, secure, and manageable Web server for hosting anything on the Web. It can be administered through the command line with various options for managing sites, applications, and more.

Common Options

Option Description
start site_name Starts the specified website.
stop site_name Stops the specified website.
restart site_name Restarts the specified website.
appcmd list site Lists all sites running on the IIS server.
appcmd list app Lists all applications configured on the IIS server.

Usage Examples 

# Start a website
iisreset /start "MyWebsite"

# Stop a website
iisreset /stop "MyWebsite"

# Restart a website
iisreset /restart "MyWebsite"

# List all sites
appcmd list site

# List all applications
appcmd list app

Cheat Sheet 

Start site:    iisreset /start "site_name"
Stop site:     iisreset /stop "site_name"
Restart site:  iisreset /restart "site_name"
List sites:    appcmd list site
List apps:     appcmd list app
