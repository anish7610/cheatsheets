Invoke-Expression Command Overview

Description

The Invoke-Expression command in Windows PowerShell is used to evaluate or run a string as a command or expression.

Common Options

Option Description
-Command Specifies the command or expression to be executed.
-NoNewScope Executes the command in the current scope.

Usage Examples 

# Run a simple command
Invoke-Expression "Get-Date"

# Execute a stored command in a variable
$command = "Get-Process"
Invoke-Expression $command

# Run a PowerShell script from a string
$script = "Write-Output 'Hello, World!'"
Invoke-Expression $script

Cheat Sheet 

# Execute a string as a command
Invoke-Expression "command"

# From a variable
Invoke-Expression $variable
