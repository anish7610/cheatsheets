Windows Command: tasklist

Description

The tasklist command provides a list of currently running processes on a local or remote system, displaying useful information such as process ID, session name, and memory usage.

Common Options

Option Description
/s <system> Specifies the remote system to connect to.
/u <username> Runs the command with the account permissions of the specified user.
/p <password> Specifies the password for the user account.
/fi <filter> Displays a set of tasks that match the given filter criteria.
/fo <format> Specifies the output format, such as TABLE , LIST , or CSV .
/v Displays detailed information about each task.
/m Lists all tasks currently using loaded DLL modules.

Usage Examples 

C:\> tasklist
C:\> tasklist /fi "imagename eq notepad.exe"
C:\> tasklist /v /fo csv
C:\> tasklist /s remotePC /u admin /p password /fo list

Cheat Sheet 

tasklist                    # List all tasks
tasklist /v                 # Detailed task list
tasklist /fi "status eq running" # Filter running tasks
tasklist /fo csv            # Output in CSV format
