Restore-WBFile Command

Description

The Restore-WBFile command is used in Windows PowerShell to recover files and folders from a backup created using Windows Server Backup.

Common Options

Option Description
-BackupSet Specifies the backup set from which to restore files.
-Item Defines the path(s) of the file(s) or folder(s) to restore.
-DestinationPath Specifies the path where the restored files will be placed.
-Overwrite Indicates whether to overwrite existing files at the destination.

Usage Examples 


# Restore a specific file from a backup to a given location
Restore-WBFile -BackupSet $backupSet -Item "C:\Backup\File1.txt" -DestinationPath "C:\RestoredFiles"

# Restore a directory and overwrite existing files
Restore-WBFile -BackupSet $backupSet -Item "C:\Backup\Documents" -DestinationPath "C:\RestoredDocuments" -Overwrite

Cheat Sheet
