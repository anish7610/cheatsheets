New-SqlDatabase Command

Description

The New-SqlDatabase command in PowerShell is used to create a new SQL database on a specified SQL Server instance.

Common Options

Option Description
-Name The name of the new SQL database.
-ServerInstance The SQL Server instance where the database will be created.
-Collation Specifies the collation for the newly created database.
-MaxSize Sets the maximum size of the database.
-FileGroup Designates the default filegroup for the database.

Usage Examples

Example 1: Create a Simple Database 

New-SqlDatabase -Name 'TestDB' -ServerInstance 'localhost'

Example 2: Create a Database with Specific Collation 

New-SqlDatabase -Name 'TestDB2' -ServerInstance 'localhost' -Collation 'SQL_Latin1_General_CP1_CI_AS'

Example 3: Create a Database with Maximum Size 

New-SqlDatabase -Name 'LargeDB' -ServerInstance 'localhost' -MaxSize 100GB

Cheat Sheet 


New-SqlDatabase -Name <DatabaseName> -ServerInstance <ServerName> [-Collation <CollationName>] [-MaxSize <Size>]
