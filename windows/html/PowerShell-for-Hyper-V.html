PowerShell for Hyper-V

PowerShell for Hyper-V is a command-line tool used to manage, configure, and automate tasks related to Hyper-V virtualization on Windows.

Common Options

Option Description
-VMName <String> Specifies the name of the virtual machine.
-Path <String> Specifies the path for configuration files.
-MemoryStartupBytes <Int64> Configures the initial memory assigned to the VM.
-Credential <PSCredential> Provides user credentials for authentication.
-ComputerName <String> Specifies the name of the computer to connect to.

Usage Examples 

New-VM -Name "TestVM" -MemoryStartupBytes 2GB -Path "C:\VMs\TestVM"
Start-VM -Name "TestVM"
Stop-VM -Name "TestVM"
Get-VM -Name "TestVM"
Remove-VM -Name "TestVM"

Quick Reference Cheat Sheet 

# Create a VM
New-VM -Name "VMName" -MemoryStartupBytes 4GB -Path "Path"

# Start a VM
Start-VM -Name "VMName"

# Stop a VM
Stop-VM -Name "VMName"

# List VMs
Get-VM

# Delete a VM
Remove-VM -Name "VMName"
