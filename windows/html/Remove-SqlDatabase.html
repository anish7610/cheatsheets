Remove-SqlDatabase Command

Description

The Remove-SqlDatabase command is used in PowerShell to delete a SQL Server database.

Common Options

Option Description
-ServerInstance Specifies the SQL Server instance.
-Database The name of the database to remove.
-Force Suppresses confirmation prompts.
-Confirm Prompts for confirmation before executing the command.

Usage Examples 

# Remove a database with a confirmation prompt
Remove-SqlDatabase -ServerInstance "ServerName" -Database "TestDB" -Confirm

# Remove a database without confirmation
Remove-SqlDatabase -ServerInstance "ServerName" -Database "TestDB" -Force

Cheat Sheet 

Remove-SqlDatabase -ServerInstance "ServerName" -Database "DBName"
Remove-SqlDatabase -ServerInstance "ServerName" -Database "DBName" -Force
