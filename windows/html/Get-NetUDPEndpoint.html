Get-NetUDPEndpoint Command

Description

The Get-NetUDPEndpoint command retrieves information about UDP endpoints on a system, including the local IP address, local port, and owning process.

Common Options

Option Description
-LocalAddress Specifies the local IP address to filter the results.
-LocalPort Filters endpoints by a specific local port number.
-OwningProcess Limits results to UDP connections owned by a specific process ID.

Usage Examples 

# List all UDP endpoints
Get-NetUDPEndpoint

# Filter by local IP address
Get-NetUDPEndpoint -LocalAddress 192.168.1.10

# Filter by specific local port
Get-NetUDPEndpoint -LocalPort 53

# Filter by owning process ID
Get-NetUDPEndpoint -OwningProcess 1234
