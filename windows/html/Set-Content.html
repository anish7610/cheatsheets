Set-Content Command

The Set-Content command in Windows PowerShell writes data to a file, replacing existing content.

Common Options

Option Description
-Path Specifies the file path to write content to.
-Value The content to be written into the file.
-NoNewline Prevents a newline from being added after the content.
-Encoding Specifies the file's text encoding, such as UTF8, ASCII.

Usage Examples 

# Write "Hello, World!" to a file
Set-Content -Path "example.txt" -Value "Hello, World!"

# Write without a newline
Set-Content -Path "example.txt" -Value "Hello, World!" -NoNewline

# Write with UTF8 encoding
Set-Content -Path "example.txt" -Value "Hello, World!" -Encoding UTF8

Cheat Sheet 


# Basic Usage
Set-Content -Path "file.txt" -Value "Content"

# No Newline
Set-Content -Path "file.txt" -Value "Content" -NoNewline
