Set-Acl Command in Windows

Description

The Set-Acl command in Windows PowerShell is used to modify the access control list (ACL) of a file or directory. This command allows you to set permissions and control access to file system objects.

Common Options

Option Description
-Path Specifies the path to the item to update the ACL on.
-AclObject Provides the access control list to apply to the specified path.
-LiteralPath Specifies the path to the item, without wildcard expansion.
-WhatIf Shows what would happen if the command is executed, without making changes.

Usage Examples

Example 1: Set ACL on a File 

Get-Acl -Path "C:\example\file.txt" | Set-Acl -Path "C:\example\file.txt"

Example 2: Set ACL on a Directory using a Saved ACL 


$acl = Get-Acl -Path "C:\example\template\"
Set-Acl -Path "C:\example\newfolder\" -AclObject $acl

Cheat Sheet 


Set-Acl -Path <path> -AclObject <acl>
Set-Acl -LiteralPath <path> -AclObject <acl>
