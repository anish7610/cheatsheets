Get-Partition Command Overview

The Get-Partition command in Windows PowerShell retrieves information about the disk partitions available on a system. It helps manage, query, and troubleshoot disk partitions.

Common Options

Option Description
-DiskNumber Specifies the disk number to query partitions from.
-PartitionNumber Specifies a particular partition to get details about.
-CimSession Runs the command on a remote session or computer.

Usage Examples 

# Retrieve all partitions on all disks
Get-Partition

# Get details of all partitions on disk 1
Get-Partition -DiskNumber 1

# Retrieve information about a specific partition on a specific disk
Get-Partition -DiskNumber 1 -PartitionNumber 2

Cheat Sheet 

Get-Partition               # List all partitions
Get-Partition -DiskNumber n # Partitions on disk n
Get-Partition -PartitionNumber m -DiskNumber n # Details of partition m on disk n
