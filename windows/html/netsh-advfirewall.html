netsh advfirewall Overview

The netsh advfirewall command is a Windows tool used to manage and configure the Windows Firewall with Advanced Security, providing control over firewall settings for network profiles, rules, and more.

Common Options

Option Description
set allprofiles state Enable or disable the firewall for all profiles (Domain, Private, Public)
add rule Add a new inbound or outbound rule
delete rule Remove an existing rule by name
show currentprofile Display the current active firewall profile settings
set logging Configure logging options for the firewall

Usage Examples 

netsh advfirewall set allprofiles state on
netsh advfirewall set allprofiles state off
netsh advfirewall firewall add rule name="Allow SSH" protocol=TCP dir=in localport=22 action=allow
netsh advfirewall firewall delete rule name="Allow SSH"
netsh advfirewall show currentprofile

Cheat Sheet 

Enable all: netsh advfirewall set allprofiles state on
Disable all: netsh advfirewall set allprofiles state off
Add rule: netsh advfirewall firewall add rule name="[RuleName]" protocol=[Protocol] dir=[in|out] localport=[Port] action=[allow|block]
Delete rule: netsh advfirewall firewall delete rule name="[RuleName]"
Show profile: netsh advfirewall show currentprofile
