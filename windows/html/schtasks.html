Windows `schtasks` Command

The `schtasks` command is used to create, delete, query, change, run, and end scheduled tasks on a Windows machine.

Common Options

Option Description
/Create Create a new scheduled task.
/Delete Delete an existing scheduled task.
/Query Display all scheduled tasks or detailed information about specific tasks.
/Change Modify a scheduled task's settings.
/Run Execute a scheduled task on demand.
/End Stop a running scheduled task.

Usage Examples 

s:> schtasks /Create /SC DAILY /TN "Backup" /TR "C:\Backup.bat" /ST 02:00
s:> schtasks /Delete /TN "Backup"
s:> schtasks /Query /TN "Backup"
s:> schtasks /Change /TN "Backup" /ST 03:00
s:> schtasks /Run /TN "Backup"
s:> schtasks /End /TN "Backup"

Cheat Sheet 


Create: schtasks /Create /SC {FREQ} /TN {NAME} /TR {TASK} /ST {TIME}
Delete: schtasks /Delete /TN {NAME}
Query:  schtasks /Query /TN {NAME}
Change: schtasks /Change /TN {NAME} /ST {TIME}
Run:    schtasks /Run /TN {NAME}
End:    schtasks /End /TN {NAME}
