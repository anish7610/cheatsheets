Get-VM Command Overview

The Get-VM command is used in Windows PowerShell to retrieve information about virtual machines (VMs) on a Hyper-V server.

Common Options

Option Description
-Name Specifies the name of the virtual machine to retrieve.
-ComputerName Specifies the Hyper-V host from which to retrieve virtual machines.
-Id Specifies the unique identifier of the virtual machine.

Usage Examples

Retrieve All Virtual Machines 

Get-VM

Retrieve a Specific Virtual Machine by Name 

Get-VM -Name "MyVM"

Retrieve Virtual Machines from a Specific Host 

Get-VM -ComputerName "HyperVHost"

Retrieve a Virtual Machine by ID 

Get-VM -Id "12345-abcde-67890"

Cheat Sheet 

Get-VM                      # List all VMs
Get-VM -Name "Name"         # List VM by name
Get-VM -ComputerName "Host" # List VMs from a host
Get-VM -Id "ID"             # List VM by ID
