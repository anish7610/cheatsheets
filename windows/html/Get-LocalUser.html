Get-LocalUser Command Overview

Get-LocalUser is a Windows PowerShell command used to retrieve information about local user accounts on a system.

Common Options

Option Description
-Name Specifies the name of the user to retrieve details for.
-SID Displays the security identifier for a user.

Usage Examples 


# List all local users
Get-LocalUser

# Get details for a specific user by name
Get-LocalUser -Name "Username"

# Get the security identifier (SID) of a user
(Get-LocalUser -Name "Username").SID

Cheat Sheet 


Get-LocalUser              # List all users
Get-LocalUser -Name "User" # Details for specific user
(Get-LocalUser -Name "User").SID # Get user SID
