Remove-VM Command

Description

The Remove-VM command is used in Windows PowerShell to delete virtual machines from Hyper-V. It removes the specified VM along with its configuration and associated resources.

Common Options

Option Description
-Name Name of the VM to remove.
-Force Removes the VM without confirmation prompts.
-WhatIf Shows what would happen if the command runs, without executing it.
-Confirm Prompts for confirmation before executing the command.

Usage Examples 

# Remove a specific VM by name
Remove-VM -Name "TestVM"

# Remove a VM without confirmation prompts
Remove-VM -Name "TestVM" -Force

# Simulate VM removal to see potential effects
Remove-VM -Name "TestVM" -WhatIf

# Remove a VM with confirmation prompt
Remove-VM -Name "TestVM" -Confirm

Cheat Sheet 

# Basic Remove
Remove-VM -Name "VMName"

# Force Remove
Remove-VM -Name "VMName" -Force

# Simulate Remove
Remove-VM -Name "VMName" -WhatIf
