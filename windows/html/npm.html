npm Command in Windows

The npm command is a package manager for Node.js, used to install, update, and manage JavaScript packages.

Common Options

Option Description
install Install a package to your project.
update Update all the outdated packages.
uninstall Remove a package from your project.
list List installed packages.
init Create a new package.json file.

Usage Examples 



npm install express


npm update


npm uninstall lodash


npm list -g --depth=0


npm init

Cheat Sheet 


npm install <package>  # Install a package
npm update                  # Update packages
npm uninstall <package>  # Remove a package
npm list                    # List packages
npm init                    # Create package.json
