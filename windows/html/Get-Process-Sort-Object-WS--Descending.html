Understanding `Get-Process | Sort-Object WS -Descending`

Command Description

The command retrieves a list of active processes on a Windows system, sorts them by memory usage (Working Set), and displays them in descending order.

Common Options

Option Description
-Name Filters processes by name.
-Id Filters processes by process ID.
-FileVersionInfo Includes file version information for process modules.
-Module Includes module information for the processes.
-ComputerName Specifies the name of a remote computer.

Usage Examples 

Get-Process | Sort-Object WS -Descending

Lists processes sorted by descending memory usage. 

Get-Process -Name chrome | Sort-Object WS -Descending

Filters and sorts Chrome processes by memory usage. 

Get-Process -Id 1234

Displays process details for a specific process ID.
