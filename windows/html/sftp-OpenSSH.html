sftp (OpenSSH) Command Overview

Description

The sftp command in OpenSSH is a secure file transfer program that operates over SSH, enabling secure file transfers between a local and a remote system.

Common Options

Option Description
-b batchfile Runs commands from the specified batch file.
-C Enables compression.
-i identity_file Specifies a private key for user authentication.
-P port Specifies the port to connect to on the remote host.
-r Recursively copies entire directories.

Usage Examples

Connect to Remote Host 

sftp user@host

Download a File 

sftp user@host:/path/to/remote/file /local/destination

Upload a File 

sftp /local/file user@host:/path/to/remote/destination

Download a Directory 

sftp -r user@host:/path/to/remote/dir /local/destination

Cheat Sheet 


# Connect
sftp user@host

# Download file
sftp user@host:/remote/file /local/path

# Upload file
sftp /local/file user@host:/remote/path

# Use specific port
sftp -P port user@host
