Stop-Job Command in Windows

Description

The Stop-Job command in Windows PowerShell is used to stop one or more running background jobs.

Common Options

Option Description
-Id Specifies the ID of the job(s) to stop.
-Name Specifies the name of the job(s) to stop.
-InputObject Accepts a job object representing the job(s) to stop.
-Force Forces the job to stop, even if it's in a running state.

Usage Examples 

# Stop a job by ID
Stop-Job -Id 2

# Stop a job by name
Stop-Job -Name "MyBackgroundJob"

# Stop all running jobs
Get-Job | Stop-Job

# Stop a job using the job object
$job = Get-Job -Id 3
Stop-Job -InputObject $job

# Forcefully stop a job
Stop-Job -Id 5 -Force

Cheat Sheet 

Stop-Job -Id <job_id>
Stop-Job -Name <job_name>
Stop-Job -InputObject <job_object>
Stop-Job -Force
