Taskkill Command Guide

Description

The taskkill command is used in Windows to terminate tasks or processes by their process ID (PID) or image name.

Common Options

Option Description
/F Forcefully terminate the process.
/IM Terminate by image name (e.g., notepad.exe).
/PID Terminate by process ID.
/T Terminate the specified process and any child processes.

Usage Examples 

taskkill /IM notepad.exe /F
taskkill /PID 1234
taskkill /IM chrome.exe /T

Cheat Sheet 


taskkill /IM [imagename] /F
taskkill /PID [processid]
taskkill /IM [imagename] /T
