Import-Module Command Overview

The Import-Module command in Windows PowerShell is used to import specified modules into the current session, enabling access to the cmdlets, functions, and aliases that the module contains.

Common Options

Option Description
-Name Specifies the name of the module to import.
-Force Forces the module to be reloaded even if it is already imported.
-PassThru Returns a module object representing the imported module.
-Scope Indicates the scope in which the module is imported (e.g., Global).
-Verbose Provides detailed information about the import operation.

Usage Examples 

Import-Module -Name 'ActiveDirectory'
Import-Module -Name 'Azure' -Force
$module = Import-Module -Name 'PSReadline' -PassThru

Cheat Sheet 


Import-Module -Name 'ModuleName'
Import-Module -Name 'ModuleName' -Force
Import-Module -Name 'ModuleName' -Verbose
