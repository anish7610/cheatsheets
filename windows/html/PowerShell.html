Guide to PowerShell Command

Description

PowerShell is a task automation and configuration management framework consisting of a command-line shell and scripting language.

Common Options

Option Description
-NoExit Does not exit after running startup commands.
-Command Executes specified commands and exits.
-File Executes a script.
-ExecutionPolicy Changes the user preference for the PowerShell script execution policy.
-WindowStyle Sets the window style (Normal, Minimized, Maximized, Hidden).

Usage Examples 


# Open PowerShell and keep it running
powershell -NoExit

# Execute a simple command
powershell -Command "Get-Process"

# Run a script file
powershell -File "C:\Path\To\Script.ps1"

# Open PowerShell with a minimized window
powershell -WindowStyle Minimized

# Set the execution policy for the session
powershell -ExecutionPolicy Bypass
