Windows Command: Get-Acl

Description

The Get-Acl command retrieves the access control list (ACL) of a specified file or resource, showing permissions and security information.

Common Options

Option Description
-Path Specifies the path to the file or resource.
-LiteralPath Specifies a path that is used exactly as typed.
-Audit Retrieves the audit rules of the specified path.
-AllCentralAccessPolicies Retrieves all central access policies of the specified path.

Usage Examples 

# Retrieve ACL of a file
Get-Acl -Path "C:\example\file.txt"

# Retrieve ACL using LiteralPath to handle special characters
Get-Acl -LiteralPath "C:\path[with]special#chars?.txt"

# View ACL with audit information
Get-Acl -Path "C:\example\file.txt" -Audit

Cheat Sheet 

Get-Acl -Path "file_or_directory_path"
Get-Acl -LiteralPath "exact_path"
Get-Acl -Path "path" -Audit
