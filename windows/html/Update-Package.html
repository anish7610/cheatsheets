Windows Command: Update-Package

Description

The Update-Package command in Windows PowerShell updates an installed package to its latest available version, typically used with NuGet packages in development environments.

Common Options

Option Description
-ProjectName Specifies the project to update the package for.
-Id Updates a specific package by its identifier.
-Source Specifies the package source to use for the update.
-Version Specifies the version to update the package to.
-WhatIf Shows what would happen if the command runs, without executing it.

Usage Examples 

      
# Update all packages in a project
Update-Package

# Update a specific package in a single project
Update-Package -Id ExamplePackage -ProjectName MyProject

# Update to a specific version from a specific source
Update-Package -Id ExamplePackage -Version 2.0.0 -Source MySource

Cheat Sheet 

      
Update-Package -Id PackageId -ProjectName Project
Update-Package -Id PackageId -Version Version -Source Source
