Get-EventLog Command

Description

The Get-EventLog command in Windows PowerShell retrieves entries from the event logs on local or remote computers. The -LogName Application option specifies the Application log.

Common Options

Option Description
-LogName Specifies the name of the log to retrieve.
-Newest Gets the most recent entries.
-EntryType Filters events by type (e.g., Error, Warning, Information).
-Source Filters events by source name.
-After Gets events that occurred after a specific date and time.
-Before Gets events that occurred before a specific date and time.

Usage Examples 

Get-EventLog -LogName Application -Newest 10

Retrieves the 10 most recent events from the Application log. 

Get-EventLog -LogName Application -EntryType Error

Filters and displays only error events from the Application log. 

Get-EventLog -LogName Application -Source "SomeSource"

Shows the events from the Application log generated by the specified source.
