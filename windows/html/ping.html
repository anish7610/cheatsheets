Ping Command in Windows

Description

The ping command is used to test the network connectivity between a source computer and a specified destination, typically using ICMP echo requests and replies.

Common Options

Option Description
-t Pings the specified host until stopped. Use Ctrl+C to stop.
-n Specifies the number of echo requests to send. Example: -n 5 .
-l Sets the size of the send buffer. Example: -l 1024 .
-4 Forces using IPv4.
-6 Forces using IPv6.

Usage Examples 

ping google.com

Pings Google's server to check connectivity. 

ping -n 3 example.com

Sends 3 echo requests to example.com and displays the results. 

ping -t 192.168.1.1

Pings the IP address 192.168.1.1 continuously until stopped.
