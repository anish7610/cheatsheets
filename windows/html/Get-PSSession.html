Get-PSSession Overview

The Get-PSSession command in Windows PowerShell retrieves the current PowerShell sessions (PSSessions) on the local computer. These sessions can be used to run remote commands.

Common Options

Option Description
-Name Specify the session name to filter returned sessions.
-ComputerName Return sessions connected to specified computers.
-InstanceId Retrieve sessions using specific instance IDs.
-State Filter sessions by their state (e.g., Opened, Closed).

Usage Examples 

# Retrieve all active sessions
Get-PSSession

# Filter sessions by name
Get-PSSession -Name "SessionName"

# List sessions for a specific computer
Get-PSSession -ComputerName "RemotePC"

# Find sessions by instance ID
Get-PSSession -InstanceId "GUID"

# Get sessions that are opened
Get-PSSession -State "Opened"

Quick Reference Cheat Sheet 

# List all sessions
Get-PSSession

# Filter by name
Get-PSSession -Name "YourSessionName"

# Sessions by computer
Get-PSSession -ComputerName "YourComputerName"

# By state
Get-PSSession -State "Opened"
