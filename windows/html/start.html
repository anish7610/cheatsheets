Windows `start` Command

A command used to open a new window, or launch an application or document.

Common Options

Option Description
/MAX Start the program maximized.
/MIN Start the program minimized.
/WAIT Start the program and wait for it to end.
/B Start without creating a new window. Handles Ctrl+C .
Title Specified title for the new window.
Path or Command Path to the executable or document.

Realistic Usage Examples 

start notepad.exe
start /MAX explorer
start /WAIT calc

Cheat Sheet 

start /MAX [program]    # Maximize window
start /MIN [program]    # Minimize window
start /WAIT [program]   # Wait for program to close
start "" [program]      # Open with title
