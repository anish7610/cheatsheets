WMIC Process Command

The wmic process command in Windows is used to interact with and retrieve information about running processes.

Common Options

Option Description
/output:<filename> Redirects output to a file.
get name,processid Lists process names with their IDs.
where "name='process.exe'" Filters processes by name.
call terminate Terminates a specified process.

Usage Examples 

wmic process list brief
wmic process get name,processid
wmic process where "name='notepad.exe'" get processid
wmic process where processid=1234 call terminate
wmic process get name,processid /output:c:\processes.txt
