whoami Command in Windows

Description

The whoami command displays the username of the current user session. It's useful for verifying user identity in scripts or command-line workflows.

Common Options

Option Description
/user Displays detailed user information.
/groups Displays the groups the current user belongs to.
/fqdn Displays the fully qualified domain name.
/logonid Displays the user's logon ID.

Examples 

whoami

Displays the current username. 

whoami /user

Displays detailed information about the current user. 

whoami /groups

Shows the groups the user belongs to.

Cheat Sheet 

whoami          # Show current username
whoami /user    # Detailed user info
whoami /groups  # List user groups
whoami /fqdn    # Fully qualified domain name
