Get-SqlDatabase Command Overview

The Get-SqlDatabase command in Windows PowerShell is used to retrieve information about SQL Server databases. It is part of the SQL Server module for PowerShell.

Common Options

Option Description
-ServerInstance Specifies the SQL Server instance to query.
-DatabaseName Filters to a specific database by name.
-Credential Provides login credentials for accessing the server.

Usage Examples 

# Retrieve all databases on a specific server instance
Get-SqlDatabase -ServerInstance "MyServer\SQLInstance"

# Retrieve a specific database using name
Get-SqlDatabase -ServerInstance "MyServer\SQLInstance" -DatabaseName "MyDatabase"

# Using credentials to access the server
Get-SqlDatabase -ServerInstance "MyServer\SQLInstance" -Credential (Get-Credential)

Cheat Sheet 


Get-SqlDatabase -ServerInstance "Server\Instance" [-DatabaseName "Name"] [-Credential (Get-Credential)]
