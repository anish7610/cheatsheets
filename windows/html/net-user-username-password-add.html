Windows Command: net user [username] [password] /add

The net user command is used to manage user accounts on Windows systems. The /add option creates a new user account.

Common Options

Option Description
/add Adds a new user account.
/delete Deletes an existing user account.
/active:{yes|no} Enables or disables an account.
/passwordchg:{yes|no} Allows or disallows password changes.
/expires:{date|never} Sets an expiration date for the account.
/fullname:" Name " Sets the user's full name.

Usage Examples 

net user alice P@ssw0rd /add
net user bob /delete
net user charlie /active:no
net user dave /passwordchg:no

Cheat Sheet 

net user username password /add
net user username /delete
net user username /active:{yes|no}
net user username /passwordchg:{yes|no}
