Get-ComputerInfo Command Overview

Description

The Get-ComputerInfo command in Windows PowerShell is used to retrieve detailed information about the computer's hardware and operating system.

Common Options

Option Description
-Property Specifies which properties to display, provides flexibility to view specific details.
-ErrorAction Determines the action to take if the command encounters an error.

Usage Examples 


# Retrieve all computer information
Get-ComputerInfo

# View only specific properties, e.g., OS version and architecture
Get-ComputerInfo -Property "CsName", "WindowsVersion", "WindowsBuildLabEx"

# Handle errors gracefully
Get-ComputerInfo -ErrorAction SilentlyContinue

Cheat Sheet 


# Basic Information
Get-ComputerInfo

# Specific Properties
Get-ComputerInfo -Property "WindowsVersion"

# Error Handling
Get-ComputerInfo -ErrorAction Stop
