Remove-LocalGroupMember Command

The Remove-LocalGroupMember command removes users or groups from a local security group on a Windows system.

Common Options

Option Description
-Group Specifies the name or SID of the local group.
-Member Specifies the users or groups to remove (name or SID).
-Confirm Prompts for confirmation before executing the command.
-WhatIf Shows what would happen if the command runs, without executing it.

Usage Examples 

# Remove a single user from the Administrators group
Remove-LocalGroupMember -Group "Administrators" -Member "JohnDoe"

# Remove multiple users from the Users group
Remove-LocalGroupMember -Group "Users" -Member "User1", "User2"

# Remove a user with confirmation prompt
Remove-LocalGroupMember -Group "Guests" -Member "GuestUser" -Confirm

# Simulate removing a user to see potential changes
Remove-LocalGroupMember -Group "Power Users" -Member "TestUser" -WhatIf

Cheat Sheet 

Remove-LocalGroupMember -Group <GroupName> -Member <UserName> [-Confirm] [-WhatIf]
