Get-Website Command

Description

The Get-Website command is used in Windows PowerShell to retrieve information about websites configured on an IIS (Internet Information Services) server.

Common Options

Option Description
-Name Specify the name of the website to retrieve. If omitted, retrieves all websites.
-PhysicalPath Include the physical path of the website in the output.
-State Include the state (Started, Stopped) of the website in the output.

Usage Examples 

# Retrieve all websites
Get-Website

# Retrieve a specific website by name
Get-Website -Name "MySite"

# Retrieve websites including their physical paths
Get-Website | Select-Object Name, PhysicalPath

# Retrieve websites with their state
Get-Website | Select-Object Name, State

Cheat Sheet 

Get-Website               # List all websites
Get-Website -Name "Site"   # List specific site
Get-Website | Select-Object Name, PhysicalPath, State
