ipconfig Command Guide

Description

The ipconfig command is used in Windows to display and manage the network configuration of the local computer.

Common Options

Option Description
/all Displays detailed information about the network configuration.
/release Releases the current DHCP configuration for all adapters.
/renew Renews the DHCP configuration for all adapters.
/flushdns Clears the DNS resolver cache.
/displaydns Displays the contents of the DNS resolver cache.

Usage Examples 

            
ipconfig /all
ipconfig /release
ipconfig /renew
ipconfig /flushdns
ipconfig /displaydns

Cheat Sheet 

            
ipconfig /all         # View detailed config
ipconfig /release     # Release DHCP lease
ipconfig /renew       # Renew DHCP lease
ipconfig /flushdns    # Clear DNS cache
ipconfig /displaydns  # Show DNS cache
