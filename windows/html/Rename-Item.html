Rename-Item Command

The Rename-Item cmdlet in PowerShell is used to rename files, directories, or other items.

Common Options

Option Description
-Path Specifies the path to the item being renamed.
-NewName Specifies the new name for the item.
-Force Forces the command to rename an item, even read-only files or files in use.
-PassThru Returns an object that represents the renamed item.

Usage Examples

Basic File Rename 

Rename-Item -Path "C:\Example\oldname.txt" -NewName "newname.txt"

Directory Rename with Force 

Rename-Item -Path "C:\Example\OldFolder" -NewName "NewFolder" -Force

Renaming and Returning the New Name 

Rename-Item -Path "C:\Example\oldfile.txt" -NewName "newfile.txt" -PassThru

Cheat Sheet 


- Basic Rename: Rename-Item -Path <old_name> -NewName <new_name>
- With Force: Rename-Item -Path <old_name> -NewName <new_name> -Force
- Return Object: Rename-Item -Path <old_name> -NewName <new_name> -PassThru
