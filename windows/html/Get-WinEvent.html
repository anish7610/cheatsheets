Get-WinEvent Command Overview

Description

The Get-WinEvent command retrieves event log data from local and remote computers, supporting both classic and Windows Event logs.

Common Options

Option Description
-LogName Specifies the name of the event log to query.
-ProviderName Filters events by provider.
-MaxEvents Limits the number of events returned.
-FilterHashtable Uses a hashtable to filter events.
-Oldest Sorts events in oldest-first order.
-Newest Sorts events in newest-first order.
-ComputerName Specifies the remote computer(s) to query.
-Credential Specifies a user account for the connection.

Examples 


# Retrieve the last 10 events from the System log
Get-WinEvent -LogName System -MaxEvents 10

# Retrieve events from the Application log, filtered by a specific provider
Get-WinEvent -LogName Application -ProviderName "Microsoft-Windows-Security-Auditing"

# Get events from a remote computer
Get-WinEvent -LogName Security -ComputerName "RemotePC" -Credential (Get-Credential)

# Filter events using a hashtable for EventID 4624
Get-WinEvent -FilterHashtable @{LogName='Security'; ID=4624}
