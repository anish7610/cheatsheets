Windows Command: Get-NetIPAddress

The Get-NetIPAddress cmdlet retrieves IP address configuration information on a device.

Common Options

Option Description
-InterfaceAlias Specifies the network interface by name.
-AddressFamily Limits results to IPv4 or IPv6 addresses.
-IPAddress Filters results for a specific IP address.
-CimSession Runs the command on a remote device.

Usage Examples 

# Retrieve all IP addresses
Get-NetIPAddress

# Get IPv4 addresses only
Get-NetIPAddress -AddressFamily IPv4

# Retrieve IPs for a specific interface
Get-NetIPAddress -InterfaceAlias "Ethernet"

# Filter for a specific IP address
Get-NetIPAddress -IPAddress 192.168.1.1
