Get-LocalGroupMember Command

Description

The Get-LocalGroupMember command retrieves members of a local group on Windows systems.

Common Options

Option Description
-Group Specifies the name of the local group.
-Member Filters results to include only specified members.
-Credential Runs the command with specified user credentials.

Examples 

# Example 1: Get all members of the Administrators group
Get-LocalGroupMember -Group "Administrators"

# Example 2: Get a specific member from a group
Get-LocalGroupMember -Group "Users" -Member "JohnDoe"

Cheat Sheet 

Get-LocalGroupMember -Group "GroupName"
Get-LocalGroupMember -Group "GroupName" -Member "MemberName"
