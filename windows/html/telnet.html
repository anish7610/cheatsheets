Windows Telnet Command

Description

The telnet command is used to communicate with another host using the Telnet protocol. It is mainly used for troubleshooting and remote access to servers.

Common Options

Option Description
-a Attempts automatic logon with the current user name.
-e escapechar Sets the escape character. Default is CTRL+] .
-f Enables file transfer.
-l user Specifies user name for logon.
-t term Specifies terminal type. Supported types are VT100, VT52, ANSI.

Usage Examples 

# Connect to a remote server on port 23
telnet example.com 23

# Connect with automatic logon using current user name
telnet -a example.com 23

# Connect specifying user name for logon
telnet -l username example.com 23

Cheat Sheet 

telnet host [port]           # Connect to host
telnet -a host [port]       # Automatic logon
telnet -e c host [port]     # Set escape char to c
telnet -l user host [port]  # Logon with user
