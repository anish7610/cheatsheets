New-PSSession Command Overview

The New-PSSession command in Windows PowerShell creates a persistent connection to a local or remote computer.

Common Options

Option Description
-ComputerName Specifies the remote computer(s) to connect to.
-Credential Specifies the user account credentials to use for the session.
-ConfigurationName Specifies the session configuration to use.
-Port Specifies the port to connect to on the remote computer.
-UseSSL Establishes a secure connection using SSL.

Usage Examples 

# Create a session to a single remote computer
$s = New-PSSession -ComputerName "Server01"

# Create a session with specific credentials
$cred = Get-Credential
$s = New-PSSession -ComputerName "Server01" -Credential $cred

# Create a session using a secure SSL connection
$s = New-PSSession -ComputerName "Server01" -UseSSL

# Create a session with a specific configuration
$s = New-PSSession -ComputerName "Server01" -ConfigurationName "MyConfig"

Cheat Sheet 

# Basic session
New-PSSession -ComputerName "ServerName"

# With credentials
New-PSSession -ComputerName "ServerName" -Credential (Get-Credential)

# Secure connection
New-PSSession -ComputerName "ServerName" -UseSSL

# Custom configuration
New-PSSession -ComputerName "ServerName" -ConfigurationName "ConfigName"
