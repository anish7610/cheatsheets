Exit-PSSession Command

The Exit-PSSession command is used in PowerShell to end an interactive session with a remote computer.

Common Options

Option Description
-WhatIf Shows what would happen if the command runs without executing it.
-Confirm Prompts for confirmation before executing the command.

Usage Examples 

# End a remote session gracefully
Exit-PSSession

# Preview the consequences of ending a session
Exit-PSSession -WhatIf

# Prompt for confirmation before exiting the session
Exit-PSSession -Confirm

Cheat Sheet 

Exit-PSSession               # End remote session
Exit-PSSession -WhatIf       # Preview end
Exit-PSSession -Confirm      # Confirm before exit
