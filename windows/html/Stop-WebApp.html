Stop-WebApp Command

The Stop-WebApp command is used to stop a web application running within an IIS web server in Windows.

Common Options

Option Description
-Name Specifies the name of the web application to stop.
-Site Specifies the site where the web application is hosted.

Usage Examples 

# Stop a web application named 'MyApp' in the default website
Stop-WebApp -Name "MyApp" -Site "Default Web Site"

# Stop a specific web application on a custom site
Stop-WebApp -Name "BlogApp" -Site "MyBlog"

Quick Reference Cheat Sheet 

# Basic usage
Stop-WebApp -Name "AppName" -Site "SiteName"
