System Restore Command Guide

Description

The `System Restore` command in Windows allows users to roll back system files, program files, and registry settings to a previous state using restore points, aiding in system recovery.

Common Options

Option Description
/r Restarts the computer and initiates System Restore.
/create Creates a new restore point manually.
/status Displays the current status of System Restore.
/disable Disables System Restore for all drives.
/enable Enables System Restore for all drives.

Usage Examples

Create a Restore Point 

systemrestore /create

Initiate System Restore 

systemrestore /r

Check System Restore Status 

systemrestore /status

Disable System Restore 

systemrestore /disable

Enable System Restore 

systemrestore /enable

Cheat Sheet 


    Create     : systemrestore /create
    Restore    : systemrestore /r
    Status     : systemrestore /status
    Disable    : systemrestore /disable
    Enable     : systemrestore /enable
