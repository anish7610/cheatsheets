Net Group Command in Windows

Description

The net group command is used to manage group accounts on Windows servers or domains. It allows you to add, delete, and manage user group memberships.

Common Options

Option Description
groupname Name of the group to manage.
/add Adds a new group or a user to an existing group.
/delete Deletes a group or a user from a group.
/domain Performs the operation on a domain controller.

Usage Examples 

# View all groups
net group

# Add a new group named "Developers"
net group Developers /add

# Add a user "JohnDoe" to the "Developers" group
net group Developers JohnDoe /add

# Remove a user "JohnDoe" from the "Developers" group
net group Developers JohnDoe /delete

# Delete the entire "Developers" group
net group Developers /delete

Cheat Sheet
