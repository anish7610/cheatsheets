Get-WmiObject Command

The Get-WmiObject command in Windows PowerShell retrieves management information from local and remote computers using Windows Management Instrumentation (WMI).

Common Options

Option Description
-Class Specifies the WMI class to query.
-Namespace Specifies the WMI namespace.
-ComputerName Specifies the target computer(s).
-Filter Provides a query string to narrow the retrieval.
-Credential Specifies credentials for remote connections.

Usage Examples

Retrieve System Information 

Get-WmiObject -Class Win32_OperatingSystem

Get Processor Info on a Remote Machine 

Get-WmiObject -Class Win32_Processor -ComputerName RemotePC -Credential Domain\User

List Services with a Filter 

Get-WmiObject -Class Win32_Service -Filter "State='Running'"

Compact Cheat Sheet 


Get-WmiObject -Class 
Get-WmiObject -Class  -ComputerName 
Get-WmiObject -Class  -Filter ""
