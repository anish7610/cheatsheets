Perfmon Command Overview

Description

The perfmon command launches the Windows Performance Monitor tool, which provides a visual interface for monitoring system performance metrics.

Common Options

Option Description
/report Generate a system diagnostics report.
/res Open the Resource Monitor.
/sys Open the Performance Monitor window.
/n filename Open a specific MMC console file.

Examples 

# Open Performance Monitor
perfmon /sys

# Generate and view a system diagnostics report
perfmon /report

# Launch the Resource Monitor
perfmon /res

Cheat Sheet 

perfmon /sys     # Launch Performance Monitor
perfmon /report  # Generate diagnostics report
perfmon /res     # Launch Resource Monitor
perfmon /n   # Open specific MMC file
