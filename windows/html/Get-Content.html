Get-Content Command Overview

Description

The Get-Content command in Windows is used to read and display the contents of a file. It is commonly used in PowerShell to access text data for viewing and processing.

Common Options

Option Description
-Path Specifies the file path to read the content from.
-TotalCount Reads a specified number of lines from the start of the file.
-Tail Displays the last specified number of lines from the file.
-Wait Continuously checks the file for new content, similar to 'tail -f'.
-Encoding Specifies the file encoding (e.g., UTF8, ASCII).

Usage Examples 

Get-Content -Path "C:\example\file.txt"
Get-Content -Path "C:\example\file.txt" -TotalCount 10
Get-Content -Path "C:\example\file.txt" -Tail 5
Get-Content -Path "C:\example\file.txt" -Encoding UTF8
