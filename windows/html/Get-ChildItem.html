Get-ChildItem Command Overview

The Get-ChildItem command in Windows PowerShell retrieves the items and child items in one or more specified locations.

Common Options

Option Description
-Path Specifies the path to the directory.
-Recurse Gets items in all child directories.
-File Lists only files.
-Directory Lists only directories.
-Filter Specifies a filter for the items.
-Include Includes only items that match the specified pattern.
-Exclude Excludes items that match the specified pattern.

Usage Examples 

Get-ChildItem -Path C:\Documents
Get-ChildItem -Path C:\Documents -Recurse
Get-ChildItem -Path C:\Documents -File
Get-ChildItem -Path C:\Documents -Directory
Get-ChildItem -Path C:\Documents -Filter *.txt
Get-ChildItem -Path C:\Documents -Include *.txt, *.docx
Get-ChildItem -Path C:\Documents -Exclude *.tmp

Cheat Sheet 

Get-ChildItem [options] -Path 
-Recurse: Include subdirectories
-File: Only files
-Directory: Only directories
-Filter: Filter items
-Include: Include patterns
-Exclude: Exclude patterns
