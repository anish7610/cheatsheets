Get-VMNetworkAdapter Command

Description

The Get-VMNetworkAdapter command is used in Windows PowerShell to retrieve information about virtual network adapters of virtual machines managed by Hyper-V.

Common Options

Option Description
-VMName <String> Specifies the name of the virtual machine.
-VMId <Guid> Specifies the GUID of the virtual machine.
-SwitchName <String> Filters network adapters connected to a specific virtual switch.
-ManagementOS Retrieves network adapters on the management operating system.
-CimSession <CimSession[]> Runs the command on remote sessions.

Usage Examples 


# Retrieve network adapters for a specific VM
Get-VMNetworkAdapter -VMName "MyVM"

# Get network adapters connected to a specific switch
Get-VMNetworkAdapter -SwitchName "MySwitch"

# List adapters on the management OS
Get-VMNetworkAdapter -ManagementOS

Cheat Sheet 


Get-VMNetworkAdapter -VMName "VM"
