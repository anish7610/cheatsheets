Windows `dir` Command

Description

The dir command is used in the Windows operating system to list the files and directories in a specified directory.

Common Options

Option Description
/p Pauses after each screen of information.
/q Displays the owner of the file.
/s Lists all files in the specified directory and its subdirectories.
/w Displays the listing in wide format, showing more files per line.
/a Displays files with specified attributes (e.g., /a:h for hidden files).
/o Lists files by files in a sorted order (e.g., /o:n by name).

Usage Examples 

# Lists files in the current directory with details
dir

# Lists files in wide format
dir /w

# Lists all files, including subdirectories
dir /s

# Lists files by name, in alphabetical order
dir /o:n

# Lists hidden files
dir /a:h

# Lists with pause after each screen
dir /p

Cheat Sheet
