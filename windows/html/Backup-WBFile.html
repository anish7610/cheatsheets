Backup-WBFile Command Overview

Description

The Backup-WBFile command in Windows PowerShell is used to include file specifications in a backup policy.

Common Options

Option Description
-Policy Specifies the backup policy to update.
-FileSpec Defines the set of files and directories to include.
-SystemState Includes the system state in the backup.

Usage Examples 

# Add files to the backup policy
Backup-WBFile -Policy $BackupPolicy -FileSpec C:\Users\Documents

# Include system state in the backup
Backup-WBFile -Policy $BackupPolicy -SystemState

# Add specific folders to the backup policy
Backup-WBFile -Policy $BackupPolicy -FileSpec D:\Projects\Code

Cheat Sheet 

# Add files: 
Backup-WBFile -Policy $BackupPolicy -FileSpec <Path>

# Include system state:
Backup-WBFile -Policy $BackupPolicy -SystemState
