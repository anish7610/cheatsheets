Get-SqlServer Command

Description

The Get-SqlServer command in PowerShell is used to retrieve information about SQL Server instances. It is commonly utilized for database management tasks.

Common Options

Option Description
-Name Specifies the SQL Server instance name to retrieve.
-Credential Allows specifying user credentials to access the SQL Server instance.
-Version Returns only instances of a specific SQL Server version.
-ServerInstance Specifies the server and instance name in a network.
-Force Forces the command to ignore warnings and proceed with execution.

Usage Examples 

Get-SqlServer -Name "MyServerInstance"
Get-SqlServer -ServerInstance "MyServer\Instance"
Get-SqlServer -Name "MyServerInstance" -Version "2019"
Get-SqlServer -ServerInstance "MyServer\Instance" -Credential (Get-Credential)
