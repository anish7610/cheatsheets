Get-Process Command

The Get-Process command in Windows retrieves information about processes running on a local or remote machine.

Common Options

Option Description
-Name Gets the process with the specified name.
-Id Gets the process with the specified process ID.
-ComputerName Gets processes on a remote computer.
-FileVersionInfo Displays file version information for the process.

Usage Examples 

# List all running processes
Get-Process

# Get processes by name
Get-Process -Name "notepad"

# Get a specific process by ID
Get-Process -Id 1234

# Get processes from a remote computer
Get-Process -ComputerName "RemotePC"

# Display version information of a process
Get-Process -Name "chrome" -FileVersionInfo

Cheat Sheet 

# List processes: Get-Process
# By name: Get-Process -Name "name"
# By ID: Get-Process -Id 1234
# Remote: Get-Process -ComputerName "PCName"
# Version info: Get-Process -Name "name" -FileVersionInfo
