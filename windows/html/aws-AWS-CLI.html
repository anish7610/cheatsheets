AWS CLI Overview

Description

The AWS Command Line Interface (CLI) is a unified tool to manage AWS services. With just one tool to download and configure, you can control multiple AWS services from the command line and automate them through scripts.

Common Options

Option Description
--region Specifies the AWS region to send requests to.
--output Specifies the output format (json, text, table).
--profile Uses a specific profile from your credential file.
--help Displays help information for a command.
--version Displays the version of the AWS CLI.

Usage Examples 

# List s3 buckets
aws s3 ls

# Upload a file to a specific bucket
aws s3 cp myfile.txt s3://mybucket/

# Describe EC2 instances
aws ec2 describe-instances --region us-west-1

# Use a specific profile
aws s3 ls --profile myprofile

Cheat Sheet 

# Configure CLI
aws configure

# List AWS services
aws s3 ls
aws ec2 describe-instances

# Specify region
aws <command> --region <region>

# Use specific profile
aws <command> --profile <profile-name>

# Display help
aws <command> --help
