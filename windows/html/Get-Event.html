Get-Event Command Overview

Description

The Get-Event command in Windows PowerShell retrieves events from the event queue in the current session. It is useful for handling and monitoring system events that have been created or triggered.

Common Options

Option Description
-SourceIdentifier <string> Specifies the source identifier of the events to retrieve.
-Newest <int> Retrieves the specified number of most recent events.
-After <DateTime> Returns events that were created after the specified date and time.

Usage Examples 

# Retrieve the most recent event
Get-Event -Newest 1

# Get events that have a specific source identifier
Get-Event -SourceIdentifier "MyCustomEvent"

# Retrieve all events after a specific date and time
Get-Event -After (Get-Date).AddDays(-1)

Cheat Sheet 

Get-Event -Newest 5
Get-Event -SourceIdentifier "Identifier"
Get-Event -After (Get-Date).AddMinutes(-10)
