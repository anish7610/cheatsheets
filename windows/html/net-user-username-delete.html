net user [username] /delete

Description

The net user [username] /delete command is used to remove a user account from a Windows system.

Common Options

Option Description
[username] The account name of the user to delete.
/domain Perform operation on the domain controller rather than the local machine.

Usage Examples 


net user johndoe /delete
net user janedoe /delete /domain

Cheat Sheet 


net user [username] /delete
net user [username] /delete /domain
