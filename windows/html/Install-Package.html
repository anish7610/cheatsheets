Install-Package Command

Description

The Install-Package command in Windows PowerShell is used to install software packages from various package sources. It integrates with package management solutions to streamline software installations and updates.

Common Options

Option Description
-Name The name of the package to install.
-Source The source repository from which to get the package.
-RequiredVersion The specific version of the package to install.
-Force Forces the installation even if the package is already installed.
-Credential Provides the user credentials for the package source.

Usage Examples 

# Install the latest version of a package
Install-Package -Name "packageName"

# Install a specific version of a package
Install-Package -Name "packageName" -RequiredVersion "1.2.3"

# Install a package from a specific source
Install-Package -Name "packageName" -Source "SourceRepository"

# Force installation of a package
Install-Package -Name "packageName" -Force

Quick Reference Cheat Sheet
