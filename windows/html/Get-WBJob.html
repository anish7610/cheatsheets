Get-WBJob Command Overview

The Get-WBJob command in Windows PowerShell retrieves information about backup jobs that have been created or are in progress on the system.

Common Options

Option Description
-Previous Displays the last backup job that was run before the current one.
-From DateTime Gets jobs starting from a specific date and time.
-To DateTime Gets jobs up until a specific date and time.

Usage Examples 

# Retrieve the current backup job
Get-WBJob

# Retrieve the last backup job
Get-WBJob -Previous

# Retrieve backups from a specific time range
Get-WBJob -From "2023-01-01" -To "2023-01-31"

Cheat Sheet 

# Current job
Get-WBJob

# Last job
Get-WBJob -Previous

# Jobs from/to date
Get-WBJob -From "YYYY-MM-DD" -To "YYYY-MM-DD"
