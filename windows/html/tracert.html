Tracert Command Overview

Description

The tracert command in Windows is used to display the path and transit times of data packets as they travel through a network to a specified destination.

Common Options

Option Description
-d Do not resolve addresses to hostnames.
-h maximum_hops Set the maximum number of hops to search for the target.
-w timeout Set the timeout in milliseconds for each reply.

Usage Examples 

tracert www.example.com
tracert -d www.example.com
tracert -h 15 www.example.com
tracert -w 2000 www.example.com

Cheat Sheet 

tracert [options] target
-d     No hostnames
-h #   Max hops
-w #   Timeout ms
