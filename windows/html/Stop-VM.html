Stop-VM Command

Description

The Stop-VM command is used in Windows PowerShell to stop a running virtual machine (VM) on a Hyper-V server.

Common Options

Option Description
-Name Specifies the name of the VM to stop.
-Force Forces the VM to stop immediately without a graceful shutdown.
-AsJob Runs the command as a background job.
-PassThru Returns an object that represents the VM.

Usage Examples 

# Gracefully stop a VM by name
Stop-VM -Name "MyVirtualMachine"

# Force stop a VM
Stop-VM -Name "MyVirtualMachine" -Force

# Stop a VM and run the command as a background job
Stop-VM -Name "MyVirtualMachine" -AsJob

# Stop a VM and return the VM object
Stop-VM -Name "MyVirtualMachine" -PassThru
