Docker Command

The docker command is used to interact with Docker containers and images. It allows you to manage containers, create images, and perform other container-related tasks.

Common Options

Option Description
docker run Start a new container from an image.
docker ps List running containers.
docker images List all images.
docker stop Stop a running container.
docker rm Remove a container.
docker rmi Remove an image.
docker pull Download an image from a registry.
docker exec Execute a command inside a running container.

Usage Examples 

# Start a new container from an image (e.g., nginx)
docker run -d -p 80:80 nginx

# List running containers
docker ps

# Stop a running container
docker stop container_id

# Remove a container
docker rm container_id

# Pull the latest image of Ubuntu
docker pull ubuntu:latest

# Execute a shell inside a running container
docker exec -it container_id /bin/bash
