Vagrant Command Overview

Vagrant is a command-line tool for building and managing virtualized development environments. It provides a consistent workflow for development teams and simplifies the setup of environments.

Common Options

Option Description
up Starts and provisions the Vagrant environment.
ssh Connects to machine via SSH.
halt Stops the running Vagrant machine.
destroy Stops and deletes all traces of the Vagrant machine.
status Outputs status of the current Vagrant environment.
box list Lists all available Vagrant boxes.
provision Re-runs the provisioning on a running Vagrant machine.

Usage Examples 

# Start the Vagrant environment
vagrant up

# Connect to the Vagrant machine via SSH
vagrant ssh

# Halt the Vagrant environment
vagrant halt

# Destroy the Vagrant environment
vagrant destroy

# Check the status of the Vagrant environment
vagrant status

# Re-run provisioning scripts
vagrant provision

# List all available boxes
vagrant box list
