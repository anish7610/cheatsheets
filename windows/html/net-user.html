Net User Command Overview

The net user command is used in Windows to manage user accounts.

Common Options

Option Description
net user Lists all user accounts.
net user <username> Displays detailed information about a specific user.
net user <username> <password> Sets or changes a user's password.
/add Adds a new user account.
/delete Deletes a user account.

Usage Examples 

net user JohnDoe /add
net user JohnDoe Password123
net user JohnDoe /delete
net user Administrator *
