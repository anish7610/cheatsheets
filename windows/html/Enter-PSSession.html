Enter-PSSession Command

Description

The Enter-PSSession command is used in PowerShell to start an interactive session with a remote computer.

Common Options

Option Description
-ComputerName Specifies the name of the remote computer to connect to.
-Credential Specifies the user account to use for the remote session.
-Port Specifies the port number to use for the connection.
-ConfigurationName Specifies the session configuration to use.
-UseSSL Indicates that the command uses SSL to establish a secure connection.

Usage Examples 

# Connect to a remote computer by name
Enter-PSSession -ComputerName Server01

# Connect with specific credentials
Enter-PSSession -ComputerName Server01 -Credential Domain\User

# Connect to a remote computer using SSL
Enter-PSSession -ComputerName Server01 -UseSSL

# Connect using a different port
Enter-PSSession -ComputerName Server01 -Port 5986

# Connect with a specific session configuration
Enter-PSSession -ComputerName Server01 -ConfigurationName CustomShell

Cheat Sheet 

Enter-PSSession -ComputerName [name] -Credential [user] -UseSSL
Enter-PSSession -ComputerName [name] -Port [port] -ConfigurationName [config]
