Get-Job Command Overview

Description

The Get-Job command in PowerShell retrieves information about jobs that are running in the background. It allows you to check the status of these jobs and access their results.

Common Options

Option Description
-Id Specifies an array of job IDs to retrieve information about specific jobs.
-Name Specifies an array of job names to retrieve information about specific jobs by name.
-State Filters jobs based on their state, e.g., Running, Completed, Failed.
-InstanceId Specifies an array of job instance IDs for retrieval.

Usage Examples 

# Retrieve all background jobs
Get-Job

# Retrieve a specific job by ID
Get-Job -Id 1

# Retrieve jobs by name
Get-Job -Name "Job1"

# Retrieve jobs that are completed
Get-Job -State Completed

Cheat Sheet 

Get-Job                  # List all jobs
Get-Job -Id           # Get job by ID
Get-Job -Name       # Get job by Name
Get-Job -State     # Filter by State
