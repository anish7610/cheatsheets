Hyper-V Manager Command Guide

Description

Hyper-V Manager is a Windows tool used to create, configure, and manage virtual machines and resources.

Common Options

Option Description
Create VM Initiate a new virtual machine.
Connect to Server Connect to a local or remote Hyper-V server.
Edit Disk Make changes to virtual hard disks.
Inspect Disk View detailed information about virtual disks.
Manage Snapshots Create or restore virtual machine snapshots.

Usage Examples

Create a Virtual Machine 

Hyper-V Manager > Action > New > Virtual Machine > Follow Wizard

Connect to a Hyper-V Server 

Hyper-V Manager > Action > Connect to Server > Enter Server Name

Edit a Virtual Disk 

Hyper-V Manager > Action > Edit Disk > Browse > Choose Action

Cheat Sheet 

Create VM: Action > New > Virtual Machine
Connect Server: Action > Connect to Server
Edit Disk: Action > Edit Disk
Inspect Disk: Action > Inspect Disk
Snapshot: Manage VM Snapshots
