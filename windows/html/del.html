Windows Command: del

Description

The del command is used to delete one or more files from a directory.

Common Options

Option Description
/P Prompts for confirmation before deleting each file.
/F Force deletes read-only files.
/S Deletes specified files from all subdirectories.
/Q Deletes without prompting for confirmation.
/A Deletes files based on attributes. Example: /A:H for hidden files.

Examples

  • Delete a specific file: 
    del file.txt
  • Delete all .txt files in a directory: 
    del *.txt
  • Delete files with confirmation: 
    del /P *.log
  • Delete read-only files without asking: 
    del /F /Q file.txt
  • Delete all files from subdirectories: 
    del /S *.tmp

Cheat Sheet 


del [filename]
del /P [filename]   # Prompt before delete
del /F /Q [filename] # Force delete read-only, no prompt
del /S [pattern]   # Delete in subdirectories
del *.txt          # Delete all .txt files
