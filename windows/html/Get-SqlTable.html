Get-SqlTable Command Overview

The Get-SqlTable command in PowerShell is used to retrieve metadata and information about tables in a SQL Server database.

Common Options

Option Description
-ServerInstance Specifies the SQL Server instance to connect to.
-Database Specifies the database name from which the table information is retrieved.
-Schema Filters the tables by a specific schema.
-Name Specifies the name of the table to retrieve information for.
-Credential Provides a credential object for authentication.

Usage Examples 

# Retrieve all tables in a specific database
Get-SqlTable -ServerInstance "MyServer" -Database "MyDatabase"

# Retrieve details of a specific table
Get-SqlTable -ServerInstance "MyServer" -Database "MyDatabase" -Name "MyTable"

# Retrieve tables within a specific schema
Get-SqlTable -ServerInstance "MyServer" -Database "MyDatabase" -Schema "MySchema"

Cheat Sheet 

-ServerInstance : SQL Server instance
-Database      : Target database
-Schema        : Table schema filter
-Name          : Specific table name
-Credential    : Authentication info
