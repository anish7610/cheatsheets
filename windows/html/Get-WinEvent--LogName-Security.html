Get-WinEvent Command

Description

The Get-WinEvent -LogName Security command retrieves events from the Windows Security event log, allowing administrators to monitor and analyze security-related activities.

Common Options

Option Description
-MaxEvents Specifies the maximum number of events to return.
-FilterXPath Applies an XPath query to filter specific events.
-FilterHashTable Uses a hash table to filter events based on criteria like ID or provider.
-Oldest Retrieves events from oldest to newest.

Usage Examples 

# Retrieve the 10 most recent security events
Get-WinEvent -LogName Security -MaxEvents 10

# Retrieve events with a specific ID (e.g., 4624)
Get-WinEvent -LogName Security -FilterHashTable @{ID=4624}

# Use XPath to filter events related to login failures
Get-WinEvent -LogName Security -FilterXPath "*[System[EventID=4625]]"
    
# Get the oldest 5 security events
Get-WinEvent -LogName Security -MaxEvents 5 -Oldest

Cheat Sheet 

Get-WinEvent -LogName Security -MaxEvents 
Get-WinEvent -LogName Security -FilterHashTable @{}
Get-WinEvent -LogName Security -FilterXPath ""
Get-WinEvent -LogName Security -Oldest
