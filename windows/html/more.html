Windows more Command

Description

The more command displays the contents of a file or output from another command, one screen at a time.

Common Options

Option Description
/C Clear screen before displaying page.
/P Pauses after each screen of information.
/S Squeeze multiple blank lines into a single one.
/Tn Set tab stops every n spaces (default is 8).

Usage Examples 

  • more filename.txt

    Displays the file filename.txt page by page. 

  • dir | more

    Displays the output of the dir command one page at a time. 

  • type filename.txt | more /C

    Clears the screen and displays filename.txt page by page.

Cheat Sheet 


more [options] [filename]
dir | more
more /C /T4 filename.txt
