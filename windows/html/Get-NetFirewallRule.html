Get-NetFirewallRule Command Overview

The Get-NetFirewallRule command is a Windows PowerShell cmdlet used to retrieve information about firewall rules on the local computer.

Common Options

Option Description
-DisplayName Filters results by the rule's display name.
-Enabled Specifies whether to retrieve only enabled or disabled rules.
-Direction Filters rules by the direction of network traffic (Inbound/Outbound).
-Action Filters rules by action taken (Allow/Block).

Usage Examples 

# View all firewall rules
Get-NetFirewallRule

# View all enabled firewall rules
Get-NetFirewallRule -Enabled True

# Find a rule by display name
Get-NetFirewallRule -DisplayName "My Firewall Rule"

# List all outbound rules that block traffic
Get-NetFirewallRule -Direction Outbound -Action Block

Quick Cheat Sheet 

# List all rules
Get-NetFirewallRule

# List enabled rules
Get-NetFirewallRule -Enabled True

# Filter by display name
Get-NetFirewallRule -DisplayName "Name"

# Outbound block rules
Get-NetFirewallRule -Direction Outbound -Action Block
