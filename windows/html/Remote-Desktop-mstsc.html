Remote Desktop (mstsc)

Description

The `mstsc` command launches the Remote Desktop Connection client, allowing users to connect to remote computers or virtual machines running Remote Desktop Protocol (RDP) server.

Common Options

Option Description
/v: server Specifies the remote computer and port (e.g., example.com:3389).
/admin Connects you to the session for administering a server.
/f Starts Remote Desktop in full-screen mode.
/w: width /h: height Specifies the dimensions of the Remote Desktop window.
/multimon Enables support for multiple monitors.

Usage Examples 

      
# Connect to a remote computer using default settings
mstsc /v:192.168.1.10

# Connect to a remote computer as admin
mstsc /v:192.168.1.10 /admin

# Full-screen connection
mstsc /v:example.com /f

# Connection with specific screen dimensions
mstsc /v:example.com /w:1280 /h:720

# Connect using multiple monitors
mstsc /v:example.com /multimon

Cheat Sheet 

      
mstsc /v: [/admin] [/f] [/w:] [/h:] [/multimon]
