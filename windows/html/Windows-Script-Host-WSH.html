Windows Script Host (WSH)

Windows Script Host (WSH) is a scripting utility that allows you to run scripts directly in the Windows environment. It supports scripts written in VBScript and JScript.

Common Options

Option Description
-e Executes the script in interpreted mode.
//B Batch mode; suppresses script errors and prompts.
//H:cscript Changes the default scripting host to CScript.
//H:wscript Changes the default scripting host to WScript.
//nologo Prevents WSH banner from appearing.
//S Saves the current command-line options for this user.
//T:nn Time out in seconds; maximum time allowed for the script to run.
//X Executes the script in debugger mode.

Usage Examples 


cscript //nologo myscript.vbs
wscript //B myscript.js
cscript //H:cscript //nologo script.vbs
