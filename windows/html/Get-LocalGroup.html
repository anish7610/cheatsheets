Understanding the `Get-LocalGroup` Command

Description

The Get-LocalGroup command retrieves information about local security groups on Windows, primarily used to manage permissions on systems.

Common Options

Option Description
-Name Specifies the name of the local group to retrieve.
-SID Retrieves a local group by its security identifier (SID).

Usage Examples 


# List all local groups
Get-LocalGroup

# Retrieve a specific group by name
Get-LocalGroup -Name "Administrators"

# Retrieve a group by security identifier (SID)
Get-LocalGroup -SID "S-1-5-32-544"

Cheat Sheet 


# List all groups
Get-LocalGroup

# Get group by name
Get-LocalGroup -Name "GroupName"

# Get group by SID
Get-LocalGroup -SID "GroupSID"
