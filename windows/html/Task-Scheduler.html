Task Scheduler Command Overview

The Task Scheduler command automates tasks by scheduling them to run at specified times or intervals on Windows systems.

Common Options

Option Description
/Create Creates a new scheduled task.
/Delete Deletes a scheduled task.
/Query Lists all scheduled tasks.
/Change Changes properties of an existing task.
/Run Runs a scheduled task immediately.

Usage Examples 


# Create a task to run a script daily at 8 AM
schtasks /Create /SC DAILY /TN "BackupScript" /TR "C:\Scripts\backup.bat" /ST 08:00

# Delete a previously scheduled task
schtasks /Delete /TN "BackupScript"

# List all scheduled tasks 
schtasks /Query

# Change the time of a task to 9 PM
schtasks /Change /TN "BackupScript" /ST 21:00

# Run a task immediately
schtasks /Run /TN "BackupScript"

Cheat Sheet 


# Create Task: schtasks /Create /SC frequency /TN "Name" /TR "Path" /ST HH:MM
# Delete Task: schtasks /Delete /TN "Name"
# List Tasks:  schtasks /Query
# Change Task: schtasks /Change /TN "Name" /ST HH:MM
# Run Task:    schtasks /Run /TN "Name"
