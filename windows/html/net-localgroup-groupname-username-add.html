Windows Command: net localgroup

Description

The net localgroup [groupname] [username] /add command adds a user to a specific local group on a Windows system. This is used for managing user permissions and access.

Common Options

Option Description
[groupname] Specifies the name of the group.
[username] Specifies the name of the user to add.
/add Adds a specified user to a specified group.
/delete Removes a specified user from a specified group.

Usage Examples 

net localgroup Administrators JohnDoe /add
net localgroup Users JaneDoe /add
net localgroup PowerUsers Alice /add

Cheat Sheet 


Add user to group: net localgroup [groupname] [username] /add
Remove user from group: net localgroup [groupname] [username] /delete
View group members: net localgroup [groupname]
