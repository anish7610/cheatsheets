New-WebApp Command Overview

Description

The New-WebApp command is used in Windows environments to create a new web application in IIS (Internet Information Services).

Common Options

Option Description
-Name Specifies the name of the web application.
-Site Indicates the site to which the application belongs.
-PhysicalPath Defines the physical path for the web application's files.
-ApplicationPool Specifies the application pool for the web application.

Usage Examples 

# Create a new web application
New-WebApp -Name "MyApp" -Site "Default Web Site" -PhysicalPath "C:\inetpub\wwwroot\myapp"

# Create a web application with a specific application pool
New-WebApp -Name "MyApp" -Site "Default Web Site" -PhysicalPath "C:\inetpub\wwwroot\myapp" -ApplicationPool "MyAppPool"

Cheat Sheet 


New-WebApp -Name "AppName" -Site "SiteName" -PhysicalPath "Path" [-ApplicationPool "PoolName"]
