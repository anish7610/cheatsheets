Visual Studio (devenv) Command Line

Description

The devenv command launches Microsoft Visual Studio and performs various tasks such as building projects or solutions from the command line.

Common Options

Option Description
/build Builds the specified solution or project.
/rebuild Cleans and then builds the specified solution or project.
/clean Deletes build outputs for the specified solution or project.
/project Specifies the project to build within a solution.
/out Specifies a file to receive logs of build operations.

Usage Examples 

# Build a solution
devenv MySolution.sln /build

# Rebuild a project within a solution
devenv MySolution.sln /rebuild /project MyProject

# Clean a solution and log output
devenv MySolution.sln /clean /out clean_output.txt

Cheat Sheet 


devenv  /build               # Build
devenv  /rebuild             # Rebuild
devenv  /clean               # Clean
devenv  /rebuild /project    # Rebuild specific project
