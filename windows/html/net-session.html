Net Session Command

Description

The net session command is used to manage server computer connections. It displays active connections or sessions to the computer.

Common Options

Option Description
/delete Ends a session with a specific computer and closes all open files on the computer for the session.
/? Displays help for the command.

Examples 

net session
net session \\ComputerName /delete

Cheat Sheet 

net session        # List all active sessions
net session /delete  # Delete all sessions
net session \\Name /delete  # End session with specific computer
