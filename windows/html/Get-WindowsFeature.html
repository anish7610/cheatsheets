Get-WindowsFeature Command Overview

The Get-WindowsFeature command in Windows PowerShell is used to list the roles and features installed or available on a Windows Server.

Common Options

Option Description
-Name Specifies the name of the feature to retrieve information about.
-ComputerName Runs the command on a remote computer.
-Credential Runs the command under a specified user account.
-LogPath Specifies a path for the command logs.
-ErrorVariable Stores error messages in the specified variable.

Usage Examples 

# List all features and their installation state
Get-WindowsFeature

# Get details of a specific feature by name
Get-WindowsFeature -Name Web-Server

# Check features on a remote server
Get-WindowsFeature -ComputerName Server01

# Run the command with specific credentials
$cred = Get-Credential
Get-WindowsFeature -Credential $cred

Compact Cheat Sheet 

# List all features
Get-WindowsFeature

# Specific feature
Get-WindowsFeature -Name FeatureName

# Remote server
Get-WindowsFeature -ComputerName ServerName
