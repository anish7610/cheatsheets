winget Command

Windows Package Manager for installing, upgrading, configuring, and removing software.

Common Options

Option Description
install Install a package by name.
show Display information about a package.
search Search for a package by name or keyword.
list List installed packages.
upgrade Upgrade the specified package or all packages.
uninstall Uninstall a package.
help Show help information for the command.

Usage Examples 


# Install a package
winget install "7zip"

# Show information about a package
winget show "notepad++"

# Search for a package
winget search "Visual Studio Code"

# List installed packages
winget list

# Upgrade all packages
winget upgrade --all

# Uninstall a package
winget uninstall "Google Chrome"

Cheat Sheet 


winget install [package]
winget show [package]
winget search [keyword]
winget list
winget upgrade [package|--all]
winget uninstall [package]
