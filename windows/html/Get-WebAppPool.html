Get-WebAppPool Command

Description

The Get-WebAppPool command is used to retrieve information about application pools in Internet Information Services (IIS) on Windows systems.

Common Options

Option Description
-Name <string> Specifies the name of the application pool to retrieve.
-PSPath <string> Specifies the path to the IIS configuration hierarchy.

Usage Examples 

# Retrieve all application pools
Get-WebAppPool

# Retrieve a specific application pool by name
Get-WebAppPool -Name "DefaultAppPool"

Cheat Sheet 

# List all app pools
Get-WebAppPool

# Get details of a specific pool
Get-WebAppPool -Name "YourAppPoolName"
