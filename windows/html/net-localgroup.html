Windows Command: net localgroup

The net localgroup command is used to manage local group accounts on Windows, allowing administrators to add or remove users from groups.

Common Options

Option Description
/add Adds a group or user account to a local group.
/delete Deletes a group or removes a user from a group.
<groupname> Specifies the name of the group to manage.
<username> Specifies the name of the user to add or remove.
/domain Performs the operation on a domain controller, not locally.

Example Usage 

net localgroup Administrators JohnDoe /add
net localgroup Users /delete
net localgroup "Power Users" JaneDoe /add

Cheat Sheet 

# Add user to group
net localgroup [groupname] [username] /add

# Remove user from group
net localgroup [groupname] [username] /delete

# Delete group
net localgroup [groupname] /delete
