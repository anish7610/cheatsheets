Eclipse Command on Windows

The Eclipse command is used to start the Eclipse IDE from the command line, allowing developers to manage projects, workspaces, and plugins efficiently.

Common Options

Option Description
-data <workspace-path> Specifies the workspace location.
-nosplash Starts Eclipse without showing the splash screen.
-clean Clears the configuration area before starting.
-vm <jvm-path> Specifies the JVM to run Eclipse.
-application <id> Runs the specified application within the workspace.

Usage Examples 

# Start Eclipse with a specific workspace
eclipse -data "C:\path\to\workspace"

# Start Eclipse without splash screen
eclipse -nosplash

# Start Eclipse with a clean configuration
eclipse -clean

# Start Eclipse with a specific JVM
eclipse -vm "C:\path\to\jvm\bin\javaw.exe"

Cheat Sheet 

-data <workspace>  # Set workspace
-nosplash              # No splash screen
-clean                 # Clean config
-vm <jvm-path>      # Specify JVM
-application <id>    # Run application
