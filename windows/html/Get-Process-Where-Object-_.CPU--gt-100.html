Understanding the Windows Command

Short Description

The command Get-Process | Where-Object { $_.CPU -gt 100 } retrieves a list of processes and filters them to show only those using more than 100 CPU units.

Common Options and Explanations

Option Explanation
-ProcessName "Name" Filter by the name of the process.
-Id PID Select a process by its ID.
-FileVersionInfo Include version information for process executables.
-IncludeUserName Show user names for processes (requires admin rights).

Usage Examples 

# List processes using more than 200 CPU units
Get-Process | Where-Object { $_.CPU -gt 200 }

# Find processes named "notepad" with CPU usage over 50
Get-Process -ProcessName "notepad" | Where-Object { $_.CPU -gt 50 }

# List all processes with version info
Get-Process -FileVersionInfo

Cheat Sheet 

# Basic filter
Get-Process | Where-Object { $_.Property -Condition Value }

Get-Process | Where-Object { $_.CPU -gt 100 }
