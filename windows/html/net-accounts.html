Net Accounts Command

The net accounts command is used to set and display user account policies, such as password and logon requirements, on a Windows system.

Common Options

Option Description
/minpwlen Sets minimum password length.
/maxpwage Sets maximum number of days a password is valid.
/minpwage Sets minimum number of days before a password can be changed.
/uniquepw Requires that a specified number of previous passwords cannot be reused.
/forcelogoff Sets auto logoff delay after a forced logoff warning.
/help Displays help for the command.

Usage Examples 

net accounts /minpwlen:8

Set the minimum password length to 8 characters. 

net accounts /maxpwage:30

Set passwords to expire every 30 days. 

net accounts /uniquepw:5

Ensure that the previous 5 passwords cannot be reused.

Cheat Sheet 

/minpwlen:n     # Minimum password length
/maxpwage:n     # Maximum password age
/minpwage:n     # Minimum password age
/uniquepw:n     # Unique passwords count
/forcelogoff:n  # Auto logoff delay
