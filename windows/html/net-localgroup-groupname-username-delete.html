Windows Command: net localgroup [groupname] [username] /delete

Description

The net localgroup [groupname] [username] /delete command is used to remove a user from a specified local group on a Windows machine.

Common Options

Option Description
[groupname] The name of the local group from which the user will be removed.
[username] The name of the user to be removed from the group.
/delete Indicates the action to remove the user from the group.

Usage Examples 

# Remove user 'john' from the 'Administrators' group
net localgroup Administrators john /delete

# Remove user 'guest' from the 'Users' group
net localgroup Users guest /delete

Cheat Sheet 

# Remove user from group
net localgroup [groupname] [username] /delete
