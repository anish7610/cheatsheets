Windows Command: Get-Volume

Description

The Get-Volume command retrieves information about the disk volumes present on a Windows system.

Common Options

Option Description
-DriveLetter Specifies the drive letter of the volume to get information for.
-FileSystemLabel Filters volumes by the file system label.
-CimSession Runs the command in a user-specified session or computer.
-HealthStatus Filters volumes by health status (e.g., Healthy, Unhealthy).

Usage Examples 

Get-Volume
Get-Volume -DriveLetter C
Get-Volume -FileSystemLabel Data
Get-Volume -HealthStatus Healthy

Cheat Sheet 

Get-Volume            # Get all volumes
Get-Volume -DriveLetter D  # Get volume with drive letter D
Get-Volume -FileSystemLabel WorkDocs # Volumes with label
Get-Volume -HealthStatus Warning  # Volumes with Unhealthy status
