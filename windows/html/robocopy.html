Robocopy Command Guide

Description

Robocopy is a robust file copy command for Windows, designed to handle large file transfers, and provide options for retrying, mirroring directories, and more.

Common Options

Option Description
/S Copy subdirectories, excluding empty ones.
/E Copy all subdirectories, including empty ones.
/MIR Mirror a directory tree (equivalent to combining /E and /PURGE).
/PURGE Delete destination files and directories that no longer exist in the source.
/MOV Move files (delete from source after copying).
/ETA Show estimated time of arrival of copied files.
/R:n Number of retries on failed copies (default is 1 million).
/W:n Wait time between retries in seconds (default is 30).
/LOG:file Log output to a specified file.

Usage Examples 

# Basic copy with subdirectories
robocopy C:\Source D:\Destination /S

# Mirror directories
robocopy C:\Source D:\Destination /MIR

# Move files and log the action
robocopy C:\Source D:\Destination /MOV /LOG:C:\logfile.txt

# Retry 3 times and wait 5 seconds between retries
robocopy C:\Source D:\Destination /R:3 /W:5

Cheat Sheet 

robocopy   [options]
/S          # Copy subdirectories
/E          # Include empty directories
/MIR        # Mirror directories
/PURGE      # Remove non-existent files/directories
/MOV        # Move files
/ETA        # Show ETA
/R:n        # Retry count
/W:n        # Wait time between retries
/LOG:file   # Output log file
