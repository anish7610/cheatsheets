Invoke-Sqlcmd

Invoke-Sqlcmd is a command-line tool in PowerShell used to run T-SQL commands on SQL Server. It allows for automation of database tasks and queries directly from a script.

Common Options

Option Description
-Query Specifies the T-SQL query to be executed.
-ServerInstance Specifies the SQL Server instance.
-Database Specifies the target database.
-Username Specifies the SQL Server login account.
-Password Specifies the password for the SQL Server account.
-InputFile Specifies a file containing the T-SQL script to execute.

Usage Examples 


# Example 1: Run a simple query
Invoke-Sqlcmd -ServerInstance "ServerName" -Database "DatabaseName" -Query "SELECT * FROM TableName"

# Example 2: Run a query with authentication
Invoke-Sqlcmd -ServerInstance "ServerName" -Database "DatabaseName" -Username "User" -Password "Pass" -Query "SELECT * FROM TableName"

# Example 3: Run a query from a file
Invoke-Sqlcmd -ServerInstance "ServerName" -Database "DatabaseName" -InputFile "C:\Path\To\Script.sql"

Cheat Sheet 

Invoke-Sqlcmd -ServerInstance "Server" -Database "DB" -Query "T-SQL"
Invoke-Sqlcmd -ServerInstance "Server" -Database "DB" -InputFile "File.sql"
