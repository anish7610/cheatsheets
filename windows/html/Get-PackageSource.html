Get-PackageSource Command

Get-PackageSource is a PowerShell command used to list software package sources registered on a system.

Common Options

Option Description
-ProviderName Specifies the package provider name.
-Name Filters results by package source name.
-IsTrusted Returns only package sources that are trusted.

Examples

List All Package Sources 

Get-PackageSource

Filter by Provider Name 

Get-PackageSource -ProviderName NuGet

Get Trusted Package Sources 

Get-PackageSource -IsTrusted

Cheat Sheet 

Get-PackageSource
Get-PackageSource -ProviderName <ProviderName>
Get-PackageSource -Name <SourceName>
Get-PackageSource -IsTrusted
