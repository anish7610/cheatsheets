Get-NetTCPConnection

Description

The Get-NetTCPConnection command is a PowerShell cmdlet in Windows used to get information about current TCP connections.

Common Options

Option Description
-State Filters connections by their state (e.g., Established, Listen).
-LocalPort Filters connections by local port.
-RemotePort Filters connections by remote port.
-LocalAddress Filters connections by local address.
-RemoteAddress Filters connections by remote address.
-OwningProcess Filters connections by owning process ID.

Examples 

# Get all established connections
Get-NetTCPConnection -State Established

# Filter by local port
Get-NetTCPConnection -LocalPort 80

# Filter by remote address
Get-NetTCPConnection -RemoteAddress 192.168.1.1

# Get connections for a specific process
Get-NetTCPConnection -OwningProcess 1234

Cheat Sheet
