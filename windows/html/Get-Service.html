Windows `Get-Service` Command

Description

The Get-Service command in Windows PowerShell retrieves status information about services on a local or remote machine.

Common Options

Option Description
-Name Specifies the service name. Wildcards are supported.
-ComputerName Queries services on a remote computer.
-DisplayName Filters services by their display name.
-Status Filters services by their status (e.g., Running, Stopped).

Usage Examples 

# List all services on the local machine
Get-Service

# Get services by name with wildcard
Get-Service -Name "W*"

# Check the status of a specific service
Get-Service -Name "Spooler"

# List all running services
Get-Service -Status Running

# Query services on a remote machine
Get-Service -ComputerName "REMOTE-PC" -Name "Spooler"

Cheat Sheet 

Get-Service -Name "ServiceName"
Get-Service -Status Running
