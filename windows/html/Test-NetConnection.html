Test-NetConnection Command Overview

Description

The Test-NetConnection command is a versatile Windows cmdlet used to diagnose network connectivity. It provides details about network connection status, latency, and more.

Common Options

Option Description
-ComputerName Specifies the remote system to test.
-Port Tests the connection to a specific port.
-TraceRoute Performs a tracert to the specified destination.
-InformationLevel Controls the amount of output information (e.g., Detailed ).

Usage Examples 

# Test network connectivity to google.com
Test-NetConnection -ComputerName google.com

# Test connectivity to a specific port on google.com
Test-NetConnection -ComputerName google.com -Port 443

# Perform a traceroute to google.com
Test-NetConnection -ComputerName google.com -TraceRoute

# Get detailed connection information to google.com
Test-NetConnection -ComputerName google.com -InformationLevel Detailed

Cheat Sheet 

# Basic connectivity test
Test-NetConnection [-ComputerName] 

# Test specific port
Test-NetConnection -ComputerName  -Port 

# Perform traceroute
Test-NetConnection -ComputerName  -TraceRoute
