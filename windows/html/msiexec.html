msiexec Command Overview

Description

The msiexec command is used to install, repair, or uninstall Windows Installer packages with the extension .msi . It is a command-line utility for managing Windows Installer actions.

Common Options

Option Description
/i package Install the specified package .
/x package Uninstall the specified package .
/f package Repair the specified package .
/qn Run the installer with no user interface.
/quiet An alternative to /qn for quiet installation without UI.
/l*v logfile Log installation information to the specified logfile with verbose output.

Usage Examples 


    
    msiexec /i example.msi

    
    msiexec /x example.msi

    
    msiexec /f example.msi

    
    msiexec /i example.msi /quiet /l*v install.log

Cheat Sheet 


    Install: msiexec /i [package]
    Uninstall: msiexec /x [package]
    Repair: msiexec /f [package]
    Quiet Install: msiexec /i [package] /quiet
    Log: msiexec /i [package] /l*v [logfile]
