Azure CLI (`az`)

Description

The Azure Command-Line Interface (CLI) is a tool for managing Azure resources directly from the command line. It allows performing a variety of management and deployment tasks on Azure.

Common Options

Option Description
--help Show help information for a command or group.
--output / -o Specify the format of command output (json, table, tsv).
--query Filter results using a JMESPath query string.
--verbose Increase logging verbosity for debugging.
--debug Show all debug information.

Usage Examples 

# Log in to Azure
az login

# List all resource groups
az group list --output table

# Create a new resource group
az group create --name MyResourceGroup --location eastus

# Show details of a specific resource group
az group show --name MyResourceGroup --output json

# Delete a resource group
az group delete --name MyResourceGroup --yes --no-wait

Cheat Sheet 

az login                     # Log in to Azure
az --help                    # Get help
az  --help         # Get help for specific resource
az  list -o table  # List resources in table format
az  show -o json   # Show resource details in JSON
