New-Item Command Overview

Description

The New-Item command in Windows PowerShell is used to create a new file, directory, or other type of item.

Common Options

Option Description
-Path Specifies the location where the new item will be created.
-Name Specifies the name of the new item.
-ItemType Defines the type of item to create. Common types are File , Directory .
-Value Provides the content for the new file.
-Force Allows creation of a new item in a read-only location.

Usage Examples 

# Create a new directory
New-Item -Path "C:\Example" -Name "NewFolder" -ItemType "Directory"

# Create a new file with content
New-Item -Path "C:\Example" -Name "NewFile.txt" -ItemType "File" -Value "Hello, World!"

# Create a new empty file
New-Item -Path "C:\Example" -Name "EmptyFile.txt" -ItemType "File"

Cheat Sheet 

New-Item -Path <path> -Name <name> -ItemType <type> [-Value <value>] [-Force]
