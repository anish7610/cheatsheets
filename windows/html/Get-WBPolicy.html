Get-WBPolicy Command Overview

The Get-WBPolicy cmdlet retrieves the current backup policy for a Windows Server. It's used to check the configuration of the backup schedule, backup targets, and items included in backups.

Common Options

Option Description
-PolicyEditable Gets a copy of the backup policy that can be modified.

Usage Examples

Retrieve Current Backup Policy 

Get-WBPolicy

Get Editable Copy of Backup Policy 

Get-WBPolicy -PolicyEditable

Cheat Sheet 

# Retrieve current backup policy
Get-WBPolicy

# Get editable copy of backup policy
Get-WBPolicy -PolicyEditable
