Windows Command: copy

Description

The copy command is used to copy files from one location to another in Windows.

Common Options

Option Description
/a Treats the files as ASCII text files.
/b Treats the files as binary files.
/v Verifies that new files are written correctly.
/y Suppresses prompt to confirm overwriting a file.

Usage Examples 

C:\> copy file1.txt file2.txt
C:\> copy /y source.txt C:\backup\
C:\> copy /v /b image.jpg D:\images\

Cheat Sheet 

copy [options] source destination
copy file1.txt C:\folder\
copy /y file.txt C:\backup\
copy /v /b source.bin D:\backup\
