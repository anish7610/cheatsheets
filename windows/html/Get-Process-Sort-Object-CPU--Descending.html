Understanding the Command

The command Get-Process | Sort-Object CPU -Descending in Windows PowerShell retrieves a list of all running processes, sorted by CPU usage in descending order.

Common Options

Option Description
Get-Process Retrieves information about local or remote processes.
Sort-Object Sorts objects based on property values.
-Descending Sorts the results in descending order.
CPU Specifies the property to sort by (CPU usage).

Usage Examples 

Get-Process | Sort-Object CPU -Descending
# Lists processes with the highest CPU usage first.

Get-Process | Sort-Object WorkingSet -Descending | Select-Object -First 5
# Displays the top 5 processes consuming the most memory.

Get-Process -Name "chrome" | Sort-Object CPU -Descending
# Sorts Chrome processes by CPU usage.
