Docker Compose Command Overview

Docker Compose is a tool for defining and running multi-container Docker applications. Using a docker-compose.yml file, you can configure your appâ€™s services, networks, and volumes in one place.

Common Options

Option Description
up Builds, (re)creates, starts, and attaches to containers for a service.
down Stops containers and removes containers, networks, volumes, and images created by up .
build Builds or rebuilds services.
ps Lists containers.
logs Views output from containers.
config Validates and views the Compose file.
restart Restarts running services.

Usage Examples 

# Start all services defined in docker-compose.yml
docker-compose up

# Start services in detached mode (background)
docker-compose up -d

# Stop and remove all containers defined in docker-compose.yml
docker-compose down

# Build or rebuild services
docker-compose build

# View logs for all services
docker-compose logs

# View configuration details
docker-compose config

# List all running containers
docker-compose ps
