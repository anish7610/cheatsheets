Choco Command

Choco is a command-line package manager for Windows that simplifies the installation, upgrading, and management of software packages.

Common Options

Option Description
install Installs a package.
upgrade Upgrades a package to the latest version.
uninstall Uninstalls a package.
list Lists installed or available packages.
search Searches for packages online.
help Displays help information about commands.

Usage Examples 

# Install a package
choco install git

# Upgrade a package
choco upgrade googlechrome

# Uninstall a package
choco uninstall vlc

# List installed packages
choco list --localonly

# Search for a package
choco search firefox
