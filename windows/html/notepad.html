Notepad Command

Description

The `notepad` command is used to open a simple text editor in Windows. It is commonly used for editing text files.

Common Options

Option Description
notepad Opens Notepad without any file loaded.
notepad <filename> Opens the specified file in Notepad.

Usage Examples 

notepad
notepad example.txt
notepad "C:\Path\To\Your\File.txt"

Cheat Sheet 

notepad             # Open Notepad
notepad file.txt    # Open file.txt in Notepad
notepad "C:\path\to\file.txt"  # Open a specific file
