Windows Command: netsh interface

Description

The netsh interface command is used to manage and configure network interfaces on Windows systems. It allows users to view and adjust networking settings.

Options and Explanations

Option Description
show interface Displays a list of network interfaces and their status.
set interface Configures parameters such as enabling/disabling an interface.
ip set address Assigns a static IP address to a network interface.
ip show config Displays IP configuration of network interfaces.
reset Resets all interfaces to default settings.

Usage Examples 

netsh interface show interface

netsh interface set interface "Ethernet" admin=disable

netsh interface ip set address "Ethernet" static 192.168.1.10 255.255.255.0 192.168.1.1

netsh interface ip show config

netsh interface reset all

Cheat Sheet 

show interface - List interfaces
set interface "Name" admin=disable - Disable interface
ip set address "Name" static IP Subnet Gateway - Set static IP
ip show config - Show IP config
reset all - Reset settings
