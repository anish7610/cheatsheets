Get-SqlInstance Command Overview

Description

The Get-SqlInstance command retrieves a list of SQL Server instances and their properties. It is part of the SQLServer module in PowerShell.

Common Options

Option Description
-ServerInstance Specifies the SQL Server instance to query.
-Credential Provides credentials when connecting to the SQL Server.
-ComputerName Specifies the remote computer's name where the SQL instances are checked.
-Version Filters results by SQL Server version.

Usage Examples 

# List all SQL instances on the local machine
Get-SqlInstance

# Retrieve SQL instances on a specific server
Get-SqlInstance -ComputerName "SERVER01"

# Get SQL instances using specific credentials
$cred = Get-Credential
Get-SqlInstance -Credential $cred -ComputerName "SERVER02"

# Filter instances by SQL Server version
Get-SqlInstance -Version "15.0"
