Windows Command: type

Description

The type command is used in Windows to display the contents of a text file on the command line.

Common Options

Option Description
(none) Outputs the entire content of a file to the terminal.

Usage Examples

Display a Single File 

type filename.txt

Display Multiple Files 

type file1.txt file2.txt

Combine and View Files 

type file1.txt file2.txt > combined.txt

Cheat Sheet 


type filename.txt        # Display contents of a file
type file1.txt file2.txt # Display multiple files
type file > newfile.txt  # Redirect output to a new file
