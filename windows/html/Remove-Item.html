Windows Command: Remove-Item

Description

The Remove-Item command deletes files, directories, and other data locations in Windows PowerShell.

Common Options

Option Description
-Path Specifies the path to the item to be removed.
-Recurse Deletes all items in a specified location, including the subdirectory items.
-Force Forces the command to remove items that are read-only or hidden.
-WhatIf Shows what would happen if the command runs without actually executing it.
-Confirm Prompts for confirmation before executing the removal.

Examples 

# Delete a single file
Remove-Item -Path "C:\Example\file.txt"

# Delete a directory and all contents
Remove-Item -Path "C:\Example\Directory" -Recurse

# Forcefully delete a read-only file
Remove-Item -Path "C:\Example\read-only-file.txt" -Force

# Show what would happen without deleting
Remove-Item -Path "C:\Example\file.txt" -WhatIf

Cheat Sheet 


Remove-Item -Path  [-Recurse] [-Force] [-WhatIf] [-Confirm]
