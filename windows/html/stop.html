Windows Command: stop

Description

The stop command is typically used to halt and stop services running on a Windows operating system.

Common Options

Option Description
/y Confirms the stopping of a service without prompt.
/force Forces the service to stop if it does not respond.
/nowait Returns immediately without waiting for stop confirmation.

Usage Examples 

stop someServiceName /y
stop someServiceName /force
stop someServiceName /nowait

Cheat Sheet 

stop [serviceName] [/y] [/force] [/nowait]
